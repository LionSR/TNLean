\section{Source ambiguity}

In \cite[Theorem~\leanid{thm1}, lines~1170--1192]{Cirac2016MPDO_arXiv} the input is:
\begin{align}
    \forall N,\quad V_A^{(N)}\sim V_B^{(N)}.
\end{align}
The phrase does not explicitly state whether the proportionality scalar is allowed to vanish.

\section{Two formal readings}

For the weak reading
\begin{align}
    \forall N,\ \exists c_N\in\mathbb C,
    \quad V_A^{(N)}=c_N\,V_B^{(N)}.
\end{align}
For the projective reading
\begin{align}
    \forall N,\ \exists c_N\in\mathbb C^\times,
    \quad V_A^{(N)}=c_N\,V_B^{(N)}.
\end{align}

Reading \((P)\) is the projective (nonzero) interpretation used in the formal
\leanid{NonzeroProportionalMPV2} predicate.

\section{Why zero is not acceptable in the contradiction line}

For block decompositions
\begin{align}
    V_A^{(N)} &=\sum_{j=1}^{g_A}\mu_j^N a_j(N),\qquad
    V_B^{(N)}=\sum_{\ell=1}^{g_B}\nu_\ell^N b_\ell(N),
\end{align}
the fixed-block relation becomes
\begin{align}
    \sum_{j=1}^{g_A}\mu_j^N a_j(N)
    =c_N\,\nu_k^N b_k(N),
\end{align}
in the tail-free base step.
Linear independence then forces
\begin{align}
    c_N\nu_k^N=0.
\end{align}
To contradict, one needs \(c_N\nu_k^N\neq 0\). This is immediate from \((P)\), but not from \((W)\):
\begin{align}
    c_N &=0\implies V_A^{(N)}=0\cdot V_B^{(N)},
\end{align}
which gives no nonzero coefficient contradiction.

In the source fixed-block step, the branch in which a contradiction is extracted is
involving normalized and nonzero sector states, so \(c_N\in\mathbb C^\times\) is the effective reading.

\section{Compatibility with source CF normalization}

Under the normalization in \cite[Definition~\leanid{II\_CF1}]{Cirac2016MPDO_arXiv}, selected blocks satisfy
\begin{align}
    \|b_k(N)\|^2
    =\langle b_k(N),b_k(N)\rangle\to1
\end{align}
and hence \(b_k(N)\neq0\) for large \(N\).
Thus zero coefficient cases are not compatible with that part of the intended contradiction argument.

Conclusion:
The formal hypothesis \((P)\) is exactly the source usage on contradiction branches.

\section{Effect on formal routing}

There is no new theorem needed to keep the block-selection argument faithful.
The formal projective predicate is \leanid{NonzeroProportionalMPV2}
and it is stricter than
\leanid{ProportionalMPV2}.
because \leanid{ProportionalMPV2} allows \(c_N=0\).
If a future lemma later needs the weak form, one can add a bridge lemma:
\begin{align}
    (\forall N,\ c_N\in\mathbb C)\wedge(V_A^{(N)}\neq0)\wedge(V_B^{(N)}\neq0)
    \Longrightarrow c_N\in\mathbb C^\times.
\end{align}

\section{Conclusion}

The choice \((P)\) is not an extra axiom; it is a formalization of the source's projective
reading in the non-decaying-then-cancel branch used by \cite[Theorem~\leanid{thm1}]{Cirac2016MPDO_arXiv}.
