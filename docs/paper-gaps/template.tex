\section{The cited statement}

This note is a model for the documents in \path{docs/paper-gaps/}.  A real note
should replace the symbols below by the notation of the cited article and should
identify the source passage by a bibliographic citation.  The visible document
should be a mathematical explanation, not a form to be filled out.

Suppose that the cited source proves a theorem under hypotheses
\(\mathcal H\):
\begin{align}
    \mathcal H \Longrightarrow C.
    \tag{\path{thm:source}}
\end{align}
Here \(C\) is the conclusion later used in the argument.  In an actual note,
the first paragraph should say where this implication appears and where it is
used, for example \cite[Theorem~4.4]{Cirac2021Matrix}.

\section{The formal statement}

Assume that the formal development proves instead
\begin{align}
    \mathcal H\wedge\mathcal K \Longrightarrow C.
    \tag{\path{thm:formal}}
\end{align}
The additional condition \(\mathcal K\) may be harmless, may follow from
\(\mathcal H\), or may be a genuine strengthening.  The point of the note is to
decide which of these alternatives is correct.

For instance, if the formal theorem is named \leanid{modelCorrected}, that name
belongs in a footnote together with the file location, because it is traceability
data rather than part of the mathematical assertion.\footnote{Model formal
    declaration: \leanid{modelCorrected} in \doclink{TNLean/Path/To/File}.}
The main text should still state the theorem as mathematics.

\section{The calculation}

The comparison should isolate the exact place where the hypotheses differ.  In
this model the cited proof supplies an intermediate estimate
\begin{align}
    F_N           &\leq \varepsilon_N, \tag{\path{eq:source-bound}}\\
    \varepsilon_N &\xrightarrow[N\to\infty]{}0.
\end{align}
The formal proof, however, obtains the same estimate only after using
\(\mathcal K\):
\begin{align}
    \mathcal H\wedge\mathcal K
    \quad\Longrightarrow\quad
    F_N \leq \varepsilon_N
    \quad\Longrightarrow\quad
    C.
\end{align}
If \(\mathcal K\) is a trace normalization, a nonzero scalar condition, or a
choice of gauge, the note should write it explicitly, for example
\begin{align}
    \tr(\rho)=1,\qquad c_N\neq 0,\qquad
    B^i=\zeta X A^iX^{-1}.
\end{align}

\section{Verdict}

The conclusion should state the present mathematical status.  If
\(\mathcal K\) follows from \(\mathcal H\), the note should give the derivation.
If it does not, the formal theorem is a restricted result:
\begin{align}
    \mathcal H\wedge\mathcal K \Longrightarrow C
    \qquad\text{is not the same theorem as}\qquad
    \mathcal H \Longrightarrow C.
\end{align}
If the discrepancy is only a local correction, the note should say so and
identify the corrected formula.  If the issue remains open, it should say what
has been checked and what remains to be proved.
