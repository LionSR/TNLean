\section{Source definitions used by \leanid{thm1} and \leanid{II\_cor2}}

In canonical form,
\begin{align}
    A^i             &=\bigoplus_{k=1}^r\mu_kA_k^i,\\
    \text{with }A_k &\text{ normal},\\
    V^{(N)}(A)      &=\sum_{k=1}^r\mu_k^N V^{(N)}(A_k).
\end{align}
The MPV expansion is
\begin{align}
    V^{(N)}(A)          &=\sum_{k=1}^r\mu_k^N V^{(N)}(A_k),\\
    \forall k,\ |\mu_k| &\le1,\qquad \exists k_0,\ |\mu_{k_0}|=1.
\end{align}
This is not optional; it is part of the source convention in
\cite[Section~II]{Cirac2016MPDO_arXiv}.

\section{What the source uses in Corollary~II.2}

Under global proportionality, for each block \(j\) with multiplicities \(r_{a,j},r_{b,j}\),
\begin{align}
    \sum_{q=1}^{r_{a,j}}\mu_{j,q}^N &=\sum_{q=1}^{r_{b,j}}(\nu_{j,q}e^{i\phi_j})^N,\qquad \forall N\ge1
\end{align}
from which, with finite-length power-sum separation
\begin{align}
    \sum_{q=1}^{r_{a,j}}\lambda_{a,q}^N &=\sum_{q=1}^{r_{b,j}}\lambda_{b,q}^N,\quad N=1,\dots,\max(r_{a,j},r_{b,j}),
\end{align}
one gets \(r_{a,j}=r_{b,j}\) and \(\mu_{j,q}=\nu_{j,q}e^{i\phi_j}\) by
\cite[Lemma~\leanid{Lem:app\_simple}]{Cirac2016MPDO_arXiv}.

\section{Where the implemented route departed}

The implementation introduced sequences \(a_{N,j}\), limits \(a_j^\infty\ne0\), and convergence hypotheses at infinity
\begin{align}
    \lim_{N\to\infty}a_{N,j} &=a_j^\infty,\qquad
    a_j^\infty\neq0,
\end{align}
for every \(j\), and similarly for \(B\), replacing the finite power-sum input.
For the strict source normalization above, the strict subdominant condition implies
\begin{align}
    \forall j\ge2,\ |\mu_{j}| &<1
    \Longrightarrow\ |\mu_j|^N\to0.
\end{align}
Hence for strict one-representative sectors \(a_{N,j}=(\mu_j)^N\), the required nonzero limit is generally impossible for subdominant blocks.

\section{What is formally present}

The normalization has now been added as an explicit field in
\leanid{IsNormalCanonicalFormBNT}
at index \(0\)/\(1\) depending on convention.
\begin{align}
    |\mu_0|             &=1,\\
    \forall k,\ |\mu_k| &\le|\mu_0|.
\end{align}
is explicit.

The limit-at-infinity coefficient package has been removed; the boundary now keeps only the block-permutation and gauge-phase comparison pending the missing source-faithful residual comparison lemma.

\section{Blockwise equations that should be formalized}

The intended faithful route is:
\begin{align}
    V_a^{(N)} &=X_N V_b^{(N)}
\end{align}
with coefficient-by-coefficient input from finite power sums at either one witness length (strict one-representative case) or finitely many bounded lengths in the multi-multiplicity case.
In either case, the decisive algebra is finite identities of the form
\begin{align}
    \sum_{q=1}^{r_{a,j}}\mu_{j,q}^N\stackrel{N\in\mathcal N_j}{=}\sum_{q=1}^{r_{b,j}}(\nu_{j,q}e^{i\phi_j})^N.
\end{align}
From this one gets:
\begin{align}
    r_{a,j} &=r_{b,j},\qquad \mu_{j,q}=\nu_{j,q}e^{i\phi_j},
\end{align}
From this one gets \(B_j=e^{i\phi_j}Y_jA_{\sigma(j)}Y_j^{-1}\).

\section{Source cross-check}

The source uses the finite-length step from
\cite[Lemma~\leanid{Lem:app\_simple}]{Cirac2016MPDO_arXiv},
not a global limit-on-\(N\) nonzero scalar hypothesis.
