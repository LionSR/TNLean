\section{Notation}

For a tensor $A=(A_i)_{i=1}^d$ with bond dimension $D$, write
\begin{align}
  \Gamma_N^A(X)(i_1,\ldots,i_N)
    =\tr(A_{i_1}\cdots A_{i_N}X)
    \qquad (X\in M_D(\mathbb C)).
\end{align}
Set
\begin{align}
  \mathcal G_N^A
  =
  \{\Gamma_N^A(X):X\in M_D(\mathbb C)\},
  \qquad
  V^{(N)}(A)=\Gamma_N^A(I).
\end{align}
For the cyclic chain, with $\operatorname{res}_{s,L}$ denoting restriction to
the length-$L$ interval beginning at $s\in \mathbb Z/N\mathbb Z$, the local
ground-space condition is
\begin{align}
  \operatorname{ChainGround}_{N,L}(A)
  =
  \bigcap_{s\in \mathbb Z/N\mathbb Z}
  \{\psi:\operatorname{res}_{s,L}(\psi)\in\mathcal G_L^A\}.
\end{align}

\section{The source assertion}

Cirac et al. describe the degenerate parent-Hamiltonian ground space for a
matrix product state with more than one block in canonical form in Section~IV.C
of \cite{Cirac2021Matrix}.\footnote{For
traceability, this note corresponds to
\href{https://github.com/LionSR/TNLean/issues/905}{issue \#905} and
\href{https://github.com/LionSR/TNLean/issues/870}{issue \#870}. The relevant
source lines are \path{Papers/2011.12127/TN-Review-main.tex}:2112--2128.}
The source's block-injective canonical form is a block-diagonal
physical--virtual correspondence. If
\begin{align}
  \widetilde A^i=\bigoplus_{j=1}^g A_j^i,
\end{align}
then
\begin{align}
  \forall X=(X_j)_j\in\bigoplus_{j=1}^g M_{D_j}(\mathbb C),
  \qquad
  \exists (c_i(X))_i,\qquad
  X=\sum_i c_i(X)\widetilde A^i.
\end{align}
After blocking, the corresponding word-span statement is
\begin{align}
  \exists m_0\quad \forall m\ge m_0,\qquad
  \operatorname{span}\{(A_1^\omega,\ldots,A_g^\omega):|\omega|=m\}
  =
  \prod_{j=1}^g M_{D_j}(\mathbb C).
\end{align}

The resulting parent-Hamiltonian theorem in \cite[Section~IV.C]{Cirac2021Matrix}
is
\begin{align}
  N\ge L_0+1
  \quad\Longrightarrow\quad
  \ker(H_N)=\operatorname{span}\{V^{(N)}(A_j):j=1,\ldots,g\}.
\end{align}
The sentence preceding the theorem fixes the boundary form used in the proof:
\begin{align}
  X=\bigoplus_{j=1}^g X_j,
  \qquad
  Y=\bigoplus_{j=1}^g Y_j
  \qquad
  (X_j,Y_j\in M_{D_j}(\mathbb C)).
\end{align}

\section{The block-diagonal intersection identity}

The older representation paper \cite[Theorem~\texttt{2blocks.2}]{PerezGarcia2007MPSReps}
gives the finite-chain identity used by the source argument.\footnote{Source
location: \path{Papers/quant-ph_0608197/MPSarchive.tex}:1424--1456.} It fixes
the local length
\begin{align}
  L\ge 3(b-1)(L_0+1)+1.
\end{align}
Under this hypothesis, for $m\ge L$,
\begin{align}
  \mathbb C^d\otimes\left(\bigoplus_j \mathcal G_m^{A_j}\right)
  \cap
  \left(\bigoplus_j \mathcal G_m^{A_j}\right)\otimes\mathbb C^d
  =
  \bigoplus_j \mathcal G_{m+1}^{A_j}.
\end{align}
For a vector $\psi$ in the left-hand side,
\begin{align}
  \psi
  &=
  \sum_j
  \sum_{i_1,\ldots,i_{m+1}}
  \tr\!\left(A^j_{i_{m+1}} C^j_{i_1} A^j_{i_2}\cdots A^j_{i_m}\right)
  |i_1\cdots i_{m+1}\rangle \notag\\
  &=
  \sum_j
  \sum_{i_1,\ldots,i_{m+1}}
  \tr\!\left(D^j_{i_{m+1}} A^j_{i_1} A^j_{i_2}\cdots A^j_{i_m}\right)
  |i_1\cdots i_{m+1}\rangle.
\end{align}
The direct-sum separation lemma and the one-block injectivity condition identify,
for every $j,i_1,i_{m+1}$,
\begin{align}
  A^j_{i_{m+1}} C^j_{i_1}
  =
  D^j_{i_{m+1}} A^j_{i_1}.
\end{align}
With
\begin{align}
  E^j=\sum_{i_1} C^j_{i_1}A^j_{i_1},
\end{align}
and the normalization
\begin{align}
  \sum_{i_1} A^j_{i_1}A^{j\dagger}_{i_1}=I,
\end{align}
the same proof concludes
\begin{align}
  A^j_{i_{m+1}} C^j_{i_1}
  =
  A^j_{i_{m+1}} E^j A^j_{i_1},
\end{align}
which puts the vector in $\bigoplus_j \mathcal G_{m+1}^{A_j}$.

\section{The formal boundary}

The live blueprint records an analogous statement in source notation, without
claiming that it is formalized.\footnote{Blueprint locations:
\path{blueprint/src/chapter/ch14_parent_hamiltonian.tex}:3114--3124 for the
parent ground-space definition, 3190--3228 for the block lift, 3829--3843 for the
direct-sum boundary formula, and 3972--4004 for the final span statement.} The
remaining statement is the block-diagonal boundary-condition equality
\begin{align}
  \left[
    (\forall j,\ \mu_j\ne 0)
    \ \wedge\
    L\ge 3(b-1)(L_0+1)+1
    \ \wedge\
    N\ge L+L_0
  \right]
  \Longrightarrow
  \notag\\
  \operatorname{ChainGround}_{N,L}\!\left(\bigoplus_j\mu_j A_j\right)
  =
  \sum_j \operatorname{ChainGround}_{N,L}(A_j),
\end{align}
with the right side interpreted as the physical subspace generated by the
individual block chain-ground spaces. The corresponding blueprint target records
the weight hypothesis but weaker length hypotheses:
\begin{align}
  \forall j,\ \mu_j\ne 0,
  \qquad
  1<L,
  \qquad
  N\ge L+1.
\end{align}
For
\begin{align}
  \widetilde A_i=\bigoplus_j \mu_j A_i^j,
  \qquad
  X=\bigoplus_j X_j,
\end{align}
the boundary-matrix formula is
\begin{align}
  \Gamma_N^{\widetilde A}(X)(i_1,\ldots,i_N)
  &=
  \sum_j \mu_j^N
  \tr(A^j_{i_1}\cdots A^j_{i_N}X_j) \notag\\
  &=
  \sum_j \mu_j^N\Gamma_N^{A_j}(X_j)(i_1,\ldots,i_N).
\end{align}
After the block-diagonal intersection identity is iterated around the periodic
chain, the one-block parent-Hamiltonian uniqueness theorem turns each summand
into a scalar multiple of $V^{(N)}(A_j)$.

Earlier issue descriptions and comments mention names from an older formal
formulation.\footnote{Examples include
\leanid{parentHamiltonianGroundSpace}, \leanid{bntSpan}, and
\leanid{parentHamiltonianGroundSpace_le_bntSpan_of_block_decomposition}.}
Those names are absent from current main. They should be read only as historical
labels for the same mathematical target, not as declarations to be restored
before the block-diagonal theorem is proved. The surviving
\leanid{parentHamiltonianGroundSpaceES} declaration belongs to the energy-space
martingale gap development, not to this BNT block-decomposition statement.

\section{Conclusion}

This is not a source-error claim. The classification is an open gap: the
block-diagonal intersection identity and its periodic-chain iteration have not
yet been proved in Lean. Once that theorem is available, issue \#905 supplies
the structural decomposition needed for issue \#870, and the final degenerate
parent-Hamiltonian ground-space equality follows by applying the one-block
uniqueness theorem to each block.
