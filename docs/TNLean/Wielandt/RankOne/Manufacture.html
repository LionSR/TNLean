    Manufacturing a rank-one matrix in the range of two-sided multiplication #

    This file provides a small helper lemma used in the rank-one step of the Wielandt proof:

    If a column vector φ lies in the range of left-multiplication by P (as a linear map on vectors), and a row vector ψ lies in the range of right-multiplication by Q (again as a linear map on vectors), then the rank-one matrix Matrix.vecMulVec φ ψ lies in the range of the (two-sided) linear map X ↦ P * X * Q.
