    Rank-one element construction step (partial) #

    This file provides a partial step towards Wielandt Lemma 2(b): constructing a bounded-length element of wordSpan whose image lies in the "invertible block" (the direct sum of generalized eigenspaces for nonzero (eigen-)values).

    In the Jordan-form proof of arXiv:0909.5347 Lemma 2(b), this corresponds to the construction of an exponent r such that A₁^r = A₁^r P, where P projects onto the generalized eigenspaces for nonzero eigenvalues.

    We do not yet construct a rank-one element, but we construct a nonzero element in bounded wordSpan that kills the zero generalized eigenspace.

    Word-span membership for powers of a word matrix #

    theorem MPSTensor.evalWord_pow_mem_wordSpan {d D : ℕ} (A : MPSTensor d D) (w : List (Fin d)) (k : ℕ) :
    A.evalWord w ^ k ∈ A.wordSpan (k * w.length)

    If M = evalWord A w, then the matrix power M ^ k lies in the fixed-length word span at length k * w.length.

    This is a small helper for later bounded-length constructions.

    theorem MPSTensor.pow_mulVec_eq_smul_of_mulVec_eq_smul {D : ℕ} (M : Matrix (Fin D) (Fin D) ℂ) (φ : Fin D → ℂ) (μ : ℂ) (heig : M.mulVec φ = μ • φ) (k : ℕ) :
    (M ^ k).mulVec φ = μ ^ k • φ

    If M *ᵥ φ = μ • φ, then applying powers of M to φ scales by powers of μ.

    This is the basic eigenvector/power identity used to prove nontriviality of word powers.

    Linear-algebraic "kill the nilpotent block" lemma #

    source
    theorem MPSTensor.WielandtRankOne.range_pow_le_iSup_maxGenEigenspace_ne_zero {D : ℕ} (f : Module.End ℂ (Fin D → ℂ)) :
    LinearMap.range (f ^ D) ≤ ⨆ (μ : ℂ), ⨆ (_ : μ ≠ 0), f.maxGenEigenspace μ

    The range of f ^ D is contained in the sum of generalized eigenspaces for nonzero (eigen-)values.

    This is the "kill the nilpotent block" step: the power f ^ D annihilates the zero generalized eigenspace and therefore maps everything into the direct sum of nonzero generalized eigenspaces.

    The key ingredient is the generalized eigenspace decomposition ⨆ μ, maxGenEigenspace f μ = ⊤ over an algebraically closed field.

    A bounded wordSpan element killing the nilpotent block #

    source
    theorem MPSTensor.exists_nonzero_pow_evalWord_mem_wordSpan_range_le {d D : ℕ} (A : MPSTensor d D) (w₀ : List (Fin d)) (μ : ℂ) (φ : Fin D → ℂ) (hμ : μ ≠ 0) (hφ : φ ≠ 0) (heig : (A.evalWord w₀).mulVec φ = μ • φ) :
    ∃ P ∈ A.wordSpan (D * w₀.length), P ≠ 0 ∧ (Matrix.toLin' P).range ≤ ⨆ (ν : ℂ), ⨆ (_ : ν ≠ 0), Module.End.maxGenEigenspace (Matrix.toLin' (A.evalWord w₀)) ν

    Nonzero bounded word-span element whose range lies in the nonzero generalized eigenspaces.

    Let M := evalWord A w₀. If M has an eigenvector φ ≠ 0 with eigenvalue μ ≠ 0, then the power M ^ D:

    • lies in the fixed-length word span wordSpan A (D * w₀.length),
    • is nonzero, and
    • has image contained in the sum of generalized eigenspaces of Matrix.toLin' M for nonzero eigenvalues.

    This matches the Jordan-form step A₁^r = A₁^r P in arXiv:0909.5347 Lemma 2(b).

    It is still weaker than the missing rank-one construction.
