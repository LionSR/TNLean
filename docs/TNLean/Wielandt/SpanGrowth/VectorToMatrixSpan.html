    Lemma 2(b) (start): from vector spanning to fixed-length matrix spanning #

    This module starts closing the main remaining gap in the Quantum Wielandt proof (arXiv:0909.5347, Lemma 2(b)).

    We currently formalize the algebraic assembly part of Lemma 2(b):

    • If (fixed-length) word products applied to a vector φ span all of ℂ^D, and
    • If we can produce the rank-one operators |φ⟩⟨e_j| as word products of a fixed length,

    then word products of a (longer) fixed length span all matrices.

    What remains for the full lemma is the construction of these rank-one operators from the eigenvalue/Fitting decomposition analysis.

    Main results #

    • MPSTensor.wordSpan_mul_le — fixed-length word spans multiply into the span at the summed length
    • MPSTensor.wordSpan_top_of_mul — full word span persists at positive multiples
    • MPSTensor.wordSpan_eq_top_of_blockTensor_wordSpan_eq_top — full blocked word span gives a full unblocked word span

      Eigenvector padding: cumulative vector span → fixed-length vector span #

      To use fixed-length (wordSpan) arguments downstream, we often need a fixed-length version of the vector span:

      • cumulativeVectorSpan A φ n is spanned by words of length ≤ n.
      • vectorSpreadSpan A φ n is spanned by words of length exactly n.

      If one Kraus operator has an eigenvector φ with eigenvalue μ ≠ 0, we can pad any shorter word by appending copies of this Kraus operator, turning a ≤-length statement into an exact-length statement.

      source
      theorem MPSTensor.cumulativeVectorSpan_le_vectorSpreadSpan_of_eigenvector {d D : ℕ} (A : MPSTensor d D) (φ : Fin D → ℂ) (n : ℕ) (i₀ : Fin d) (μ : ℂ) (hμ : μ ≠ 0) (heig : (A i₀).mulVec φ = μ • φ) :
      A.cumulativeVectorSpan φ n ≤ A.vectorSpreadSpan φ n

      If A i₀ *ᵥ φ = μ • φ with μ ≠ 0, then any shorter word action can be padded up to length n without changing the spanned subspace (up to a nonzero scalar).

      Concretely, this proves cumulativeVectorSpan A φ n ≤ vectorSpreadSpan A φ n.

      Vector spanning → fixed-length matrix spanning (assembly step) #

      source
      theorem MPSTensor.wordSpan_eq_top_of_vectorSpreadSpan_eq_top_of_rankOneBasis {d D : ℕ} (A : MPSTensor d D) (φ : Fin D → ℂ) {n m : ℕ} (hVec : A.vectorSpreadSpan φ n = ⊤) (hRankOne : ∀ (j : Fin D), Matrix.vecMulVec φ (Pi.single j 1) ∈ A.wordSpan m) :
      A.wordSpan (n + m) = ⊤

      Lemma 2(b) (assembly step, rank-one hypothesis).

      Assume:

      • vectorSpreadSpan A φ n = ⊤, i.e. length-n word products applied to φ span all of ℂ^D.
      • For each basis vector e_j, the rank-one operator |φ⟩⟨e_j| (implemented as Matrix.vecMulVec φ (Pi.single j 1)) lies in wordSpan A m.

      Then wordSpan A (n+m) = ⊤.

      This is the part of Lemma 2(b) that turns rank-one operators + vector spanning into full matrix spanning.

      The remaining hard part (TODO) is to construct these rank-one operators from the Jordan/Fitting analysis of an eigenvalue word.
