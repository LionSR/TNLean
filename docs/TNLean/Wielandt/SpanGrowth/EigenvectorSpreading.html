    Eigenvector Spreading (Lemma 2(a)) #

    This file formalizes Lemma 2(a) of arXiv:0909.5347 (Sanz, Pérez-García, Wolf, Cirac).

    Paper statement: Let A = {A₁, ..., A_d} be Kraus operators for a primitive channel on M_D. Suppose A₁|φ⟩ = μ|φ⟩ with μ ≠ 0. Define H_n(A, φ) = span{A_{i₁}···A_{iₙ}|φ⟩ : all words of length n}, K_n(A, φ) = span(⋃{m≤n} H_m(A, φ)). Then K{D-1}(A, φ) = ℂ^D.

    Definitions #

    • vectorSpreadSpan A φ n — H_n(A, φ), the span of all images of φ under length-n word products
    • cumulativeVectorSpan A φ n — K_n(A, φ), the cumulative vector span (all word products of length ≤ n applied to φ)

    Main results #

    • cumulativeVectorSpan_mono: K_n ≤ K_{n+1}
    • cumulativeVectorSpan_stable: If K_n = K_{n+1} then K_m = K_n for m ≥ n
    • eigenvector_mem_cumulativeVectorSpan: The eigenvector anchoring trick
    • cumulativeVectorSpan_finrank_le: dim(K_n) ≤ D
    • cumulativeVectorSpan_finrank_strict_mono: strict inclusion → strict dim growth
    • eigenvector_spreading: The main theorem K_{D-1} = ⊤

    References #

    • Sanz, Pérez-García, Wolf, Cirac, A quantum version of Wielandt's inequality, arXiv:0909.5347, Lemma 2(a)

        Connection between IsNormal and cumulativeVectorSpan reaching ⊤ #

        source
        theorem MPSTensor.cumulativeVectorSpan_eq_top_of_cumulativeSpan_eq_top {d D : ℕ} (A : MPSTensor d D) (φ : Fin D → ℂ) (hφ : φ ≠ 0) {N : ℕ} (htop : A.cumulativeSpan N = ⊤) :
        A.cumulativeVectorSpan φ N = ⊤

        If cumulativeSpan A N = ⊤ (all D×D matrices are linear combinations of word products), then cumulativeVectorSpan A φ N = ⊤ for any nonzero φ.

        The argument: since every matrix is a linear combination of evalWord A w, the standard basis matrix Matrix.single j k 1 is in the span. Then (Matrix.single j k 1) *ᵥ φ = φ k • e_j is in K_N. Since φ ≠ 0, some φ k ≠ 0, so e_j ∈ K_N for all j, hence K_N = ⊤.

        source
        theorem MPSTensor.eigenvector_spreading {d D : ℕ} [NeZero D] (A : MPSTensor d D) (φ : Fin D → ℂ) (hφ : φ ≠ 0) (_i₀ : Fin d) (_μ : ℂ) (_hμ : _μ ≠ 0) (_heig : (A _i₀).mulVec φ = _μ • φ) (hNormal : A.IsNormal) :
        A.cumulativeVectorSpan φ (D - 1) = ⊤

        Lemma 2(a): If the channel is normal and A has an eigenvector with nonzero eigenvalue, then K_{D-1}(A, φ) = ℂ^D.

        Paper: H_{D-1}(A, φ) = ℂ^D (arXiv:0909.5347, Lemma 2(a)).

        Deviation: we prove the cumulative conclusion K_{D-1} = ⊤ instead of the paper's single-level conclusion H_{D-1} = ⊤. This is enough for every downstream use in this formalization.

        The eigenvector data are kept in the statement for API compatibility with the paper and with downstream wrappers. The current cumulative proof uses only hφ together with the normality witness.
