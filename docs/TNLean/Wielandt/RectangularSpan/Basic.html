    Rectangular Span Foundations and Lemma 2(b) Assembly #

    This module contains the foundational rectangular-span API used in the Wielandt bound: blocking preserves normality, blocked eigenvector transfer, the basic one-sided and cumulative rectangular spans, and the conditional and blocked assembly steps for Lemma 2(b).

    The later growth, stabilization, universality, and sharp quantitative backends live in TNLean.Wielandt.RectangularSpan.Growth and TNLean.Wielandt.RectangularSpan.Universality.

    Section 2: Blocking preserves normality #

    source
    theorem MPSTensor.wordSpan_le_wordSpan_blockTensor {d D : ℕ} (A : MPSTensor d D) (L n : ℕ) :
    A.wordSpan (n * L) ≤ (A.blockTensor L).wordSpan n

    Every generator of wordSpan A (n * L) lies in wordSpan (blockTensor A L) n.

    Proof by induction on n: each word of length (n+1)*L factors into a block of size L (giving a blocked Kraus operator) and a remainder of size n*L (handled by the inductive hypothesis).

          theorem MPSTensor.wielandt_lemma2b_conditional {d D : ℕ} [NeZero D] (A : MPSTensor d D) (hNormal : A.IsNormal) (i₀ : Fin d) (μ : ℂ) (hμ : μ ≠ 0) (φ : Fin D → ℂ) (hφ : φ ≠ 0) (heigφ : (A i₀).mulVec φ = μ • φ) (i₁ : Fin d) (ν : ℂ) (hν : ν ≠ 0) (ψ : Fin D → ℂ) (hψ : ψ ≠ 0) (heigψ : (A i₁).transpose.mulVec ψ = ν • ψ) {m : ℕ} (hRankOne : Matrix.vecMulVec φ ψ ∈ A.wordSpan m) :
          A.wordSpan (D - 1 + (m + (D - 1))) = ⊤

          Lemma 2(b) conditional assembly.

          If IsNormal A, and we have single-index eigenvectors (column and row) and a rank-one element in bounded wordSpan, then wordSpan = ⊤.

          Combines eigenvector spreading, row spreading, and rank-one reduction.

          Section 7: Blocked assembly #

          source
          theorem MPSTensor.wielandt_blocked_assembly {d D : ℕ} [NeZero D] (A : MPSTensor d D) (hNormal : A.IsNormal) (L : ℕ) (hL : 0 < L) (σ₀ : Fin L → Fin d) (φ : Fin D → ℂ) (hφ : φ ≠ 0) (μ : ℂ) (hμ : μ ≠ 0) (heigφ : (A.evalWord (List.ofFn σ₀)).mulVec φ = μ • φ) (τ₀ : Fin L → Fin d) (ψ : Fin D → ℂ) (hψ : ψ ≠ 0) (ν : ℂ) (hν : ν ≠ 0) (heigψ : (A.evalWord (List.ofFn τ₀)).transpose.mulVec ψ = ν • ψ) {m_blocked : ℕ} (hRankOne : Matrix.vecMulVec φ ψ ∈ (A.blockTensor L).wordSpan m_blocked) :
          A.wordSpan ((D - 1 + (m_blocked + (D - 1))) * L) = ⊤

          Assembly at the blocked level.

          Reduces the Wielandt bound to producing a rank-one element in the word span of the blocked tensor. The blocking period L absorbs the word lengths of both the column and row eigenvectors.

          Conclusion: #

          wordSpan A ((D - 1 + (m_blocked + (D - 1))) * L) = ⊤.
