    Rectangular ranges for matrix multiplication #

    This file collects basic linear-algebra facts about the range of the linear map X ↦ P * X on the matrix algebra Matrix (Fin D) (Fin D) ℂ.

    The key point is that left multiplication acts independently on columns: (P * X).col j = P *ᵥ (X.col j).

    As a consequence:

    • range (mulLeft P) is exactly the submodule of matrices whose columns lie in range (Matrix.toLin' P).
    • The range has dimension D * rank(P).

    These lemmas are intended for the dimension-growth step in Wielandt Lemma 2(b).
