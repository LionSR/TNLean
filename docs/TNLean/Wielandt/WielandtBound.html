    Wielandt cumulative chain #

    This file records the cumulative Wielandt chain: the four standard consequences of normality that feed into the fixed-length Wielandt bound (Theorem 1 of arXiv:0909.5347).

    The Theorem 1 statements (case (1), (2), (3) bounds on the Kraus-rank index $i(A)$) are in Inequality/Bounds.lean. This file only provides the cumulative-span and eigenvector-spreading inputs needed by the Proposition 3(c)→(b) proof in Primitivity/Normal.lean.

    Main result #

    • wielandt_chain: under IsNormal A, simultaneously:
      1. cumulativeSpan A (D ^ 2) = ⊤
      2. $\exists w_0,\ |w_0| \le D^2 \land \operatorname{tr}(A^{w_0}) \ne 0$
      3. $\exists w_0, \mu \ne 0, \varphi \ne 0$ with $|w_0| \le D^2$ and $A^{w_0}\varphi = \mu\varphi$
      4. $\forall \varphi \ne 0,\ \operatorname{cumulativeVectorSpan} A \varphi (D^2) = \top$

    References #

    • Sanz, Pérez-García, Wolf, Cirac, A quantum version of Wielandt's inequality, arXiv:0909.5347

    The cumulative Wielandt chain #

    The four standard consequences of normality. These are combined below in one theorem so that Primitivity/Normal.lean gets all outputs at once.

    source
    theorem MPSTensor.wielandt_chain {d D : ℕ} [NeZero D] (A : MPSTensor d D) (hN : A.IsNormal) :
    A.cumulativeSpan (D ^ 2) = ⊤ ∧ (∃ (w₀ : List (Fin d)), w₀.length ≤ D ^ 2 ∧ (A.evalWord w₀).trace ≠ 0) ∧ (∃ (w₀ : List (Fin d)) (μ : ℂ) (φ : Fin D → ℂ), w₀.length ≤ D ^ 2 ∧ μ ≠ 0 ∧ φ ≠ 0 ∧ (A.evalWord w₀).mulVec φ = μ • φ) ∧ ∀ (φ : Fin D → ℂ), φ ≠ 0 → A.cumulativeVectorSpan φ (D ^ 2) = ⊤

    The cumulative Wielandt chain: under IsNormal A, the four standard cumulative consequences hold simultaneously.

    1. Cumulative span T_{D²}(A) = M_D(ℂ) (Lemma 1, cumulative version)
    2. A word w₀ of length ≤ D² with nonzero trace
    3. A nonzero eigenvalue μ ≠ 0 and eigenvector φ ≠ 0 for evalWord A w₀
    4. For any nonzero φ, the cumulative vector span reaches ⊤ in D² steps

    Paper: arXiv:0909.5347, proof of Theorem 1, before the fixed-length upgrade in Lemma 2(b).

    The fixed-length bounds on i(A) (cases (1), (2), (3)) are in Inequality/Bounds.lean.
