    Brouwer fixed-point theorem on density matrices #

    This module derives Brouwer's fixed-point theorem for the compact convex set of density matrices from the general retract-based fixed-point theorem proved in TNLean.Topology.CompactRetractFixedPoint and the explicit density-matrix retraction from TNLean.Channel.DensityRetract.

    source
    theorem brouwer_fixedPoint_densityMatrices {D : ℕ} [NeZero D] {f : Matrix (Fin D) (Fin D) ℂ → Matrix (Fin D) (Fin D) ℂ} (hf_cont : ContinuousOn f (densityMatrices D)) (hf_map : Set.MapsTo f (densityMatrices D) (densityMatrices D)) :
    ∃ ρ ∈ densityMatrices D, f ρ = ρ

    Brouwer fixed point theorem on density matrices.

    If f is continuous on the set of density matrices and maps density matrices to density matrices, then it has a fixed point in the set of density matrices.
