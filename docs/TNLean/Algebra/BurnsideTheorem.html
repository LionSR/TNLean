    Burnside theorem for matrix algebras #

    This file proves the Burnside/Jacobson density theorem for the subalgebra of Matrix (Fin D) (Fin D) ℂ generated by the Kraus operators of an MPS tensor.

    The main result is MPSTensor.burnside_matrix: if a family of matrices A : MPSTensor d D acts irreducibly on ℂ^D (no nontrivial invariant submodule), then the unital subalgebra they generate is all of M_D(ℂ).

    The proof follows the standard strategy:

    • transport matrices to Module.End ℂ (Fin D → ℂ) via Matrix.toLinAlgEquiv',
    • use IsIrreducibleAction to obtain a simple module,
    • apply Schur's lemma over the algebraically closed field ℂ,
    • use Jacobson density (Module.Finite.toModuleEnd_moduleEnd_surjective) to conclude surjectivity onto all endomorphisms.

          Burnside's theorem (Jacobson density + Schur) #

          source
          theorem MPSTensor.burnside_matrix {d D : ℕ} [NeZero D] (A : MPSTensor d D) (hIrr : A.IsIrreducibleAction) :
          A.algSpan = ⊤

          Burnside/Jacobson density theorem (matrix form).

          If the Kraus operators of A : MPSTensor d D act irreducibly on ℂ^D (in the sense IsIrreducibleAction), then the unital subalgebra they generate is all of M_D(ℂ).

          Implementation outline:

          • Transport algSpan A along Matrix.toLinAlgEquiv' to a subalgebra of Module.End ℂ (Fin D → ℂ).
          • IsIrreducibleAction gives a simple module.
          • Schur (over an algebraically closed field) identifies the commuting endomorphisms with scalars.
          • Jacobson density then forces surjectivity onto all endomorphisms.
