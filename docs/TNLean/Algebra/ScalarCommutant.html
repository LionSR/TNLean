    Scalar commutant lemma: center of M_n(R) = R·1 #

    If a matrix Z commutes with every element of a spanning set of M_n(R), then Z is a scalar matrix. This is the algebraic fact that the center of the full matrix algebra over a commutative ring is trivial.

    Main results #

    • Matrix.isScalar_of_commute_span_eq_top: if Z commutes with a spanning set, then Z = scalar n c for some c.

    A matrix that commutes with a spanning set of M_n(R) lies in the center, hence is a scalar matrix.
