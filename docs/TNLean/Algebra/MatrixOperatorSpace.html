    Matrix operator-space infrastructure #

    This module centralizes the operator-norm and restricted-scalar infrastructure used for square complex matrices and their continuous linear endomorphisms.

    Main declarations #

    • TNLean.MatrixCLM — continuous complex endomorphisms of square matrices
    • TNLean.matrixEndEquiv — the finite-dimensional equivalence between linear and continuous matrix endomorphisms
    • TNOperatorSpace scoped instances — operator-norm, real-scalar, and finite-dimensional structure shared across the semigroup and channel files
