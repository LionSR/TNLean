    Auxiliary matrix lemmas #

    General-purpose matrix lemmas that are not specific to any chapter's theory. Extracted from various files for reusability.

    Main results #

    • Matrix.sum_mul_mul: pull fixed left/right factors through a finite sum
    • Matrix.dim_le_of_mulVec_injective: injective mulVec implies dimension bound
    • Matrix.PosSemidef.mulVec_eq_zero_left/right: kernel containment for PSD matrix sums
    • Matrix.trace_eq_of_charpoly_eq: equal characteristic polynomials imply equal traces

    ∑ i : Fin d, L * M i * R = (L * ∑ i : Fin d, M i) * R

    Pull fixed left and right matrix factors through a finite sum indexed by Fin d.

    theorem Matrix.dim_le_of_mulVec_injective {D₁ D₂ : ℕ} [NeZero D₂] (X : Matrix (Fin D₁) (Fin D₂) ℂ) (h_inj : ∀ (v : Fin D₂ → ℂ), X.mulVec v = 0 → v = 0) :
    D₂ ≤ D₁

    If multiplication by a rectangular matrix has trivial kernel, then the source dimension is at most the target dimension.

    Kernel intersection for PSD matrices #

    source
    theorem Matrix.PosSemidef.mulVec_eq_zero_left {D : ℕ} {A B : Matrix (Fin D) (Fin D) ℂ} (hA : A.PosSemidef) (hB : B.PosSemidef) (v : Fin D → ℂ) (hv : (A + B).mulVec v = 0) :
    A.mulVec v = 0

    For PSD matrices A and B, ker(A + B) ⊆ ker(A). Proof: v†(A+B)v = v†Av + v†Bv = 0 with both nonneg implies v†Av = 0.

    Characteristic polynomial lemmas #

    source
    theorem Matrix.trace_eq_of_charpoly_eq {D : ℕ} [NeZero D] (T U : Matrix (Fin D) (Fin D) ℂ) (h : T.charpoly = U.charpoly) :
    T.trace = U.trace

    If two square matrices over ℂ have the same characteristic polynomial, they have the same trace. This follows from the fact that the trace is the negation of the next-to-leading coefficient of the characteristic polynomial.
