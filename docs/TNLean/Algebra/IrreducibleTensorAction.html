    Irreducible tensor implies irreducible action #

    This file closes the missing direction

    IsIrreducibleTensor A → IsIrreducibleAction A.

    The key idea is classical: if W ≤ ℂ^D is a nontrivial invariant subspace, then the orthogonal projection onto W (in the finite-dimensional Hilbert space EuclideanSpace ℂ (Fin D)) is a nontrivial Hermitian idempotent matrix P satisfying (1 - P) * A i * P = 0 for all i. This produces HasInvariantProj A, contradicting IsIrreducibleTensor A.
