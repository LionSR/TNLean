    Hermitian matrix helpers #

    This file provides small reusable helpers for Hermitian complex matrices over Fin D: the unitary identities for the eigenvector matrix, the standard U * diagonal * Uᴴ spectral decomposition, and the adjoint identity for the matrix-vector dot-product pairing.

    source
    theorem Matrix.IsHermitian.spectral_decomp_eq_of_generalIndex {n : Type u_1} [Fintype n] [DecidableEq n] {M : Matrix n n ℂ} (hM : M.IsHermitian) :
    M = (↑hM.eigenvectorUnitary * diagonal fun (j : n) => ↑(hM.eigenvalues j)) * (↑hM.eigenvectorUnitary).conjTranspose

    Spectral decomposition of a Hermitian complex matrix (polymorphic in the index type): M = U * diagonal λ * Uᴴ where U = hM.eigenvectorUnitary and λ = hM.eigenvalues are the eigenvector unitary and real eigenvalues provided by Mathlib's Hermitian spectral theorem.

    theorem eig_conj_mul {D : ℕ} [DecidableEq (Fin D)] {M : Matrix (Fin D) (Fin D) ℂ} (hM : M.IsHermitian) :
    (↑hM.eigenvectorUnitary).conjTranspose * ↑hM.eigenvectorUnitary = 1

    Uᴴ * U = 1 for the eigenvector unitary of a Hermitian matrix.
