    Tripartite partial traces preserve the full trace #

    This file records the elementary trace-preservation identities for the tripartite partial traces introduced in TNLean.Analysis.Entropy.

    Main declarations #

    • Matrix.trace_eq_trace_traceA_ABC
    • Matrix.trace_eq_trace_traceC_ABC
    • Matrix.trace_eq_trace_traceAC_ABC

    Each theorem is proved by unfolding the relevant partial trace and reordering the resulting finite sums.

    References #

    • [M. Wolf, Quantum Channels & Operations: Guided Tour][Wolf2012QChannels]
    source
    theorem Matrix.trace_eq_trace_traceA_ABC {dA dB dC : ℕ} (ρ : Matrix (Fin dA × Fin dB × Fin dC) (Fin dA × Fin dB × Fin dC) ℂ) :
    ρ.trace = ρ.traceA_ABC.trace

    The full trace equals the trace of the A-partial trace: tr(ρ_ABC) = tr(tr_A(ρ_ABC)).
