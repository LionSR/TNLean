    def MPSTensor.MPVBlockPhaseEquiv {d DA DB : ℕ} (A : MPSTensor d DA) (B : MPSTensor d DB) :
    Prop

    MPV phase equivalence between two individual blocks with possibly different bond dimensions.

    MPVBlockPhaseEquiv A B means that the finite-chain MPV of B is a nonzero scalar power times the finite-chain MPV of A, uniformly in the chain length and physical word. The bond dimensions of A and B may differ.

    Equations
    • A.MPVBlockPhaseEquiv B = ∃ (ζ : ℂ), ζ ≠ 0 ∧ ∀ (N : ℕ) (σ : Fin N → Fin d), B.mpv σ = ζ ^ N * A.mpv σ

      structure MPSTensor.MPVCommonPhaseCover {d rA rB : ℕ} {dimA : Fin rA → ℕ} {dimB : Fin rB → ℕ} (blocksA : (j : Fin rA) → MPSTensor d (dimA j)) (blocksB : (k : Fin rB) → MPSTensor d (dimB k)) :
      Type

      Common MPV-phase cover for two block families.

      The data consist of the finite common family, the class maps from the two block families to it, the MPV-phase equivalences between each block and its chosen common block, and surjectivity of both class maps. These are the CPSV comparison data needed by the span comparison result; constructing them from the full structural SameMPV₂ data is a separate comparison step.

      • rC : ℕ
      • dimC : Fin self.rC → ℕ
      • common (c : Fin self.rC) : MPSTensor d (self.dimC c)
      • classA : Fin rA → Fin self.rC
      • classB : Fin rB → Fin self.rC
      • phaseA (j : Fin rA) : (self.common (self.classA j)).MPVBlockPhaseEquiv (blocksA j)
      • phaseB (k : Fin rB) : (self.common (self.classB k)).MPVBlockPhaseEquiv (blocksB k)
      • surjA : Function.Surjective self.classA

        def MPSTensor.MPVPhaseEquiv {d r : ℕ} {dim : Fin r → ℕ} (blocks : (k : Fin r) → MPSTensor d (dim k)) (j k : Fin r) :
        Prop

        MPV phase equivalence for a dependent block family.

        MPVPhaseEquiv blocks j k means that block k has the same MPV family as block j after multiplying length-N vectors by a nonzero scalar power ζ ^ N. Gauge-phase equivalence implies this relation, and quotienting a finite family by this relation is enough to absorb all repeated scalar-power copies into sector weights.

        This is the family-indexed specialization of MPVBlockPhaseEquiv, so the fixed-family phase-class relation shares the block-level relation for different bond dimensions rather than duplicating it.

        Equations
        • MPSTensor.MPVPhaseEquiv blocks j k = (blocks j).MPVBlockPhaseEquiv (blocks k)

              structure MPSTensor.MPVPhaseClassData {d r : ℕ} {dim : Fin r → ℕ} (blocks : (k : Fin r) → MPSTensor d (dim k)) :
              Type

              Finite class data for the MPV phase relation.

              The data consist of an enumeration of the quotient classes, a choice of representative per class, the scalar-power relation from each representative to each member, the separation property for the representatives, and the regrouping identity for finite sums over the original blocks.

              • g : ℕ
              • copies : Fin self.g → ℕ
              • copies_pos (j : Fin self.g) : 0 < self.copies j
              • enum (j : Fin self.g) : Fin (self.copies j) → Fin r
              • repr : Fin self.g → Fin r
              • enum_phase (j : Fin self.g) (q : Fin (self.copies j)) : MPVPhaseEquiv blocks (self.repr j) (self.enum j q)

              • regroup (f : Fin r → ℂ) : ∑ j : Fin self.g, ∑ q : Fin (self.copies j), f (self.enum j q) = ∑ k : Fin r, f k

                theorem MPSTensor.MPVPhaseClassData.representative_mpv_span_eq {d r : ℕ} {dim : Fin r → ℕ} {blocks : (k : Fin r) → MPSTensor d (dim k)} (classes : MPVPhaseClassData blocks) (N : ℕ) :
                Submodule.span ℂ (Set.range fun (j : Fin classes.g) => (blocks (classes.repr j)).mpvState N) = Submodule.span ℂ (Set.range fun (k : Fin r) => (blocks k).mpvState N)

                The chosen MPV phase-class representatives span exactly the same finite-length MPV subspace as the original block family.

                Each representative is one of the original blocks, giving one inclusion. Conversely, every original block occurs in a phase class and is a nonzero scalar-power multiple of that class's representative at each length, hence lies in the representative span.

                MPVPhaseClassData blocks

                Construct the finite MPV phase classes of a block family.

                The representative of each class is the first element in the finite enumeration of that class. If two representatives were gauge-phase equivalent, then they would be MPV-phase equivalent and hence lie in the same quotient class; this proves that distinct representatives are pairwise not gauge-phase equivalent.
