    Basic definitions for matrix product state tensors #

    This file contains the core definitions used throughout the MPS development: MPSTensor, word evaluation, MPV coefficients, gauge equivalence, and the notions of injectivity, block injectivity, and normality. It also proves the basic gauge-invariance lemmas for evalWord and SameMPV.

    source
    @[reducible, inline]
    abbrev MPSTensor (d D : ℕ) :
    Type

    A (periodic, translation-invariant) tensor generating an MPV family: a family of D×D matrices indexed by a physical index in Fin d.

    The name MPSTensor is kept for compatibility with the literature and the existing Lean development.

    Equations
    • MPSTensor d D = (Fin d → Matrix (Fin D) (Fin D) ℂ)

      def MPSTensor.evalWord {d D : ℕ} (A : MPSTensor d D) :
      List (Fin d) → Matrix (Fin D) (Fin D) ℂ

      Evaluate a word w = [i₁, i₂, …, iₙ] by multiplying the corresponding matrices A i₁ * A i₂ * ⋯ * A iₙ. Returns 1 for the empty word.

      Equations
      • A.evalWord [] = 1
      • A.evalWord (i :: w) = A i * A.evalWord w

        A.evalWord (w1 ++ w2) = A.evalWord w1 * A.evalWord w2

        Multiplicativity of word evaluation: evalWord A (w1 ++ w2) = evalWord A w1 * evalWord A w2.

        theorem MPSTensor.evalWord_smul {d D : ℕ} (ζ : ℂ) (A : MPSTensor d D) (w : List (Fin d)) :
        evalWord (fun (i : Fin d) => ζ • A i) w = ζ ^ w.length • A.evalWord w

        Scaling of word evaluation: scaling every matrix by a scalar ζ scales evalWord by the factor ζ ^ w.length.

          def MPSTensor.mpv {d D : ℕ} (A : MPSTensor d D) {N : ℕ} (σ : Fin N → Fin d) :
          ℂ

          The Matrix Product Vector (MPV) for system size N: for each basis state σ : Fin N → Fin d, this returns the coefficient trace (A (σ 0) * A (σ 1) * ⋯ * A (σ (N-1))).

              def MPSTensor.SameMPV {d D : ℕ} (A B : MPSTensor d D) :
              Prop

              Two tensors generate the same MPV family if they produce the same coefficient for every system size N and every basis configuration σ : Fin N → Fin d.

              Equations
              • A.SameMPV B = ∀ (N : ℕ) (σ : Fin N → Fin d), A.mpv σ = B.mpv σ

                        def MPSTensor.IsNBlkInjective {d D : ℕ} (A : MPSTensor d D) (N : ℕ) :
                        Prop

                        N-block injectivity: after blocking N sites, the set of all products A^{i₁} * ⋯ * A^{i_N} spans the full matrix algebra.

                        We index the blocked tensors by σ : Fin N → Fin d, i.e. words of length N.

                        Equations
                        • A.IsNBlkInjective N = (Submodule.span ℂ (Set.range fun (σ : Fin N → Fin d) => A.evalWord (List.ofFn σ)) = ⊤)
