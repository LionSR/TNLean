    Basic definitions for matrix product state tensors #

    This file contains the core definitions used throughout the MPS development: MPSTensor, word evaluation, MPV coefficients, gauge equivalence, and the notions of injectivity, block injectivity, and normality. It also proves the basic gauge-invariance lemmas for evalWord and SameMPV.

    source
    @[reducible, inline]
    abbrev MPSTensor (d D : ℕ) :
    Type

    A (periodic, translation-invariant) tensor generating an MPV family: a family of D×D matrices indexed by a physical index in Fin d.

    The name MPSTensor is kept for compatibility with the literature and the existing Lean development.

    Equations
    • MPSTensor d D = (Fin d → Matrix (Fin D) (Fin D) ℂ)

      def MPSTensor.evalWord {d D : ℕ} (A : MPSTensor d D) :
      List (Fin d) → Matrix (Fin D) (Fin D) ℂ

      Evaluate a word w = [i₁, i₂, …, iₙ] by multiplying the corresponding matrices A i₁ * A i₂ * ⋯ * A iₙ. Returns 1 for the empty word.

      Equations
      • A.evalWord [] = 1
      • A.evalWord (i :: w) = A i * A.evalWord w

        A.evalWord (w1 ++ w2) = A.evalWord w1 * A.evalWord w2

        Scaling of word evaluation: scaling every matrix by a scalar ζ scales evalWord by the factor ζ ^ w.length.

          The Matrix Product Vector (MPV) for system size N: for each basis state σ : Fin N → Fin d, this returns the coefficient trace (A (σ 0) * A (σ 1) * ⋯ * A (σ (N-1))).

              def MPSTensor.SameMPV {d D : ℕ} (A B : MPSTensor d D) :
              Prop

              Two tensors generate the same MPV family if they produce the same coefficient for every system size N and every basis configuration σ : Fin N → Fin d.

                  def MPSTensor.SameMPV₂Pos {d D₁ D₂ : ℕ} (A : MPSTensor d D₁) (B : MPSTensor d D₂) :
                  Prop

                  Positive-length MPV equality for possibly different bond dimensions.

                  This is useful when compressions or zero-block removals change the N = 0 coefficient but preserve all nonempty-chain coefficients. The source paper discusses "zero blocks" (arXiv:1606.00608, Section~2.3): they contribute their bond dimension at length zero and vanish from every positive-length coefficient.

                    Weak scalar proportionality of MPVs.

                      def MPSTensor.NonzeroProportionalMPV₂ {d D₁ D₂ : ℕ} (A : MPSTensor d D₁) (B : MPSTensor d D₂) :
                      Prop

                      Nonzero proportionality of MPV families.

                      Source: arXiv:1606.00608, Theorem thm1, lines 1167--1170. This is the formal reading of the source statement that two tensors generate MPV that are proportional to each other: at each length the two MPV vectors lie on the same nonzero projective line, with proportionality scalar allowed to depend on the length.

                      Local fix (projective proportionality): The source phrase "proportional to each other" is read projectively, so the scalar is nonzero. This reading is documented in docs/paper-gaps/cpsv16_nonzero_proportionality_reading.tex.

                        def MPSTensor.EventuallyNonzeroProportionalMPV₂ {d D₁ D₂ : ℕ} (A : MPSTensor d D₁) (B : MPSTensor d D₂) :
                        Prop

                        Eventual nonzero proportionality of MPV families.

                        Source context: arXiv:1606.00608, Theorem thm1, line 1182 invokes Lemma Lem1, an eventual linear-independence statement. This auxiliary predicate is the corresponding eventual version of the projective proportionality relation: for all sufficiently large lengths, the two MPV vectors lie on the same nonzero projective line. The theorem hypothesis in line 1169 gives the stronger all-length predicate NonzeroProportionalMPV₂; this predicate is used only for tail reductions where finitely many initial lengths are irrelevant to the asymptotic conclusion.
