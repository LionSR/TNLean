    Periodic MPS definitions #

    This file introduces the basic periodic MPS predicates and equivalence relations used by the periodic form theory (arXiv:1708.00029, §2.1).

                structure MPSTensor.IsPeriodic {d D : ℕ} (m : ℕ) (A : MPSTensor d D) :
                Prop

                IsPeriodic m A bundles irreducibility, left-canonical normalization, peripheral spectrum equal to the m-th roots of unity, positivity of m, and existence of a primitive m-th root.

                This is the periodic analogue of primitivity data in arXiv:1708.00029, §2.1.

                • irreducible : A.IsIrreducibleTensor

                  No nontrivial invariant projection.

                • leftCanonical : A.IsLeftCanonical

                  Left-canonical normalization.

                • period_pos : 0 < m

                  Period is positive.

                • peripheral_eq : peripheralEigenvalues A.transferMap = {μ : ℂ | μ ^ m = 1}

                  Peripheral eigenvalues are exactly m-th roots of unity.

                • primitiveRoot : ∃ (ω : ℂ), IsPrimitiveRoot ω m

                  Existence of a primitive m-th root of unity.

                      structure MPSTensor.BasisOfPeriodicTensors {d D : ℕ} (r : ℕ) :
                      Type

                      Basis data for periodic tensors in irreducible form.

                      This is a lightweight container: a family of periodic blocks indexed by Fin r, with positive periods and pairwise non-repetition.

                      • blocks : Fin r → MPSTensor d D

                        The periodic block family.

                      • period : Fin r → ℕ

                        Period of each block.

                      • periodic (k : Fin r) : IsPeriodic (self.period k) (self.blocks k)

                        Each block is periodic with its assigned period.

                      • pairwise_nonrepeated : Pairwise fun (i j : Fin r) => ¬(self.blocks i).RepeatedBlocks (self.blocks j)

                        Distinct basis blocks are not repeated versions of one another.
