    Shared sector decomposition infrastructure #

    This file collects the multiplicity layer that both the canonical-form construction and the equal-case fundamental theorem use:

    • SectorWeightData
    • SectorDecomposition
    • the basic MPV expansion formulas for SectorDecomposition.toTensor

    Higher-level coefficient comparison and Newton–Girard recovery theorems remain in TNLean.MPS.FundamentalTheorem.SectorDecomposition.

    source
    structure MPSTensor.SectorWeightData (g : ℕ) :
    Type

    Sector multiplicity and weight data over a family of basis blocks.

    copies j is the multiplicity r_j of basis block j, while weight j q is the sector weight μ_{j,q} attached to the q-th copy.

    • copies : Fin g → ℕ

      The multiplicity r_j of the basis block j.

    • copies_pos (j : Fin g) : 0 < self.copies j

      Each basis block occurs at least once.

    • weight (j : Fin g) : Fin (self.copies j) → ℂ

      The sector weight μ_{j,q} attached to the q-th copy of basis block j.

    • weight_ne_zero (j : Fin g) (q : Fin (self.copies j)) : self.weight j q ≠ 0

      All sector weights are nonzero.

        structure MPSTensor.SectorDecomposition (d : ℕ) :
        Type

        A sector decomposition: a basis of normal tensors together with per-copy sector weight data.

        This bundles the basis-block family A_j with the multiplicity and weight structure SectorWeightData.

        • basisCount : ℕ

          Number of basis blocks A_j.

        • basisDim : Fin self.basisCount → ℕ

          Bond dimension of each basis block.

        • basis (j : Fin self.basisCount) : MPSTensor d (self.basisDim j)

          The basis-block family A_j.

        • sectors : SectorWeightData self.basisCount

          Multiplicities and sector weights lying over the basis blocks.
