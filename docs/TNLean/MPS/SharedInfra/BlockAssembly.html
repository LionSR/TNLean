    Shared block-assembly infrastructure for MPS tensors #

    This file factors out the lightweight block-diagonal tensor constructor toTensorFromBlocks, its word-evaluation expansion, and its MPV expansion formula so shared infrastructure can depend on block assembly without importing the multi-block Fundamental Theorem.
