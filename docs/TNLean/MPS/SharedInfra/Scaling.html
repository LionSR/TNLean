    Shared scaling lemmas for MPS tensors #

    This module collects the tensor-scaling identities that are used both by the canonical-form construction and by transfer-normalization arguments.

    source
    theorem MPSTensor.transferMap_smul {d D : ℕ} (c : ℂ) (A : MPSTensor d D) (X : Matrix (Fin D) (Fin D) ℂ) :
    (transferMap fun (i : Fin d) => c • A i) X = (c * (starRingEnd ℂ) c) • A.transferMap X

    Scaling an MPS tensor by c scales the transfer map by c * star c.
