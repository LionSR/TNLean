    Shared Kraus-adjoint setup for cyclic-sector arguments #

    This file collects the common conversion from an irreducible TP MPS tensor to the conjugate-transposed Kraus family used in cyclic-sector decompositions.

    source
    theorem MPSTensor.conjTranspose_kraus_setup {d D : ℕ} [NeZero D] (A : MPSTensor d D) (hTP : ∑ i : Fin d, (A i).conjTranspose * A i = 1) (hIrr : A.IsIrreducibleTensor) :
    ∃ (K : MPSTensor d D) (_ : KadisonSchwarz.IsUnitalKraus K) (_ : IsIrreducibleMap K.transferMap) (ρ : Matrix (Fin D) (Fin D) ℂ) (_ : ρ.PosDef) (_ : Kraus.adjointMap K ρ = ρ), K = fun (i : Fin d) => (A i).conjTranspose

    From an irreducible TP tensor, derive the conjugate-transposed Kraus family K, its unitality and irreducibility, and a PosDef fixed point ρ of Kraus.adjointMap K.

    This bundles the common setup shared by cyclic-sector decomposition arguments.
