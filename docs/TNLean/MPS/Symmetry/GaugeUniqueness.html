    Gauge uniqueness for injective MPS tensors #

    If two invertible gauges X and Y both map an injective tensor A to the same B, then Y is a nonzero scalar multiple of X.

    This is the scalar-commutant input needed to make the symmetry gauge well-defined up to phase.

    source
    theorem MPSTensor.gauge_unique_up_to_scalar {d D : ℕ} {A B : MPSTensor d D} (hA : A.IsInjective) {X Y : GL (Fin D) ℂ} (hX : ∀ (i : Fin d), B i = ↑X * A i * ↑X⁻¹) (hY : ∀ (i : Fin d), B i = ↑Y * A i * ↑Y⁻¹) :
    ∃ (u : ℂˣ), ↑Y = ↑u • ↑X

    If two gauges send the same injective tensor A to the same tensor B, then they differ by a nonzero scalar.

    source
    theorem MPSTensor.gauge_unique_up_to_scalar_of_sameMPV {d D : ℕ} {A B : MPSTensor d D} (hA : A.IsInjective) (_hAB : A.SameMPV B) {X Y : GL (Fin D) ℂ} (hX : ∀ (i : Fin d), B i = ↑X * A i * ↑X⁻¹) (hY : ∀ (i : Fin d), B i = ↑Y * A i * ↑Y⁻¹) :
    ∃ (u : ℂˣ), ↑Y = ↑u • ↑X

    A Same-MPV corollary: after obtaining any two gauges from the single-block FT, they are unique up to a nonzero scalar.
