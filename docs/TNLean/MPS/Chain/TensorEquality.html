    Tensor equality up to a scalar on a 2-site chain #

    This file formalizes the trace-pairing reduction behind Lemma 2 of arXiv:1804.04964: if two injective pairs of local tensors agree under all virtual insertions on both bonds, then the two pairs are proportional by inverse nonzero scalars.

    The full proportionality theorem is stated as tensor_proportional. The first trace-to-product reduction steps are provided as auxiliary lemmas.

    source
    theorem MPSTensor.internal_products_eq {d D : ℕ} (A₁ A₂ B₁ B₂ : MPSTensor d D) (hInt : ∀ (X : Matrix (Fin D) (Fin D) ℂ) (i j : Fin d), (A₁ i * X * A₂ j).trace = (B₁ i * X * B₂ j).trace) (i j : Fin d) :
    A₂ j * A₁ i = B₂ j * B₁ i

    Internal insertion trace agreement implies equality of the mixed products A₂ j * A₁ i = B₂ j * B₁ i for all physical indices.

    source
    theorem MPSTensor.external_products_eq {d D : ℕ} (A₁ A₂ B₁ B₂ : MPSTensor d D) (hExt : ∀ (Y : Matrix (Fin D) (Fin D) ℂ) (i j : Fin d), (A₂ j * Y * A₁ i).trace = (B₂ j * Y * B₁ i).trace) (i j : Fin d) :
    A₁ i * A₂ j = B₁ i * B₂ j

    External insertion trace agreement implies equality of the mixed products A₁ i * A₂ j = B₁ i * B₂ j for all physical indices.

    theorem MPSTensor.tensor_proportional {d D : ℕ} (A₁ A₂ B₁ B₂ : MPSTensor d D) (hA₁ : A₁.IsInjective) (hA₂ : A₂.IsInjective) (hB₁ : B₁.IsInjective) (hB₂ : B₂.IsInjective) (hInt : ∀ (X : Matrix (Fin D) (Fin D) ℂ) (i j : Fin d), (A₁ i * X * A₂ j).trace = (B₁ i * X * B₂ j).trace) (hExt : ∀ (Y : Matrix (Fin D) (Fin D) ℂ) (i j : Fin d), (A₂ j * Y * A₁ i).trace = (B₂ j * Y * B₁ i).trace) :
    ∃ (lambda_ : ℂ), lambda_ ≠ 0 ∧ (∀ (i : Fin d), A₁ i = lambda_ • B₁ i) ∧ ∀ (j : Fin d), A₂ j = lambda_⁻¹ • B₂ j

    Lemma 2 of arXiv:1804.04964 (2-site case): two injective tensor pairs that agree under all virtual insertions on both bonds are proportional.
