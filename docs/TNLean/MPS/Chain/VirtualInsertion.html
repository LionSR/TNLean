    Physical realization map for virtual insertions #

    For an injective MPS tensor A : Fin d → M_D(ℂ), a virtual insertion X on a bond can be re-expressed as a physical operation on the local physical index. This file defines the corresponding realization maps using the decomposition map constructed in OneSidedInverse.

    source
    noncomputable def MPSTensor.physRealize {d D : ℕ} (A : MPSTensor d D) (hA : A.IsInjective) (X : Matrix (Fin D) (Fin D) ℂ) :
    Matrix (Fin d) (Fin d) ℂ

    The physical realization of a right virtual insertion. For injective A, physRealize A hA X is the d × d matrix of coefficients that rewrites each A i * X in the spanning family {A j}.

    Equations
    • A.physRealize hA X i j = (MPSTensor.decompositionMap hA) (A i * X) j
