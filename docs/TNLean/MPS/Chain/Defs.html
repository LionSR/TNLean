    Non-translation-invariant MPS chains #

    This file introduces the fixed-length periodic-chain version of an MPS tensor: site-dependent local tensors, their ordered matrix product coefficient, the corresponding equality notion at fixed chain length, and the cyclic gauge relation used by the algebraic fundamental-theorem route.

    Type

    A non-translation-invariant MPS chain with physical dimension d, bond dimension D, and n sites.

                def MPSChainTensor.GaugeEquiv {d D n : ℕ} (A B : MPSChainTensor d D n) :
                Prop

                Cyclic gauge equivalence on a periodic chain: Bₖ^σ = Zₖ Aₖ^σ Zₖ₊₁⁻¹, where the successor is taken modulo n.
