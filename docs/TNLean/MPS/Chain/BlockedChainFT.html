    Fundamental theorem endpoint for blocked chains #

    This module gives a blocked-chain endpoint. The theorem fundamentalTheorem_blockedChain is stated for blocked chains built from a common blocking length L.

    Design note: we keep the endpoint in explicit-assumption style (L, hA_block, hMPV) so downstream uses can choose blocking witnesses locally.
