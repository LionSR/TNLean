    Pure-state recovery inside the MPO formalism #

    This file studies the diagonal embedding of a pure MPS tensor into the MPO formalism and shows that the MPDO renormalization fixed-point condition recovers the usual pure-state IsRFP and IsZCL conditions.

    source
    def MPSTensor.toMPOTensor {d D : ℕ} (A : MPSTensor d D) :
    MPOTensor d D

    Embed a pure MPS tensor as a diagonal MPO with trivial bra structure: M^{ij} = δ_{ij} A^i.
