    General MPDO renormalization fixed point #

    This file records consequences of the provisional mixed-state RFP definition for MPO tensors, following the transfer-map aspect of arXiv:1606.00608, Definition 4.1.

    The full definition in the paper is phrased using trace-preserving completely positive blocking and unblocking maps. In this first pass MPOTensor.IsRFP records the transfer-map idempotence condition, which is the part already supported by the current infrastructure.

    Main results #

    • MPOTensor.isRFP_iff_isZCL: this provisional RFP condition coincides with MPOTensor.IsZCL.
