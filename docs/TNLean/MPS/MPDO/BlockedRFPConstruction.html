    Blocked-RFP construction for simple MPDOs #

    This file states the current blocked-RFP theorem for the simple-MPDO case of arXiv:1606.00608, Appendix C.2 / Theorem 4.9.

    At the present repository state, the local entropy side and the commuting-form side live in separate modules:

    • SimpleLocalStructure.lean isolates the local SAL/SSA and rank-one-T consequences of Lemmas C.2, C.4, and C.5.
    • CommutingFormBridge.lean isolates the η-local structure that carries the commuting-form witness of arXiv:1606.00608, Appendix C.2, Proposition C.8, after the sector-local neighboring operators have been assembled.
    • CommutingForm.lean states the GSNNCH / commuting-form target side of arXiv:1606.00608, Appendix C.2, Proposition C.8 and Theorem 4.9(iii).

    What is still missing is the preceding theorem turning the local simple-MPDO data into the global commuting-form property. Because of that gap, the present file is formulated around the explicit structure SimpleMPDOBlockedRFPData, which records the local Appendix C.2 hypotheses and the global commuting-form and zero-correlation-length conclusions.

    The current consequences are:

    • a blocked fusion-isometry witness at size 2, formalized as Nonempty (FusionIsometryData K 2);
    • the GSNNCH-with-ZCL case of Theorem 4.9;
    • the transfer-map fusion formulation of MPDO RFP.

    Thus this file supplies the Appendix C conclusion that later work can use once the remaining local-to-global implication has been formalized.

    structure MPOTensor.SimpleMPDOLocalStructureData :
    Type

    The local simple-MPDO structure isolated by Appendix C.2, Lemmas C.2, C.4, and C.5.

    This structure contains exactly the information proved in SimpleLocalStructure.lean:

    a normalized three-site reduced state satisfying equality in strong subadditivity, the resulting local η-structure, and a primitive real matrix T together with the rank-one conclusion extracted from the constant-trace condition.

    The structure is intentionally independent of a particular MPO tensor. The missing preceding theorem is the map from a concrete simple MPDO to this structure and then onward to HasCommutingForm: Lemma C.2 supplies the local η-structure (the Markov decomposition), Lemma C.4 supplies the trace matrix T and its primitivity, while ZCL and Lemma C.5 supply trace-power constancy and the rank-one factorization of T.

    • dA : ℕ

      Dimensions of the three contiguous local regions.

    • dB : ℕ
    • dC : ℕ
    • rhoABC : Matrix (Fin self.dA × Fin self.dB × Fin self.dC) (Fin self.dA × Fin self.dB × Fin self.dC) ℂ

      The normalized three-site reduced state entering Lemma C.2.

    • hRhoDM : self.rhoABC.PosSemidef ∧ self.rhoABC.trace = 1

      Density-matrix normalization for the local state.

    • hSSA : IsSSAEquality self.rhoABC ⋯

      Equality in strong subadditivity for rhoABC.

    • eta : Nonempty (EtaStructure self.rhoABC)

      The quantum-Markov decomposition (EtaStructure) from Lemma C.2.

    • Tdim : ℕ

      The auxiliary trace matrix T from Lemma C.4.

    • T : Matrix (Fin self.Tdim) (Fin self.Tdim) ℝ
    • hPrimitive : self.T.IsPrimitive

      Primitivity of T, supplied by Lemma C.4.

    • hTrace : self.T.trace = 1

      Trace normalization of T.

    • hTraceConst : self.T.TracePowersConstant

      Constancy of the traces of positive powers of T.

    • rankOne : ∃ (a : Fin self.Tdim → ℝ) (b : Fin self.Tdim → ℝ), self.T = Matrix.vecMulVec a b ∧ a ⬝ᵥ b = 1

      Rank-one conclusion for T.
