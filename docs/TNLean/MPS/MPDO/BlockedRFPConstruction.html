    Blocked-RFP construction for simple MPDOs #

    This file states the current blocked-RFP theorem for the simple-MPDO branch of arXiv:1606.00608, Appendix C.2 / Theorem 4.9.

    At the present repository state, the local entropy side and the commuting-form side live in separate modules:

    • SimpleLocalStructure.lean isolates the local SAL/SSA and rank-one-T consequences of Lemmas C.3--C.4.
    • CommutingForm.lean states the GSNNCH / commuting-form target side of Proposition C.6 and Theorem 4.9(iii).

    What is still missing is the preceding theorem turning the local simple-MPDO data into the global commuting-form property. Because of that gap, the present file is formulated around the explicit structure SimpleMPDOBlockedRFPData. The local Appendix C.2 hypotheses are retained there, together with a temporary compatibility relation to the ambient MPO tensor, so that the eventual local-to-global implication has a canonical target.

    The current consequences are:

    • a blocked fusion-isometry witness at size 2, formalized as Nonempty (FusionIsometryData K 2);
    • the GSNNCH-with-ZCL branch of Theorem 4.9;
    • the transfer-map fusion formulation of MPDO RFP.

    Thus this file supplies the paper-level conclusion that later work can use once the remaining local-to-global implication has been formalized.

    structure MPOTensor.SimpleMPDOLocalStructureData :
    Type

    The local simple-MPDO structure isolated by Appendix C.2, Lemmas C.3--C.4.

    This structure contains exactly the information proved in SimpleLocalStructure.lean:

    a normalized three-site reduced state satisfying equality in strong subadditivity, the resulting local η-structure, and a primitive real matrix T together with the rank-one conclusion extracted from the constant-trace condition.

    The structure is intentionally independent of a particular MPO tensor. The missing preceding theorem is the map from a concrete simple MPDO satisfying SAL/ZCL to such a structure and then onward to HasCommutingForm.

          structure MPOTensor.SimpleMPDOBlockedRFPData {d D : ℕ} (K : MPOTensor d D) :
          Type

          Auxiliary structure for the simple-MPDO blocked-RFP argument.

          The field localData contains the Appendix C.2 local structure from issue #781, the field compatible gives its temporary attachment to K, and the fields commutingForm and zcl contain the issue-#782 commuting-form conclusion and the MPO zero-correlation-length hypothesis. This is the current point where the two sibling branches meet: the missing theorem is the derivation of commutingForm from the entropy-side hypotheses attached to K.

          The present blocked-RFP consequences only use commutingForm and zcl; the pair localData / compatible is retained so that the eventual local-to-global implication already has a type-level target.

          • localData : SimpleMPDOLocalStructureData

            Local SAL/SSA/rank-one data from Appendix C.2.

          • compatible : self.localData.CompatibleWith K

            Temporary attachment of the local data to the ambient MPO tensor.

          • commutingForm : K.HasCommutingForm

            Global commuting-form / GSNNCH data.

          • zcl : K.IsZCL

            Zero correlation length, hence the current RFP predicate.
