    Transfer maps of MPS tensors #

    This file defines the transfer operator transferMap associated to an MPS tensor. It proves the basic API needed later: the evaluation formula, gauge covariance, and preservation of positive semidefiniteness. These lemmas bridge between tensor data and channel-theoretic arguments.

    source
    noncomputable def MPSTensor.transferMap {d D : ℕ} (A : MPSTensor d D) :
    Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ

    The transfer operator associated to an MPS tensor A: $$E_A(X) = \sum_i A_i X A_i^{\dagger}.$$
