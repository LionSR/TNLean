    Transfer maps under physical blocking #

    This file identifies the transfer map of a physically blocked tensor with the corresponding iterate of the original transfer map. As consequences, transferMap_blockTensor_hasEigenvalue and transferMap_blockTensor_fixedPoint transport eigenvalues and fixed points through blocking.

    source
    theorem MPSTensor.transferMap_blockTensor_apply {d D : ℕ} (A : MPSTensor d D) (L : ℕ) (X : Matrix (Fin D) (Fin D) ℂ) :
    (A.blockTensor L).transferMap X = (A.transferMap ^ L) X

    The transfer map of the physically blocked tensor agrees with the L-fold iterate of the original transfer map (apply form).

    source
    theorem MPSTensor.transferMap_blockTensor {d D : ℕ} (A : MPSTensor d D) (L : ℕ) :
    (A.blockTensor L).transferMap = A.transferMap ^ L

    The transfer map of the physically blocked tensor agrees with the L-fold iterate of the original transfer map (as linear maps).

    source
    theorem MPSTensor.transferMap_blockTensor_hasEigenvalue {d D : ℕ} (A : MPSTensor d D) (L : ℕ) {μ : ℂ} (hμ : Module.End.HasEigenvalue A.transferMap μ) :
    Module.End.HasEigenvalue (A.blockTensor L).transferMap (μ ^ L)

    Eigenvalues transport along physical blocking: if μ is an eigenvalue of transferMap A, then μ ^ L is an eigenvalue of the transfer map of the blocked tensor.

    source
    theorem MPSTensor.transferMap_blockTensor_fixedPoint {d D : ℕ} (A : MPSTensor d D) (L : ℕ) (X : Matrix (Fin D) (Fin D) ℂ) (hX : A.transferMap X = X) :
    (A.blockTensor L).transferMap X = X

    Fixed points (eigenvalue 1) are preserved under physical blocking.

    source
    theorem MPSTensor.transferMap_blockTensor_mul {d D : ℕ} (A : MPSTensor d D) (m n : ℕ) :
    ((A.blockTensor m).blockTensor n).transferMap = (A.blockTensor (m * n)).transferMap

    Iterated physical blocking is compatible with one-shot blocking by the multiplied period, at the transfer-map level: transferMap(block(block(A, m), n)) = transferMap(block(A, m * n)).
