    Multi-block canonical forms for MPS tensors #

    This file introduces the lightweight CanonicalForm structure for block-diagonal MPS tensors with injective blocks. It constructs the associated tensor CanonicalForm.toTensor and proves the block-diagonal word-evaluation and trace/reindexing lemmas used in the multi-block Fundamental Theorem.

    source
    structure MPSTensor.CanonicalForm (d : ℕ) :
    Type

    A block-injective canonical form for an MPS tensor: block diagonal with injective blocks.

    This is the lightweight data structure used later to state and prove the multi-block version of the Fundamental Theorem.

    • numBlocks : ℕ

      Number of blocks

    • blockDim : Fin self.numBlocks → ℕ

      Bond dimension of each block

    • blockTensor (k : Fin self.numBlocks) : MPSTensor d (self.blockDim k)

      The tensor for each block

    • μ : Fin self.numBlocks → ℂ

      Scaling factor for each block

    • block_injective (k : Fin self.numBlocks) : (self.blockTensor k).IsInjective

      Each block tensor is injective.

        MPSTensor d C.totalDim

        Turn a canonical form into an actual MPS tensor by putting the blocks on the diagonal.

        We use the dependent block-diagonal construction Matrix.blockDiagonal', and then reindex the resulting Σ-indexed matrix as a Fin (∑ k, blockDim k)-indexed matrix using finSigmaFinEquiv.

        Equations
        • C.toTensor i = (Matrix.reindex finSigmaFinEquiv finSigmaFinEquiv) (Matrix.blockDiagonal' fun (k : Fin C.numBlocks) => C.μ k • C.blockTensor k i)

          This is the same recursion as MPSTensor.evalWord, but it works for matrices indexed by an arbitrary finite type (in particular, the Σ-type indices produced by Matrix.blockDiagonal').

          Equations

          • evalWord A (i :: w) = A i * evalWord A w
