    Repeated words #

    This file contains elementary identities for evaluating a tensor on repeated words. These lemmas are low-level consequences of the multiplicativity of evalWord and are shared by canonical-form and Wielandt arguments.

    Evaluation on repeated words #

    source
    theorem MPSTensor.evalWord_replicate {d D : ℕ} (A : MPSTensor d D) (i : Fin d) (L : ℕ) :
    A.evalWord (List.replicate L i) = A i ^ L
