    Linear extension of the map A i ↦ B i #

    This file proves two key lemmas for the Fundamental Theorem of MPS:

    1. Existence and uniqueness of the linear extension T with T(A i) = B i, under the assumption that A is injective and SameMPV A B.
    2. Multiplicativity of T: the SameMPV condition forces T(MN) = T(M)T(N).

    Both proofs use the traceMulRightPi methods from TracePairing.lean.

    source
    theorem MPSTensor.linearExtension_exists_unique {d D : ℕ} {A B : MPSTensor d D} (hA : A.IsInjective) (hAB : A.SameMPV B) :
    ∃! T : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ, ∀ (i : Fin d), T (A i) = B i

    Lemma 1 (paper proof sketch):

    If A is injective (its matrices span the full matrix algebra) and A and B generate the same MPV family, then there exists a unique linear map T sending A i to B i.

    The key point is that SameMPV provides compatibility of all trace pairings trace (A i * A j) = trace (B i * B j), which lets us construct T via a left inverse of the map M ↦ (i ↦ trace (M * B i)).

    source
    theorem MPSTensor.linearExtension_mul {d D : ℕ} {A B : MPSTensor d D} (hA : A.IsInjective) (hAB : A.SameMPV B) {T : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ} (hT : ∀ (i : Fin d), T (A i) = B i) (M N : Matrix (Fin D) (Fin D) ℂ) :
    T (M * N) = T M * T N

    Lemma 4 (paper proof sketch, now proved):

    If T is the linear extension with T(A i)=B i and SameMPV A B, then T is multiplicative.

    The proof is trace-based:

    • Using SameMPV for length-2 words, we show traceMulRightPi B (T (A i)) = traceMulRightPi A (A i); by spanning this extends to traceMulRightPi B ∘ T = traceMulRightPi A.
    • Injectivity of A implies traceMulRightPi A is injective, hence has full-rank range. The range inclusion above forces traceMulRightPi B to have trivial kernel.
    • Using length-3 trace identities and injectivity of traceMulRightPi B, we get T (A i * A j) = B i * B j, and then extend bilinearly using spanning.
