    Linear extension of the map A i ↦ B i #

    This file proves two key lemmas for the Fundamental Theorem of MPS:

    1. Existence and uniqueness of the linear extension T with T(A i) = B i, under the assumption that A is injective and SameMPV A B.
    2. Multiplicativity of T: the SameMPV condition forces T(MN) = T(M)T(N).

    Both proofs use the traceMulRightPi methods from TracePairing.lean.

    source
    theorem MPSTensor.linearExtension_exists_unique {d D : ℕ} {A B : MPSTensor d D} (hA : A.IsInjective) (hAB : A.SameMPV B) :
    ∃! T : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ, ∀ (i : Fin d), T (A i) = B i

    If A is injective and A, B generate the same MPV family, then there exists a unique linear map $T : \MN{D} \to \MN{D}$ satisfying $T(A^i) = B^i$ for all $i$.

    source
    theorem MPSTensor.linearExtension_mul {d D : ℕ} {A B : MPSTensor d D} (hA : A.IsInjective) (hAB : A.SameMPV B) {T : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ} (hT : ∀ (i : Fin d), T (A i) = B i) (M N : Matrix (Fin D) (Fin D) ℂ) :
    T (M * N) = T M * T N

    If A is injective, A and B generate the same MPV family, and $T : \MN{D} \to \MN{D}$ is the linear map satisfying $T(A^i) = B^i$ for all $i$, then $T(MN) = T(M) T(N)$ for all $M, N \in \MN{D}$.
