    Basic parent-Hamiltonian properties #

    The parent interaction is the orthogonal projector onto (groundSpace A L)ᗮ. It therefore annihilates any vector in the ground space. The MPS state mpv A lies in the ground space at every window, which gives frustration-freeness.

    Key lemma: parent interaction kills ground space elements #

    source
    theorem MPSTensor.parentInteraction_apply_mem_groundSpace {d D : ℕ} (A : MPSTensor d D) (L : ℕ) (v : NSiteSpace d L) (hv : v ∈ A.groundSpace L) :
    (A.parentInteraction L) v = 0

    The parent interaction annihilates any vector in the ground space. This is the core property: parentInteraction A L is the orthogonal projector onto (groundSpace A L)ᗮ, so it kills everything in groundSpace A L.

    Cyclic trace invariance #

    MPS window membership #

    source
    theorem MPSTensor.mpv_window_mem_groundSpace {d D : ℕ} (A : MPSTensor d D) (L N : ℕ) (hLN : L ≤ N) (i : Fin N) (σ : Cfg d N) :
    (fun (τ : Fin L → Fin d) => A.mpv (replaceWindow L hLN i σ τ)) ∈ A.groundSpace L

    The MPS state restricted to any window of L sites lies in groundSpace A L.

    The witness is the "complement matrix": the product of A-matrices on sites outside the L-site window, cyclically ordered starting from i + L. The proof uses trace cyclicity to rotate the full N-site product so that the window indices come first, matching the groundSpaceMap definition.
