    Lemma 1: Transfer map scales quadratically #

    source
    theorem MPSTensor.transferMap_smul {d D : ℕ} (c : ℂ) (A : MPSTensor d D) (X : Matrix (Fin D) (Fin D) ℂ) :
    (transferMap fun (i : Fin d) => c • A i) X = (c * (starRingEnd ℂ) c) • A.transferMap X

    Scaling an MPS tensor by c scales the transfer map by c * star c. Specifically, transferMap (c • A) X = (c * starRingEnd ℂ c) • transferMap A X (since c * star c = ‖c‖² for complex scalars).

    Lemma 2: Left-canonical normalization preserved under unit-norm scaling #

    source
    theorem MPSTensor.leftCanonical_smul_of_norm_one {d D : ℕ} (c : ℂ) (hc : ‖c‖ = 1) (A : MPSTensor d D) (hA : ∑ i : Fin d, (A i).conjTranspose * A i = 1) :
    ∑ i : Fin d, (c • A i).conjTranspose * c • A i = 1

    If ∑ i, (A i)ᴴ * (A i) = 1 and ‖c‖ = 1, then ∑ i, (c • A i)ᴴ * (c • A i) = 1.

    This is the left-canonical (trace-preserving) normalization condition: scaling by a unit-norm complex scalar preserves the condition ∑ Aᵢᴴ Aᵢ = I.

    Theorem 5: Normalization of μ-scaled block #

    source
    theorem MPSTensor.mpv_smul_block {d : ℕ} (μ : ℂ) {D' : ℕ} (A : MPSTensor d D') {N : ℕ} (σ : Fin N → Fin d) :
    mpv (fun (i : Fin d) => μ • A i) σ = μ ^ N * A.mpv σ

    For a block tensor μ • A, the MPV decomposes as μ^N * mpv(A). This is the key factorization used in μ-normalization.

    source
    theorem MPSTensor.transferMap_smul_block {d : ℕ} (μ : ℂ) {D' : ℕ} (A : MPSTensor d D') (X : Matrix (Fin D') (Fin D') ℂ) :
    (transferMap fun (i : Fin d) => μ • A i) X = (μ * (starRingEnd ℂ) μ) • A.transferMap X

    The transfer map of μ • A is (μ * star μ) times the transfer map of A. Since μ * star μ = ‖μ‖², this means that if A has spectral radius ρ, then μ • A has spectral radius ‖μ‖² * ρ.

    theorem MPSTensor.phase_norm_one {μ : ℂ} (hμ : μ ≠ 0) :
    ‖μ / ↑‖μ‖‖ = 1

    Rescaling a tensor by μ / ‖μ‖ (the phase of μ) gives a unit-norm scalar that preserves the left-canonical condition.

    source
    theorem MPSTensor.smul_eq_norm_smul_phase {D : ℕ} (μ : ℂ) (hμ : μ ≠ 0) (M : Matrix (Fin D) (Fin D) ℂ) :
    μ • M = ↑‖μ‖ • (μ / ↑‖μ‖) • M

    For nonzero μ, writing μ = ‖μ‖ * (μ / ‖μ‖), we can split a scaled tensor μ • A into a modulus part ‖μ‖ and a phase part (μ/‖μ‖) • A.

    Theorem 6: Full normalization theorem #

    source
    theorem MPSTensor.mpv_normalize_blocks {d r : ℕ} {dim : Fin r → ℕ} (μ : Fin r → ℂ) (hμ : ∀ (k : Fin r), μ k ≠ 0) (A : (k : Fin r) → MPSTensor d (dim k)) {N : ℕ} (σ : Fin N → Fin d) :
    (toTensorFromBlocks μ A).mpv σ = (toTensorFromBlocks (fun (k : Fin r) => μ k / ↑‖μ k‖) fun (k : Fin r) (i : Fin d) => ↑‖μ k‖ • A k i).mpv σ

    PF/μ normalization: Given a block-diagonal CF tensor toTensorFromBlocks μ A, one can normalize each block so that the scaling μ_k becomes μ_k / ‖μ_k‖ (unit modulus) while absorbing ‖μ_k‖ into the block tensor. The MPV is preserved exactly.

    More precisely: toTensorFromBlocks μ A and toTensorFromBlocks μ' A' generate the same MPV when μ'_k = μ_k / ‖μ_k‖ and A'_k = ‖μ_k‖ • A_k.

    source
    theorem MPSTensor.transferMap_normalize_block {d D' : ℕ} (μ : ℂ) (hμ : μ ≠ 0) (A : MPSTensor d D') (X : Matrix (Fin D') (Fin D') ℂ) :
    (transferMap fun (i : Fin d) => (μ / ↑‖μ‖) • ↑‖μ‖ • A i) X = (transferMap fun (i : Fin d) => μ • A i) X

    Modulus-phase factorization of the transfer map: After normalizing, the transfer map of (μ_k / ‖μ_k‖) • (‖μ_k‖ • A_k) equals the transfer map of μ_k • A_k. This means the spectral properties of the transfer map are unchanged by the normalization.

    theorem MPSTensor.transferMap_norm_smul {d : ℕ} (μ : ℂ) {D' : ℕ} (A : MPSTensor d D') (X : Matrix (Fin D') (Fin D') ℂ) :
    (transferMap fun (i : Fin d) => ↑‖μ‖ • A i) X = ↑(‖μ‖ ^ 2) • A.transferMap X

    After absorbing the modulus into the tensor, the transfer map of the normalized tensor ‖μ‖ • A has spectral radius scaled by ‖μ‖². That is, E_{‖μ‖ • A}(X) = ‖μ‖² • E_A(X).

    theorem MPSTensor.leftCanonical_phase_smul {d D' : ℕ} (μ : ℂ) (hμ : μ ≠ 0) (A : MPSTensor d D') (hA : ∑ i : Fin d, (A i).conjTranspose * A i = 1) :
    ∑ i : Fin d, ((μ / ↑‖μ‖) • A i).conjTranspose * (μ / ↑‖μ‖) • A i = 1

    Phase scaling preserves left-canonical normalization: if ∑ (A i)ᴴ * (A i) = 1 and μ ≠ 0, then the phase-scaled tensor (μ / ‖μ‖) • A also satisfies ∑ ((μ/‖μ‖) • A i)ᴴ * ((μ/‖μ‖) • A i) = 1.

    This is used in the μ-normalization step of the canonical form: one can scale each block eigenvalue to unit modulus while keeping the block tensor in left-canonical gauge.
