    MPS-specific convergence helpers for MPV overlap arguments #

    MPS-specific convergence lemmas used in the fundamental theorem proof chain. The general-purpose convergence lemmas (bounded_mul_tendsto_zero, geometric_mul_bounded_tendsto_zero, sum_tendsto_one_of_diag) live in TNLean.Analysis.ConvergenceHelpers.

    Main results #

    • tendsto_inner_zero: mpvOverlap → 0 implies mpvInner → 0
    • tendsto_inner_one: mpvOverlap → 1 (self) implies mpvInner → 1
    • geometric_mul_inner_tendsto_zero: geometric(< 1) × Cauchy-Schwarz-bounded inner → 0

    theorem MPSTensor.tendsto_inner_zero {d D₁ D₂ : ℕ} (X : MPSTensor d D₁) (Y : MPSTensor d D₂) (hOv : Filter.Tendsto (fun (N : ℕ) => X.mpvOverlap Y N) Filter.atTop (nhds 0)) :
    Filter.Tendsto (fun (N : ℕ) => X.mpvInner Y N) Filter.atTop (nhds 0)

    mpvOverlap → 0 implies mpvInner → 0, via the relation mpvOverlap = star ∘ mpvInner.

    source
    theorem MPSTensor.tendsto_inner_one {d D : ℕ} (X : MPSTensor d D) (hOv : Filter.Tendsto (fun (N : ℕ) => X.mpvOverlap X N) Filter.atTop (nhds 1)) :
    Filter.Tendsto (fun (N : ℕ) => X.mpvInner X N) Filter.atTop (nhds 1)

    mpvOverlap X X → 1 (self-overlap) implies mpvInner X X → 1, via the relation mpvOverlap = star ∘ mpvInner.

    theorem MPSTensor.geometric_mul_inner_tendsto_zero {d D₁ D₂ : ℕ} (c : ℂ) (X : MPSTensor d D₁) (Y : MPSTensor d D₂) (hc : ‖c‖ < 1) (hX : Filter.Tendsto (fun (N : ℕ) => X.mpvOverlap X N) Filter.atTop (nhds 1)) (hY : Filter.Tendsto (fun (N : ℕ) => Y.mpvOverlap Y N) Filter.atTop (nhds 1)) :
    Filter.Tendsto (fun (N : ℕ) => c ^ N * X.mpvInner Y N) Filter.atTop (nhds 0)

    Geometric factor with ‖c‖ < 1 times a Cauchy-Schwarz-bounded inner product tends to zero, given that both self-overlaps converge to 1.
