    Applications of the fundamental theorem #

    This module contains:

    1. A lightweight single-block symmetry wrapper (rotatePhysical + gaugeEquiv_of_sameMPV_rotatePhysical).
    2. A periodic-form assembly lemma that isolates the only missing input for the full Corollary 4.1 of arXiv:1708.00029 §4.
    3. Preservation lemmas showing that unitary rotation of the physical index preserves the transfer map, left-canonical normalization, irreducibility, periodicity, and irreducible form II structure.

    Status for §4 (as of merged periodic FT infrastructure) #

    • Corollary 4.1 (symmetry corollary): reduced to one call to the periodic equal-case FT, now that isIrreducibleForm_rotatePhysical is fully proven.
    • Theorem 4.1 (p-refinement): still needs the periodic-block phase-distribution construction from §4.

      theorem MPSTensor.gaugeEquiv_of_sameMPV_rotatePhysical {d D : ℕ} (u : Matrix (Fin d) (Fin d) ℂ) (A : MPSTensor d D) (hA : A.IsInjective) (hSym : A.SameMPV (rotatePhysical u A)) :
      A.GaugeEquiv (rotatePhysical u A)

      Symmetry-to-virtual-gauge wrapper.

      If A is injective and has the same MPV family as its physical-leg rotation B = rotatePhysical u A, then B is gauge equivalent to A.

      This is the formal Lean bridge used in Corollary-4.1 style arguments: the nontrivial analytic/group-theoretic part is in the hypothesis SameMPV A (rotatePhysical u A), and the conclusion is provided by the single-block Fundamental Theorem.

      theorem MPSTensor.zGaugeEquiv_of_isIrreducibleForm_sameMPV_rotatePhysical {d D : ℕ} (u : Matrix (Fin d) (Fin d) ℂ) (A : MPSTensor d D) (hA : A.IsIrreducibleForm) (hRot : (rotatePhysical u A).IsIrreducibleForm) (hSym : A.SameMPV (rotatePhysical u A)) (hPeriodicEq : ∀ {X Y : MPSTensor d D} (a : X.IsIrreducibleForm) (a : Y.IsIrreducibleForm), X.SameMPV Y → ∃ (m : ℕ), 0 < m ∧ ZGaugeEquiv m X Y) :
      ∃ (m : ℕ), 0 < m ∧ ZGaugeEquiv m A (rotatePhysical u A)

      Corollary 4.1 assembly step (periodic form).

      Assume the periodic equal-case FT as a hypothesis (hPeriodicEq): whenever two tensors are in irreducible form II and generate the same MPV family, they are Z-gauge equivalent. Then the symmetry corollary follows immediately for B := rotatePhysical u A once B is known to be in irreducible form II.

      This theorem intentionally packages the current dependency boundary: no additional overlap arguments are needed here beyond the periodic equal-case FT input.

      source
      theorem MPSTensor.transferMap_rotatePhysical {d D : ℕ} (A : MPSTensor d D) (u : Matrix (Fin d) (Fin d) ℂ) (hu : u * u.conjTranspose = 1) :
      (rotatePhysical u A).transferMap = A.transferMap

      The transfer map is invariant under unitary rotation of the physical index: E_{rotatePhysical u A} = E_A when u * uᴴ = 1.

      This follows from unitary freedom of the Kraus representation (Theorem 2.18, Wolf Quantum Channels & Operations).

      theorem MPSTensor.isIrreducibleTensor_rotatePhysical {d D : ℕ} (A : MPSTensor d D) (u : Matrix (Fin d) (Fin d) ℂ) (hu : u * u.conjTranspose = 1) (hA : A.IsIrreducibleTensor) :
      (rotatePhysical u A).IsIrreducibleTensor

      IsIrreducibleTensor is preserved by unitary rotation of the physical index.

      If P were a nontrivial invariant projection for rotatePhysical u A, then multiplying the invariance condition by conj(u_{ik}) and summing over i yields (1-P) A_k P = 0 via the orthogonality uᴴ u = 1, so P would also be invariant for A, contradicting irreducibility.

      If A and B generate the same MPV family, so do their physical-index rotations rotatePhysical u A and rotatePhysical u B.

      The proof uses a strengthened induction with an arbitrary prefix word, reducing each step to the SameMPV₂ hypothesis via word concatenation.

      theorem MPSTensor.rotatePhysical_toTensorFromBlocks {d r : ℕ} {dim : Fin r → ℕ} (u : Matrix (Fin d) (Fin d) ℂ) (μ : Fin r → ℂ) (A : (k : Fin r) → MPSTensor d (dim k)) :
      rotatePhysical u (toTensorFromBlocks μ A) = toTensorFromBlocks μ fun (k : Fin r) => rotatePhysical u (A k)

      Physical-index rotation distributes over the block-diagonal assembly: rotatePhysical u (toTensorFromBlocks μ A) = toTensorFromBlocks μ (fun k => rotatePhysical u (A k)).

      This follows from linearity of blockDiagonal' and smul through finite sums.

      noncomputable def MPSTensor.isIrreducibleForm_rotatePhysical {d D : ℕ} (A : MPSTensor d D) (u : Matrix (Fin d) (Fin d) ℂ) (hu : u * u.conjTranspose = 1) (hA : A.IsIrreducibleForm) :

      IsIrreducibleForm is preserved by unitary rotation of the physical index.

      The rotated tensor uses the same block structure (r, dim, μ, period) with rotated blocks fun k => rotatePhysical u (hA.blocks k). Each block remains periodic by isPeriodic_rotatePhysical, and the SameMPV₂ condition transfers via sameMPV₂_rotatePhysical and rotatePhysical_toTensorFromBlocks.
