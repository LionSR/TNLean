    Product-pair structural connection for commuting parent Hamiltonians #

    This file states the connection between the Appendix B structural form A i = X * diag(Λ) * U i * X⁻¹ and the nearest-neighbor commutativity equations hᵢ * hⱼ = hⱼ * hᵢ used in TNLean/MPS/ParentHamiltonian/Commuting.lean.

    At the current stage of the library, the NSiteSpace formalization does not yet encode explicit tensor-product support/locality data, while IsNNCPH is defined later in ParentHamiltonian/Commuting.lean. To keep the dependency direction suitable for the eventual proof of rfp_implies_nncph, this file does not import Commuting.lean. Instead it does three things:

    1. It defines the even-chain product-pair state generated by repeating a fixed two-site amplitude.
    2. It states the locality/independence input for nearest-neighbor parent projectors as an abstract witness on the full chain space, and proves from that witness the unfolded commutativity statement for the two-site local terms.
    3. It bundles the already-proved Appendix B structural form and records the two remaining mathematical inputs: the even-chain product-of-pairs factorization and the identification of the two-site parent terms with commuting idempotents.

    The intended subsequent use is:

    • obtain AppendixBStructuralData A from rfp_nt_structural_full;
    • build AppendixBProductPairExtraction for that structural form;
    • apply ProductPairBridge.commuting_twoSite_localTerms;
    • unfold IsNNCPH in ParentHamiltonian/Commuting.lean.

    TODO(tensor-product-locality): replace the abstract commutation witness for the full-chain endomorphisms by genuine support-locality statements once NSiteSpace carries the required tensor-product factorization structure.

              structure MPSTensor.AppendixBStructuralData {d D : ℕ} (A : MPSTensor d D) :
              Type

              Bundled form of the Appendix B structural decomposition A i = X * diag(Λ) * U i * X⁻¹.

              The theorem rfp_nt_structural_full proves existence of this structural form from the RFP, normality, and left-canonical hypotheses. We keep it as a structure so the remaining product-of-pairs factorization and projector identities can refer to the same X, Λ, and U.

              • X : Matrix (Fin D) (Fin D) ℂ

                The virtual-bond change of basis.

              • Λ : Fin D → ℝ

                Positive diagonal weights.

              • U : MPSTensor d D

                The residual left-canonical tensor family.

              • hX_det : self.X.det ≠ 0

                The change-of-basis matrix is invertible.

              • hΛ_pos (k : Fin D) : 0 < self.Λ k

                The diagonal weights are strictly positive.

              • hU_left : ∑ i : Fin d, (self.U i).conjTranspose * self.U i = 1

                The residual tensor is left-canonical.

              • hA_eq (i : Fin d) : A i = (self.X * Matrix.diagonal fun (k : Fin D) => ↑(self.Λ k)) * self.U i * self.X⁻¹

                The original tensor has the Appendix B structural form.
