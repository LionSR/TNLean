    Pure-state renormalization fixed point (RFP) — definitions #

    This file defines the notion of a renormalization fixed point (RFP) for a pure MPS tensor, following arXiv:1606.00608 §3.1 (Cirac–Pérez-García–Schuch–Verstraete).

    The key definition is IsRFP A, which says that the completely positive map (CPM) associated to the MPS tensor A is idempotent: composing the transfer map with itself yields the same transfer map.

    source
    def MPSTensor.IsRFP {d D : ℕ} (A : MPSTensor d D) :
    Prop

    An MPS tensor A is a renormalization fixed point (RFP) when its transfer map is idempotent as a linear map, i.e. E_A ∘ E_A = E_A. See arXiv:1606.00608, Definition 3.2.

      theorem MPSTensor.isRFP_iff_kraus_isometry {d D : ℕ} (A : MPSTensor d D) :
      A.IsRFP ↔ ∃ (V : Matrix (Fin d × Fin d) (Fin d) ℂ), V.conjTranspose * V = 1 ∧ ∀ (i₁ i₂ : Fin d), A i₁ * A i₂ = ∑ j : Fin d, V (i₁, i₂) j • A j

      The RFP condition is equivalent to a Kraus-level condition: there exists an isometry V : Fin d × Fin d → Fin d → ℂ (i.e. a (d²×d) matrix) such that A i₁ * A i₂ = ∑ j, V (i₁, i₂) j • A j for all i₁ i₂.

      Forward direction: E² = E means the product Kraus family {Aᵢ₁ Aᵢ₂} and the single family {Aⱼ} define the same CPM. By rectangular Kraus freedom (kraus_rectangular_freedom' from KrausFreedom.lean), they are related by a rectangular isometry V with V†V = 1.

      Backward direction: proved as isRFP_of_kraus_isometry.

      See arXiv:1606.00608, Theorem 3.1 and Wolf Thm 2.1 item 4.
