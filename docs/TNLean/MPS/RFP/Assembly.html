    Assembly: RFP ⟺ ZCL equivalence #

    This file assembles the forward and backward directions of the main equivalence from arXiv:1606.00608, Theorem 3.10 (partial):

    For a canonical-form MPS tensor, being a renormalization fixed point (RFP) is equivalent to having zero correlation length (ZCL).

    The NNCPH (nearest-neighbour commuting parent Hamiltonian) direction of Theorem 3.10 is deferred to a later module.

    This reduces directly to zcl_iff_idempotent_transfer (Theorem 3.8).
