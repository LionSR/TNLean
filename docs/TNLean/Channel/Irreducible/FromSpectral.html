    Irreducibility from spectral properties (Wolf Theorem 6.4) #

    This file collects Wolf's spectral characterization of irreducibility for completely positive maps on M_D(ℂ).

    Main declarations #

    • HasSpectralProperties: a Kraus-witness formulation of the spectral properties appearing in Wolf Theorem 6.4.
    • hasSpectralProperties_of_irreducible_cp: the forward implication irreducible → spectral properties.
    • isIrreducibleMap_of_channel_posDef_fixedPoint_unique: the channel lemma used in the reverse implication.
    • isIrreducibleMap_of_hasSpectralProperties: the reverse implication spectral properties → irreducible.
    • isIrreducibleMap_iff_spectral_properties: the final iff theorem.

    Formalization note #

    Wolf states that the spectral-radius eigenvalue is non-degenerate. In the current interface we formalize exactly the part needed for the irreducibility argument: any positive-semidefinite right eigenvector for the distinguished positive eigenvalue is a scalar multiple of the Perron eigenvector.

    This is enough for the contradiction argument after the TP gauge reduction, since the competing fixed point produced from a reducible channel is again positive semidefinite.

    structure SpectralProperties {D : ℕ} (E : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ) :
    Type

    A Kraus-witness formulation of the spectral properties in Wolf Theorem 6.4.

    The left_eig field is written using the conjugate-transposed Kraus family, which is the adjoint map for the Frobenius trace pairing. The unique_psd_eigenvector field is the formal version of the required non-degeneracy: every positive-semidefinite right eigenvector for the same positive eigenvalue is proportional to ρ.

    • n : ℕ
    • K : MPSTensor self.n D
    • map_eq : E = self.K.transferMap
    • ρ : Matrix (Fin D) (Fin D) ℂ
    • σ : Matrix (Fin D) (Fin D) ℂ
    • r : ℝ
    • ρ_posDef : self.ρ.PosDef
    • σ_posDef : self.σ.PosDef
    • hr_pos : 0 < self.r
    • right_eig : E self.ρ = ↑self.r • self.ρ
    • left_eig : (MPSTensor.transferMap fun (i : Fin self.n) => (self.K i).conjTranspose) self.σ = ↑self.r • self.σ
    • unique_psd_eigenvector (τ : Matrix (Fin D) (Fin D) ℂ) : τ.PosSemidef → E τ = ↑self.r • τ → ∃ (c : ℂ), τ = c • self.ρ
    • spectralRadius_eq : spectralRadius ℂ ((Module.End.toContinuousLinearMap (Matrix (Fin D) (Fin D) ℂ)) E) = ENNReal.ofReal self.r

        Forward implication: irreducible ⇒ spectral properties #

        source
        theorem hasSpectralProperties_of_irreducible_cp {D : ℕ} [NeZero D] (E : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ) (hCP : IsCPMap E) (hIrr : IsIrreducibleMap E) (hE : E ≠ 0) :
        HasSpectralProperties E

        Wolf Theorem 6.4, forward direction for CP maps.

        If E is a nonzero irreducible CP map, then it satisfies the spectral properties appearing in Wolf's theorem: a positive-definite right eigenvector, a positive-definite left eigenvector for the adjoint Kraus map, uniqueness of PSD right eigenvectors for the Perron eigenvalue, and identification of this eigenvalue with the spectral radius.

        Channel lemma for the reverse implication #

        source
        theorem isIrreducibleMap_of_channel_posDef_fixedPoint_unique {D : ℕ} [NeZero D] (E : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ) (hE : IsChannel E) (ρ : Matrix (Fin D) (Fin D) ℂ) (hρ_pd : ρ.PosDef) (_hρ_fix : E ρ = ρ) (huniq : ∀ (σ : Matrix (Fin D) (Fin D) ℂ), σ.PosSemidef → E σ = σ → ∃ (c : ℂ), σ = c • ρ) :
        IsIrreducibleMap E

        A channel with a positive-definite fixed point and no other PSD fixed points except scalar multiples of it is irreducible.

        This is the fixed-point contradiction at the heart of Wolf's proof of Theorem 6.4. If a nontrivial invariant projection P existed, the Cesàro mean inside the corner P M_D P would produce a density-matrix fixed point supported in that corner. It cannot be a scalar multiple of a positive-definite fixed point on the whole space.
