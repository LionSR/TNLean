    Canonical gauge normalizations from transfer fixed points #

    This file states the two standard one-sided similarity gauges extracted from fixed points of the transfer map and its adjoint. It is not a formalization of a doubly stochastic gauge. Instead, gauged_unital gives the right-canonical/unital gauge, while gauged_tracePreserving gives the left-canonical/trace-preserving gauge. The aliases gauged_rightCanonical and gauged_leftCanonical state the same formulas in the project's MPS terminology.

    source
    theorem gauged_unital {d D : ℕ} [DecidableEq (Fin D)] (A : MPSTensor d D) (S ρ : Matrix (Fin D) (Fin D) ℂ) (hS_inv : S.det ≠ 0) (hSS : S * S.conjTranspose = ρ) (hfix : A.transferMap ρ = ρ) :
    ∑ i : Fin d, S⁻¹ * A i * S * (S⁻¹ * A i * S).conjTranspose = 1

    If S is invertible and S * S† = ρ, then the gauged operators A'_i = S⁻¹ A_i S satisfy ∑ A'_i A'_i† = I whenever E_A(ρ) = ρ. This is the standard right-canonical gauge construction.

    theorem gauged_rightCanonical {d D : ℕ} [DecidableEq (Fin D)] (A : MPSTensor d D) (S ρ : Matrix (Fin D) (Fin D) ℂ) (hS_inv : S.det ≠ 0) (hSS : S * S.conjTranspose = ρ) (hfix : A.transferMap ρ = ρ) :
    ∑ i : Fin d, S⁻¹ * A i * S * (S⁻¹ * A i * S).conjTranspose = 1

    Right-canonical alias of gauged_unital.

    source
    theorem gauged_tracePreserving {d D : ℕ} [DecidableEq (Fin D)] (A : MPSTensor d D) (S σ : Matrix (Fin D) (Fin D) ℂ) (hS_inv : S.det ≠ 0) (hStS : S.conjTranspose * S = σ) (hfix : (MPSTensor.transferMap fun (i : Fin d) => (A i).conjTranspose) σ = σ) :
    ∑ i : Fin d, (S * A i * S⁻¹).conjTranspose * (S * A i * S⁻¹) = 1

    If S is invertible and Sᴴ * S = σ, then the gauged operators A'_i = S * A_i * S⁻¹ satisfy ∑ A'_iᴴ * A'_i = I whenever E_A†(σ) = σ.

    This is the trace-preserving (left-canonical) analogue of gauged_unital.

    theorem gauged_leftCanonical {d D : ℕ} [DecidableEq (Fin D)] (A : MPSTensor d D) (S σ : Matrix (Fin D) (Fin D) ℂ) (hS_inv : S.det ≠ 0) (hStS : S.conjTranspose * S = σ) (hfix : (MPSTensor.transferMap fun (i : Fin d) => (A i).conjTranspose) σ = σ) :
    ∑ i : Fin d, (S * A i * S⁻¹).conjTranspose * (S * A i * S⁻¹) = 1

    Left-canonical alias of gauged_tracePreserving.
