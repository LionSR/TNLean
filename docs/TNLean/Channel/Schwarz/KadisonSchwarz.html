    Kadison–Schwarz inequality for completely positive maps #

    The Kadison–Schwarz inequality (Wolf, Proposition 6.4) states that for any unital completely positive map E, we have E(X† X) ≥ E(X)† E(X) in the Loewner order.

    This file also proves the Hilbert–Schmidt contraction property.

    Main results #

    • kadison_schwarz: the KS inequality for unital Kraus maps
    • kadison_schwarz_adjoint: KS for the adjoint channel (MPS version)
    • hilbertSchmidt_contraction: HS norm contraction for unital TP maps

    References #

    • [M. Wolf, Quantum Channels & Operations: Guided Tour, Proposition 6.4][Wolf2012QChannels]

                theorem KadisonSchwarz.kadison_schwarz {d D : ℕ} (K : Fin d → Matrix (Fin D) (Fin D) ℂ) (h_unital : IsUnitalKraus K) (X : Matrix (Fin D) (Fin D) ℂ) :
                (krausMap K (X.conjTranspose * X) - (krausMap K X).conjTranspose * krausMap K X).PosSemidef

                Kadison-Schwarz inequality (Wolf, Proposition 6.4).

                For a unital CP map E(X) = ∑ᵢ Kᵢ X Kᵢ† with ∑ᵢ Kᵢ Kᵢ† = I, we have E(X† X) ≥ E(X)† E(X) in the Loewner order, i.e., E(X† X) - E(X)† E(X) is positive semidefinite.

                The proof uses the 2×2 block matrix / Schur complement argument: the block matrix [[X†X, X†], [X, I]] is PSD (= vv† with v = [X†, I]ᵀ), and applying the unital CP map blockwise preserves PSD-ness (by 2-positivity of CP maps). The Schur complement of the (2,2)-block I gives the result.

                theorem KadisonSchwarz.kadison_schwarz_adjoint {d D : ℕ} (K : Fin d → Matrix (Fin D) (Fin D) ℂ) (h_tp : IsTPKraus K) (X : Matrix (Fin D) (Fin D) ℂ) :
                (krausAdjointMap K (X.conjTranspose * X) - (krausAdjointMap K X).conjTranspose * krausAdjointMap K X).PosSemidef

                Kadison-Schwarz for the adjoint channel (MPS version).

                If ∑ᵢ Kᵢ† Kᵢ = I (the TP / MPS normalization condition), then the adjoint Kraus map E*(X) = ∑ᵢ Kᵢ† X Kᵢ satisfies E*(X† X) ≥ E*(X)† E*(X).

                This is the version most directly useful for MPS transfer matrix analysis. The proof reduces to kadison_schwarz applied to the adjoint operators Kᵢ†, which satisfy the unitality condition ∑ᵢ Kᵢ† (Kᵢ†)† = ∑ᵢ Kᵢ† Kᵢ = I.

                theorem KadisonSchwarz.hilbertSchmidt_contraction {d D : ℕ} (K : Fin d → Matrix (Fin D) (Fin D) ℂ) (h_unital : IsUnitalKraus K) (h_tp : IsTPKraus K) (X : Matrix (Fin D) (Fin D) ℂ) :
                ((krausMap K X).conjTranspose * krausMap K X).trace ≤ (X.conjTranspose * X).trace

                Hilbert-Schmidt contraction for unital CP maps.

                For a unital CP map E, we have tr(E(X)† E(X)) ≤ tr(X† X). This says E is a contraction in the Hilbert-Schmidt (Frobenius) norm.

                Proof: By Kadison-Schwarz, E(X†X) - E(X)†E(X) ≥ 0, so tr(E(X†X)) ≥ tr(E(X)†E(X)). If additionally E is trace-preserving, then tr(E(X†X)) = tr(X†X), giving the result.

                source
                theorem KadisonSchwarz.hilbertSchmidt_contraction_adjoint {d D : ℕ} (K : Fin d → Matrix (Fin D) (Fin D) ℂ) (h_tp : IsTPKraus K) (h_unital : IsUnitalKraus K) (X : Matrix (Fin D) (Fin D) ℂ) :
                ((krausAdjointMap K X).conjTranspose * krausAdjointMap K X).trace ≤ (X.conjTranspose * X).trace

                HS contraction for the adjoint channel (MPS version).

                If ∑ Kᵢ† Kᵢ = I (TP / left-canonical) and ∑ Kᵢ Kᵢ† = I (unital / right-canonical), then the adjoint map E*(X) = ∑ Kᵢ† X Kᵢ contracts the Hilbert-Schmidt norm: tr(E*(X)† E*(X)) ≤ tr(X† X).

                Note: both conditions are needed — Kadison-Schwarz uses unitality of E* (which follows from TP of E), while the trace computation uses TP of E* (which requires unitality of E).
