    Uniqueness properties of concrete Naimark dilations #

    This file provides additional API around TNLean.Channel.POVM.

    Main definitions #

    • POVM.IsNaimarkDilation — a packaged predicate for a Naimark dilation.

    Main results #

    • POVM.isNaimarkDilation_naimark — the explicit dilation from POVM.lean satisfies the packaged IsNaimarkDilation predicate.
    • POVM.exists_isometry_mul_naimarkIsometry_of_recovery — any dilation matrix recovering the POVM through the canonical projectors factors through the canonical Naimark isometry via an isometry on the dilated space.

    References #

    • [M. Wolf, Quantum Channels & Operations: Guided Tour, Ch. 2][Wolf2012QChannels]
