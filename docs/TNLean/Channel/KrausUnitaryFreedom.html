    Unitary freedom of Kraus representations #

    This file restates the directional Kraus-freedom lemmas already proved in TNLean.Channel.KrausRepresentation and TNLean.Channel.KrausFreedom as iff statements matching Wolf Theorem 2.18.

    Main results #

    • kraus_isometry_freedom_iff — two finite Kraus families define the same completely positive map if and only if, after padding the smaller family with zeros, they are related by an isometric mixing matrix.
    • kraus_unitary_freedom_iff — same-size Kraus families define the same map if and only if they are related by a unitary mixing matrix.

    References #

    • [M. Wolf, Quantum Channels & Operations: Guided Tour, Thm 2.18][Wolf2012QChannels]

    theorem kraus_isometry_freedom_iff {D : ℕ} {ι₁ : Type u_1} {ι₂ : Type u_2} [Fintype ι₁] [Fintype ι₂] [DecidableEq ι₂] (B : ι₁ → Matrix (Fin D) (Fin D) ℂ) (A : ι₂ → Matrix (Fin D) (Fin D) ℂ) (hCard : Fintype.card ι₂ ≤ Fintype.card ι₁) :
    (∀ (X : Matrix (Fin D) (Fin D) ℂ), ∑ α : ι₁, B α * X * (B α).conjTranspose = ∑ j : ι₂, A j * X * (A j).conjTranspose) ↔ ∃ (V : Matrix ι₁ ι₂ ℂ), V.conjTranspose * V = 1 ∧ ∀ (α : ι₁), B α = ∑ j : ι₂, V α j • A j

    Wolf Thm. 2.18 (isometric form): two finite Kraus families define the same completely positive map if and only if, after padding the smaller family with zeros, they are related by an isometric mixing matrix.

    theorem kraus_unitary_freedom_iff {D : ℕ} {ι : Type u_1} [Fintype ι] [DecidableEq ι] (B A : ι → Matrix (Fin D) (Fin D) ℂ) :
    (∀ (X : Matrix (Fin D) (Fin D) ℂ), ∑ α : ι, B α * X * (B α).conjTranspose = ∑ j : ι, A j * X * (A j).conjTranspose) ↔ ∃ (U : ↥(Matrix.unitaryGroup ι ℂ)), ∀ (α : ι), B α = ∑ j : ι, ↑U α j • A j

    Wolf Thm. 2.18 (unitary form): if two Kraus families have the same finite index type, then they define the same completely positive map if and only if they are related by a unitary mixing matrix.
