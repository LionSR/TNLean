    Heisenberg-dual multiplicativity from determinant saturation #

    This file presents the Kadison--Schwarz trace-summing argument in the Heisenberg picture used in the forward direction of Wolf Theorem 6.1(2).

    Starting from a Kraus representation of a CPTP map T with ‖channelDet T‖ = 1, we show that the Heisenberg dual has determinant of modulus 1, that equality holds in Kadison--Schwarz on the standard matrix basis, and hence that each Kraus operator commutes with the Heisenberg dual on all of M_d(ℂ). The conclusion is that the Heisenberg dual is multiplicative.

    Main statements #

    • ChannelDeterminant.Internal.heisenberg_dual_multiplicative — determinant saturation forces the Heisenberg dual to be multiplicative on all matrices.

    theorem ChannelDeterminant.Internal.heisenberg_dual_multiplicative {d : ℕ} [NeZero d] {T : MatrixEnd d} (_hT : IsChannel T) (hdet : ‖channelDet T‖ = 1) (_hall : ∀ (μ : ℂ), Module.End.HasEigenvalue T μ → ‖μ‖ = 1) {r : ℕ} (K : Fin r → MatrixAlg d) (hK : ∀ (X : MatrixAlg d), T X = ∑ i : Fin r, K i * X * Matrix.conjTranspose (K i)) (hK_tp : ∑ i : Fin r, Matrix.conjTranspose (K i) * K i = 1) (Td : MatrixEnd d) (hTd : ∀ (X : MatrixAlg d), Td X = ∑ i : Fin r, Matrix.conjTranspose (K i) * X * K i) (M N : MatrixAlg d) :
    Td (M * N) = Td M * Td N

    If a CPTP map saturates the determinant bound, then its Heisenberg dual is multiplicative on all of M_d(ℂ). This is the analytic core of Wolf Theorem 6.1(2).
