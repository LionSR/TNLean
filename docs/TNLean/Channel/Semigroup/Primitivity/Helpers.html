    theorem peripheral_card_le_finrank {D : ℕ} [NeZero D] (E : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ) :
    ⋯.toFinset.card ≤ Module.finrank ℂ (Matrix (Fin D) (Fin D) ℂ)

    The peripheral spectrum of an irreducible finite-dimensional channel has cardinality at most dim(Mat).

    This is proved by choosing one nonzero eigenvector for each peripheral eigenvalue and using the linear independence of eigenvectors corresponding to distinct eigenvalues.

    theorem bounded_root_of_peripheral_closed_powers {D : ℕ} [NeZero D] (E : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ) (μ : ℂ) (hμ : μ ∈ peripheralEigenvalues E) (hclosed : ∀ (n : ℕ), μ ^ n ∈ peripheralEigenvalues E) :
    ∃ (p : ℕ), 0 < p ∧ p ≤ Module.finrank ℂ (Matrix (Fin D) (Fin D) ℂ) ∧ μ ^ p = 1

    If peripheral powers are all again peripheral, then the order of a peripheral eigenvalue is bounded by the dimension of the matrix space.

    This is the finite-dimensional pigeonhole step used to force a common period at a divisor time. The remaining input needed in the main proof is the closure of peripheral eigenvalues under powers for the chosen irreducible time slice.

    μ ^ n ∈ peripheralEigenvalues E

    Power-closure helper at an irreducible time slice.

    This is the only new semigroup-specific ingredient still missing from the refactor below: we need that after a similarity transform by a positive-definite fixed point, the irreducible channel becomes unital with an adjoint fixed point, so Wolf's peripheral-power closure theorem applies.

    The statement is substantially simpler than the original circular sorry: it is a pure channel lemma, independent of continuous-time propagation or generator kernels.
