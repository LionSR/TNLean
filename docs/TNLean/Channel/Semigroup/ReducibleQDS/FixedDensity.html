    Fixed Density ↔ Kernel Element (Wolf Prop 7.6, (1) ↔ (2)) and (1) → (3) #

    This file proves the equivalence between conditions (1) and (2) of Wolf Proposition 7.6, and the implication (1) → (3).

    (1) ↔ (2): Fixed density ↔ kernel element #

    This is the simplest equivalence, following directly from the bridge L X = 0 ↔ exp(tL) X = X ∀ t ≥ 0 (Wolf §7.1, formalized in Kernel.lean).

    (1) → (3): Fixed density → invariant compression #

    For a fixed density matrix ρ₀, every channel expSemigroup L t preserves the support of ρ₀. Taking the support projection therefore produces a nontrivial compression invariant under the whole semigroup.
