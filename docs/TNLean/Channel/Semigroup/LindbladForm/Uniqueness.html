    Lindblad Form — Uniqueness of traceless GKSL decompositions #

    This file formalizes the uniqueness direction of Wolf Proposition 7.4 (item 2): if two Lindblad / generator decompositions define the same generator and both use traceless Kraus operators, then

    • the dissipative CP parts coincide, and
    • the drift matrices agree up to an imaginary scalar multiple of the identity.
