    Dissipative drift semigroups #

    This file isolates the explicit semigroup generated by the drift term ρ ↦ -κρ - ρκ† appearing in the GKSL decomposition.

    Main results #

    • dissipativeDrift — the linear drift generator ρ ↦ -κρ - ρκ†
    • expSemigroup_dissipativeDrift_apply — exp(t · dissipativeDrift κ)(ρ) = e^{-tκ} ρ e^{-tκ†}
    • isCPMap_expSemigroup_dissipativeDrift — the drift semigroup is completely positive

    References #

    • [M. Wolf, Quantum Channels & Operations: Guided Tour, §7.1.2][Wolf2012QChannels]
