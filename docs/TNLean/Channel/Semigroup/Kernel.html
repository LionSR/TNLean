    Kernel / fixed-point connections for semigroup generators #

    This file develops witness-based notions for the mixing theory of quantum semigroups and proves the basic connection between the generator kernel and fixed points of the exponential semigroup.

    Main definitions #

    • HasFaithfulStationaryState: the generator kernel contains a positive-definite density matrix.
    • HasSimpleKernel: a positive-definite matrix spans the whole kernel.
    • HasSimpleFaithfulKernel: the kernel is one-dimensional and generated by a faithful density matrix.
    • RelaxesTo, IsRelaxingQDS: witness-based relaxation predicates for semigroup mixing.

    Main results #

    • generator_apply_eq_zero_iff_expSemigroup_fixed_nonneg: L X = 0 iff expSemigroup L t X = X for all t ≥ 0.
    • generator_apply_eq_zero_iff_fixed_nonneg: the same connection for any semigroup identified with expSemigroup.

    These statements are the semigroup-side kernel/fixed-point connections needed for Wolf Chapter 7.
