    Generator Definitions for Quantum Dynamical Semigroups #

    This file defines the core structures and predicates for generators of quantum dynamical semigroups, following Wolf §7.1.

    Main definitions #

    • Matrix.commutator — the commutator [A, B] = AB - BA.
    • Matrix.anticommutator — the anticommutator {A, B}₊ = AB + BA.
    • GeneratorDecomp — a generator decomposition L(ρ) = φ(ρ) - κρ - ρκ† with φ completely positive and κ ∈ M_d(ℂ).
    • IsCCP — conditional complete positivity: L admits a GeneratorDecomp.
    • IsTraceAnnihilating — tr(L(ρ)) = 0 for all ρ.
    • GeneratorDecomp.isTraceConstraint — φ*(𝟙) = κ + κ†.

    Main results #

    • GeneratorDecomp.isCCP — any decomposition witnesses CCP.
    • GeneratorDecomp.traceAnnihilating_of_traceConstraint — φ*(𝟙)=κ+κ† ⟹ TA.

    References #

    • [M. Wolf, Quantum Channels & Operations: Guided Tour, §7.1.2, Eq. 7.14, 7.20]

        The (φ, κ) generator decomposition (Wolf Eq. 7.14) #

        source
        structure GeneratorDecomp (D : ℕ) :
        Type

        A generator decomposition represents a linear map as L(ρ) = φ(ρ) - κρ - ρκ† where φ is completely positive and κ ∈ M_d(ℂ).

        This is the canonical form for generators of CP semigroups (Wolf Eq. 7.14).

        • φ : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ

          The completely positive part.

        • κ : Matrix (Fin D) (Fin D) ℂ

          The matrix κ in the dissipative term.

        • φ_cp : IsCPMap self.φ

          Witness that φ is completely positive.

            Conditional complete positivity (Wolf Prop 7.2) #

            source
            def IsCCP {D : ℕ} (L : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ) :
            Prop

            A linear map L : M_d(ℂ) → M_d(ℂ) is conditionally completely positive (CCP) if it can be written as L(ρ) = φ(ρ) - κρ - ρκ† for some CP map φ and matrix κ. This is Wolf Proposition 7.2, condition 1.

            Equations
            • IsCCP L = ∃ (G : GeneratorDecomp D), L = G.toLinearMap

              Trace-annihilating condition #

              source
              def IsTraceAnnihilating {D : ℕ} (L : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ) :
              Prop

              A linear map is trace-annihilating if tr(L(ρ)) = 0 for all ρ. This is the infinitesimal version of trace preservation: if T_t = exp(tL) is trace-preserving for all t ≥ 0, then L is trace-annihilating.

              Equations
              • IsTraceAnnihilating L = ∀ (ρ : Matrix (Fin D) (Fin D) ℂ), (L ρ).trace = 0
