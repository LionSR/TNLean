    Corners and compression maps for cyclic decompositions #

    This file contains the ambient corner-algebra constructions used in the cyclic decomposition of periodic irreducible channels.

    Main definitions #

    • PreservesCorner — invariance of a projection corner under a linear map.
    • cornerSubmodule — the corner algebra P \cdot M_D(\mathbb{C}) \cdot P as a submodule of the ambient matrix algebra.
    • cornerRestriction — restriction of a linear map to an invariant corner.
    • cornerCompressionLinearEquiv — the shared linear equivalence between a matrix algebra and a projection corner.
    • cornerRank — the matrix size of a projection corner.

    Main statements #

    • cornerRank_eq_trace — the corner rank agrees with the trace of the projection.

    References #

    • [M. Wolf, Quantum Channels & Operations: Guided Tour, Thm. 6.6]
