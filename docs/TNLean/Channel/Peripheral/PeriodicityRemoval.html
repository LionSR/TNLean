    Periodicity removal #

    This file contains small “common exponent” / “periodicity removal” lemmas for peripheral (eigen)value arguments, in a way that is independent of the CP / multiplicative-domain details.

    The intended use is the standard step: if all peripheral eigenvalues are roots of unity, then one can pick a common exponent p killing them, and deduce that the peripheral eigenvalues of E ^ p collapse to {1}.

    These lemmas align with arXiv:1606.00608 Appendix A (peripheral spectrum / period).

    Part 1: a common exponent for a finite set of roots of unity #

    source
    theorem exists_common_power_eq_one_of_finite (s : Finset ℂ) (h : ∀ μ ∈ s, ∃ (p : ℕ), 0 < p ∧ μ ^ p = 1) :
    ∃ (p : ℕ), 0 < p ∧ ∀ μ ∈ s, μ ^ p = 1

    If every μ ∈ s is a root of unity (witnessed by some positive power), then there is a single positive exponent p such that μ ^ p = 1 for all μ ∈ s.

    We take the Finset.lcm of the individual exponents.

    Part 2: powering kills the peripheral spectrum given a common exponent #

    source
    theorem peripheralEigenvalues_pow_eq_singleton {D : ℕ} [NeZero D] (E : Matrix (Fin D) (Fin D) ℂ →ₗ[ℂ] Matrix (Fin D) (Fin D) ℂ) {p : ℕ} (hp : 0 < p) (hper : ∀ μ ∈ peripheralEigenvalues E, μ ^ p = 1) (ρ : Matrix (Fin D) (Fin D) ℂ) (hfix : E ρ = ρ) (hne : ρ ≠ 0) :
    peripheralEigenvalues (E ^ p) = {1}

    If every peripheral eigenvalue of E satisfies μ ^ p = 1 for a fixed p > 0, and E has a nonzero fixed point, then the peripheral eigenvalues of E ^ p are exactly {1}.

    The key step is spectral mapping: spectrum (E ^ p) = (· ^ p) '' spectrum E.
