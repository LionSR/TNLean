    Kraus representation theorem (Wolf Ch. 2, Thm 2.1) #

    This file proves key properties of the Kraus representation of completely positive maps T(A) = ∑ⱼ Kⱼ A Kⱼ†.

    Main results (Wolf Thm 2.1) #

    • kraus_sum_conjTranspose_mul_of_tp — TP ⟹ ∑ᵢ Kᵢ†Kᵢ = 𝟙
    • kraus_tp_of_sum_conjTranspose_mul — ∑ᵢ Kᵢ†Kᵢ = 𝟙 ⟹ TP
    • kraus_sum_mul_conjTranspose_of_unital — unital ⟹ ∑ᵢ Kᵢ Kᵢ† = 𝟙
    • kraus_same_map_of_unitary_combination — unitary freedom (sufficient direction)

    Design notes #

    In the current TNLean codebase, IsCPMap is defined as the existence of a Kraus representation. The Choi–Jamiolkowski isomorphism (ChoiJamiolkowski.cp_iff_choi_posSemidef) provides the equivalence with positivity of the Choi matrix.

    References #

    • [M. Wolf, Quantum Channels & Operations: Guided Tour, Thm 2.1][Wolf2012QChannels]

    Unitary freedom in Kraus operators (Thm 2.1, item 4) #

    source
    theorem kraus_same_map_of_isometry_combination {D : ℕ} {ι₁ : Type u_1} {ι₂ : Type u_2} [Fintype ι₁] [Fintype ι₂] [DecidableEq ι₂] (K : ι₁ → Matrix (Fin D) (Fin D) ℂ) (K' : ι₂ → Matrix (Fin D) (Fin D) ℂ) (W : Matrix ι₁ ι₂ ℂ) (hW : W.conjTranspose * W = 1) (hK : ∀ (j : ι₁), K j = ∑ l : ι₂, W j l • K' l) (X : Matrix (Fin D) (Fin D) ℂ) :
    ∑ j : ι₁, K j * X * (K j).conjTranspose = ∑ l : ι₂, K' l * X * (K' l).conjTranspose

    Thm 2.1 item 4 (isometry freedom, sufficient direction): If W is an isometry (Wᴴ W = 1) and Kⱼ = ∑ₗ Wⱼₗ K̃ₗ, then {Kⱼ} and {K̃ₗ} define the same map: ∑ⱼ Kⱼ X Kⱼ† = ∑ₗ K̃ₗ X K̃ₗ†.

    This rectangular form specializes to the usual unitary-freedom statement when the output and input Kraus index sets have the same cardinality. The index types are arbitrary finite types (with decidable equality on the inner one), so callers with Fin-indexed or general Fintype-indexed Kraus families can reuse the same lemma.

    theorem kraus_same_map_of_unitary_combination {D : ℕ} {ι : Type u_1} [Fintype ι] [DecidableEq ι] (K K' : ι → Matrix (Fin D) (Fin D) ℂ) (U : Matrix ι ι ℂ) (hU : U.conjTranspose * U = 1) (hK : ∀ (j : ι), K j = ∑ l : ι, U j l • K' l) (X : Matrix (Fin D) (Fin D) ℂ) :
    ∑ j : ι, K j * X * (K j).conjTranspose = ∑ l : ι, K' l * X * (K' l).conjTranspose

    Thm 2.1 item 4 (unitary freedom, sufficient direction): If U is unitary (Uᴴ U = 1) and Kⱼ = ∑ₗ Uⱼₗ K̃ₗ, then {Kⱼ} and {K̃ₗ} define the same map: ∑ⱼ Kⱼ X Kⱼ† = ∑ₗ K̃ₗ X K̃ₗ†.

    theorem kraus_transition_unitary_of_hs_orthonormal {D r : ℕ} (K K' : Fin r → Matrix (Fin D) (Fin D) ℂ) (U : Matrix (Fin r) (Fin r) ℂ) (hK : ∀ (j : Fin r), K j = ∑ l : Fin r, U j l • K' l) (horthK : ∀ (i j : Fin r), ((K j).conjTranspose * K i).trace = if i = j then 1 else 0) (horthK' : ∀ (i j : Fin r), ((K' j).conjTranspose * K' i).trace = if i = j then 1 else 0) :
    U.conjTranspose * U = 1

    A converse-style uniqueness lemma for the Kraus transition matrix: if two same-size Kraus families are related by a mixing matrix U, and both families are Hilbert–Schmidt orthonormal, then U is unitary.

    This isolates the linear-algebraic core used in Wolf Thm. 2.1 item 4: orthonormal Kraus decompositions have unitary change-of-coordinates.
