    Fixed points on compact retracts of finite-dimensional real normed spaces #

    This file derives a retract-based fixed-point theorem from the vendored Brouwer fixed-point theorem on finite products of simplices.

    The main result is fixedPoint_of_compact_retract: if a compact set K in a finite-dimensional real normed space is the image of a continuous retraction of the ambient space, then every continuous self-map of K has a fixed point.

    source
    theorem fixedPoint_of_compact_retract_fin {n : ℕ} [NeZero n] {K : Set (Fin n → ℝ)} (hK_comp : IsCompact K) {r : (Fin n → ℝ) → Fin n → ℝ} (hr_cont : Continuous r) (hr_maps : Set.MapsTo r Set.univ K) (hr_id : ∀ x ∈ K, r x = x) {f : (Fin n → ℝ) → Fin n → ℝ} (hf_cont : ContinuousOn f K) (hf_maps : Set.MapsTo f K K) :
    ∃ x ∈ K, f x = x

    A compact subset of Fin n → ℝ that is a retract of the ambient space has the fixed-point property.
