    Square-lattice graph for the normal PEPS proof #

    This file records the nearest-neighbor graph on the finite rectangular coordinate lattice used in the normal PEPS proof.

    References #

    • [Molnár, Garre-Rubio, Pérez-García, Schuch, Cirac, Normal projected entangled pair states generating the same state, arXiv:1804.04964, Section 3, Theorem 3]
    source
    @[implicit_reducible]
    instance TNLean.PEPS.instLinearOrderSquareLatticeVertex (width height : ℕ) :
    LinearOrder (SquareLatticeVertex width height)

    The coordinate vertex set is ordered lexicographically. The project-level Edge type uses an ambient linear order to orient undirected graph edges, so this instance lets the square-lattice graph use the same edge type as the general PEPS development.
