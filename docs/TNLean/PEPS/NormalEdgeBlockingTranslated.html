    Translated edge blockings in the normal PEPS proof #

    This file records the coordinate-origin-parametric horizontal and vertical edge-blocking pictures used after the normalized one-edge constructions.

    References #

    • [Molnár, Garre-Rubio, Pérez-García, Schuch, Cirac, Normal projected entangled pair states generating the same state, arXiv:1804.04964, Section 3, Theorem 3]

    A translated horizontal edge in the coordinate frame of the normal edge-blocking picture.
