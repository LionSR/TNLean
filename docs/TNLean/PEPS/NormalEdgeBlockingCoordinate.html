    Coordinate edge blockings in the normal PEPS proof #

    This file records the normalized coordinate edge blocking used in the proof of the normal square-lattice PEPS theorem.

    References #

    • [Molnár, Garre-Rubio, Pérez-García, Schuch, Cirac, Normal projected entangled pair states generating the same state, arXiv:1804.04964, Section 3, Theorem 3]

            The normalized horizontal edge has the red/blue/complement blocking data used in the proof of the normal square-lattice theorem.
