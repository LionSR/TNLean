# Generality Analysis: FT-MPS Literature vs. Lean Formalization

## A. The Most General FT-MPS in the Literature

### A.1 Cirac et al. 2017 (Annals of Physics, arXiv:1606.00608)

The **Fundamental Theorem of Matrix Product Vectors** appears as **Theorem 2.1** (labeled `\label{thm1}`) in the Annals paper. It is stated for tensors in **canonical form (CF)**.

**Definition (Canonical Form — CF).** A tensor $A$ is in CF if:
$$A^i = \bigoplus_{k=1}^r \mu_k A^i_k$$
where each $A_k$ is a **normal tensor (NT)**, meaning:
1. No non-trivial projector $P$ satisfies $A^i P = P A^i P$ (no invariant subspaces — irreducibility of the matrix algebra generated by $\{A^i_k\}$).
2. The associated CP map $\mathcal{E}_k(X) = \sum_i A^i_k X (A^i_k)^\dagger$ has spectral radius 1, with **unique eigenvalue of magnitude 1** (primitivity — no $p$-periodic components).

**Theorem 2.1 (FT for proportional MPVs).** Let $A$ and $B$ be two tensors in CF, with BNT (Bases of Normal Tensors) $A^i_{k_a}$ and $B^i_{k_b}$ ($k_{a,b} = 1, \ldots, g_{a,b}$), respectively. If for all $N$, $A$ and $B$ generate MPVs that are **proportional** to each other, then:
- (i) $g_a = g_b =: g$;
- (ii) for all $k$ there exists a $j_k$, phases $\phi_k$, and non-singular matrices $X_k$ such that $B^i_k = e^{i\phi_k} X_k A^i_{j_k} X_k^{-1}$.

**Corollary 2.2 (FT for equal MPVs).** If two tensors $A$ and $B$ in CF generate the same MPV for all $N$, then:
- (i) the dimensions of $A^i$ and $B^i$ coincide;
- (ii) there exists an invertible matrix $X$ such that $A^i = X B^i X^{-1}$.

**Corollary (CFII — unitary gauge).** If the tensors are in CFII (i.e., CF + trace-preserving CP map + full-rank diagonal fixed point), then $X$ and $X_k$ above are **unitary**.

### A.2 Cirac et al. 2021 (Reviews of Modern Physics, arXiv:2011.12127)

The RMP review states the same theorem in Section IV as **Theorem 4.4** (`\label{thm1}`) and **Corollary 4.5** (`\label{II_cor2}`), with identical content to the Annals paper. (The numbering is 4.4/4.5 because Definitions 4.1–4.2 and Proposition 4.3 precede them in the shared counter.) Additionally, it states:

**Theorem 4.6** (`\label{thm:fundamental-general}`) — the most general version, from de las Cuevas et al. (arXiv:1707.05104). This handles **irreducible (not necessarily primitive) blocks**, i.e., blocks whose CP maps may have $p$-periodic eigenvalues $e^{2\pi i q/p}$:

Let $A$ and $B$ be tensors in block diagonal form $A^i = \bigoplus_k \mu_k A^i_k$ where each block's CP map is **irreducible** (but not necessarily primitive). If $|V^{(N)}(A)\rangle = |V^{(N)}(B)\rangle$ for all $N$, then there exists a diagonal matrix $Z$ and an invertible matrix $Y$ so that:
- (i) $[Z, A^i] = 0$ for all $i$,
- (ii) $Z A^i = Y B^i Y^{-1}$ for all $i$,
- (iii) $|V^{(N)}(A)\rangle = |V^{(N)}(ZA)\rangle$ for all $N$.

The matrix $Z$ accounts for the extra degree of freedom from $p$-periodic blocks: if a block $A_k$ has period $p$, its MPV vanishes except when $N$ is a multiple of $p$, so multiplying by any $p$-th root of unity doesn't change the family.

### A.3 Hierarchy of Generality

From most general to most specific:

1. **Theorem 4.6 in RMP / de las Cuevas et al. (arXiv:1707.05104)**: Irreducible blocks (possibly periodic), equal MPVs, includes diagonal $Z$ matrix. Most general.
2. **Theorem 4.4 in RMP / Theorem 2.1 in Annals (Cirac et al. 2017)**: Primitive (normal) blocks, proportional MPVs, BNT permutation with phases. Requires blocking to remove periodicity.
3. **Corollary 4.5 in RMP / Corollary 2.2 in Annals**: Primitive blocks, equal MPVs. Most commonly used version.
4. **Single-block / injective case**: One block ($r = 1$), primitive, equal MPVs. The cleanest statement: $A^i = X B^i X^{-1}$.

### A.4 Key Structural Steps in the Literature Proof

1. **Canonical form existence**: Any tensor $B$ can be brought into CF (after blocking) by decomposing into irreducible invariant subspaces and removing off-diagonal blocks.
2. **Linear independence of normal MPVs**: For $N$ large enough, the MPVs $|V^{(N)}(A_j)\rangle$ from BNT elements are linearly independent (Lemma 1 in the Annals paper, using Gram matrix convergence).
3. **Block separation / peeling**: Using the strict ordering of $|\mu_k|$ and Perron–Frobenius theory (spectral gap of mixed transfer operators), show that $V(A_k) \propto V(B_{j_k})$ for some permutation.
4. **Single-block FT**: For each normal tensor pair with $V(A) = e^{i\phi N} V(B)$, the linear extension $T(A^i) = B^i$ is a multiplicative algebra automorphism, hence inner by Skolem–Noether. This gives $B^i = e^{i\phi} X A^i X^{-1}$.
5. **Assembly**: Combine per-block gauge equivalences into a global block-diagonal gauge transform.

## B. What Our Lean Formalization Proves

### B.1 Single-Block FT (`FundamentalTheorem.lean`)

```
theorem fundamentalTheorem_singleBlock {A B : MPSTensor d D}
    (hA : IsInjective A) (hAB : SameMPV A B) : GaugeEquiv A B
```

**Hypotheses**:
- `IsInjective A`: The matrices $\{A^i\}$ span $M_D(\mathbb{C})$ (algebraic injectivity).
- `SameMPV A B`: $\mathrm{tr}(A^{i_1} \cdots A^{i_N}) = \mathrm{tr}(B^{i_1} \cdots B^{i_N})$ for all $N$ and all configurations $\sigma$.

**Conclusion**: `GaugeEquiv A B` — there exists $X \in GL(D, \mathbb{C})$ such that $B^i = X A^i X^{-1}$ for all $i$.

**Proof method**: Linear extension → multiplicativity → Skolem–Noether. **Sorry-free.** ✅

### B.2 Multi-Block Assembly (`FundamentalTheoremMulti.lean`, `FundamentalTheoremComplete.lean`)

```
theorem fundamentalTheorem_multiBlock_full
    (μ : Fin r → ℂ)
    (A B : (k : Fin r) → MPSTensor d (dim k))
    (hA : ∀ k, IsInjective (A k))
    (hSame : ∀ k, SameMPV (A k) (B k)) :
    (∀ k, GaugeEquiv (A k) (B k)) ∧
    GaugeEquiv (toTensorFromBlocks μ A) (toTensorFromBlocks μ B)
```

**Hypotheses**: Per-block injectivity + per-block `SameMPV`.
**Conclusion**: Per-block + global gauge equivalence.
**Sorry-free.** ✅

### B.3 Block Separation (`CanonicalFormSep.lean`)

```
theorem per_block_sameMPV_of_canonical_form
    (μ : Fin r → ℂ)
    (A B : (k : Fin r) → MPSTensor d (dim k))
    (hA : IsCanonicalForm μ A)
    (hB_inj : ∀ k, IsInjective (B k))
    (hB_ds : ∀ k, ∑ i, (B k i)ᴴ * (B k i) = 1)
    (hSame₂ : SameMPV₂ (toTensorFromBlocks μ A) (toTensorFromBlocks μ B)) :
    ∀ k, SameMPV (A k) (B k)
```

Where `IsCanonicalForm μ A` requires:
- `block_injective`: Each block $A_k$ spans $M_{D_k}(\mathbb{C})$.
- `ds_gauge`: $\sum_i (A_k^i)^\dagger A_k^i = I$ for each block.
- `mu_strict_anti`: $\|μ_0\| > \|μ_1\| > \cdots > \|μ_{r-1}\|$ (strict ordering by norm).
- `mu_ne_zero`: $μ_k \neq 0$ for all $k$.
- `overlap_tendsto_one`: $\lim_{N \to \infty} \text{mpvOverlap}(A_k, A_k, N) = 1$ for all $k$ (aperiodicity/primitivity consequence).

Additionally, $B$ satisfies:
- `hB_inj`: Each block $B_k$ is injective.
- `hB_ds`: Each block $B_k$ satisfies DS gauge $\sum_i (B_k^i)^\dagger B_k^i = I$.

**Proof method**: Peeling lemma (exponential decay) + overlap decay (`mpvOverlap_tendsto_zero` from spectral theory) + induction on number of blocks. **Sorry-free.** ✅

### B.4 End-to-End Canonical Form FT (`CanonicalFormSep.lean`)

```
theorem fundamentalTheorem_canonicalForm_explicit
    (μ : Fin r → ℂ)
    (A B : (k : Fin r) → MPSTensor d (dim k))
    (hA : IsCanonicalForm μ A)
    (hB_inj : ∀ k, IsInjective (B k))
    (hB_ds : ∀ k, ∑ i, (B k i)ᴴ * (B k i) = 1)
    (hSame₂ : SameMPV₂ (toTensorFromBlocks μ A) (toTensorFromBlocks μ B)) :
    ∃ (X : ∀ k, GL (Fin (dim k)) ℂ),
    ∀ k i, B k i = (X k) * A k i * (X k)⁻¹
```

**Sorry-free.** ✅

### B.5 Block Permutation Decomposition (`FundamentalTheoremComplete.lean`)

```
theorem fundamentalTheorem_multiBlock_decomposition
    (A B : (k : Fin r) → MPSTensor d (dim k))
    (hA : ∀ k, IsInjective (A k))
    (hSame : ∀ k, SameMPV (A k) (B k)) :
    ∃ (σ : Fin r ≃ Fin r) (hDeq : ∀ i, dim (σ i) = dim i)
      (X : ∀ i, GL (Fin (dim i)) ℂ), ...
```

This decomposes the Pi-algebra automorphism into a block permutation σ plus per-block inner automorphisms (gauge transforms). **Sorry-free.** ✅

## C. Gap Analysis

### C.1 Canonical Form Existence (Going from Arbitrary Tensor to CF)

**Literature**: Any tensor $B$ can be brought to CF after blocking. The algorithm:
1. Find irreducible invariant subspaces → block-diagonal decomposition.
2. Extract $\mu_k$ so that each CP map has spectral radius 1.
3. Block to remove $p$-periodic eigenvalues → each block becomes primitive (normal).

**Lean status**: **Not formalized.** The `IsCanonicalForm` structure is defined as a *predicate* on already-canonical tensors. There is no construction that takes an arbitrary tensor and produces one in CF.

**What's needed**:
- Invariant subspace decomposition for matrix $*$-algebras over $\mathbb{C}$.
- Perron–Frobenius theory for CP maps: existence and uniqueness of the Perron eigenvalue, spectral radius characterization.
- Blocking procedure: $C^{i_1,\ldots,i_p} = A^{i_1} \cdots A^{i_p}$ preserves MPV family.
- Period extraction: finding the period $p$ of the CP map and showing that $p$-blocking yields a primitive map.

**Difficulty**: **Hard.** This is a substantial body of algebra and functional analysis. Estimate: **1500–3000 lines** of new Lean code, building on Mathlib's spectral theory. Many intermediate steps (Jordan normal form for CP maps, peripheral spectrum structure, invariant subspace algorithms) are not yet in Mathlib.

### C.2 The Irreducible/Periodic Case (Period $p > 1$) — Theorem IV.4

**Literature**: Theorem IV.4 (de las Cuevas et al. 2017) handles blocks whose CP maps are irreducible but not primitive (have $p$-periodic eigenvalues $e^{2\pi i q/p}$). The extra degree of freedom is captured by a diagonal matrix $Z$ that commutes with all $A^i$ and satisfies $|V^{(N)}(A)\rangle = |V^{(N)}(ZA)\rangle$.

**Lean status**: **Not formalized.** The `IsCanonicalForm` structure explicitly assumes primitivity via `overlap_tendsto_one`. There is no handling of $p$-periodic blocks.

**What's needed**:
- Define `IsIrreducibleBlock` (CP map irreducible, not necessarily primitive).
- Formalize the periodic eigenvalue structure: if $\mathcal{E}$ is irreducible, peripheral eigenvalues are exactly $\{e^{2\pi i q/p} : q = 0, \ldots, p-1\}$ for some period $p$ dividing $D$.
- Construct the $Z$ matrix from the periodic structure.
- Prove that multiplying a $p$-periodic block by a $p$-th root of unity preserves the MPV family.
- Generalize the block separation argument to handle the $Z$-degree of freedom.

**Difficulty**: **Hard.** Estimate: **1000–2000 lines.** Requires significant spectral theory for positive maps that's partially in Mathlib but needs substantial extension.

### C.3 The Proportional MPV Case ($V(A) \propto V(B)$ vs. $V(A) = V(B)$)

**Literature**: Theorem IV.2/2.1 handles the case where $V^{(N)}(A)$ and $V^{(N)}(B)$ are merely **proportional** for each $N$ (with $N$-dependent proportionality constant). The conclusion includes a BNT permutation $j_k$ and per-block phases $\phi_k$: $B^i_k = e^{i\phi_k} X_k A^i_{j_k} X_k^{-1}$.

**Lean status**: **Only the equal case ($V(A) = V(B)$) is formalized.** The type `ProportionalMPV₂` is defined in `Defs.lean` but never used. The current formalization has `SameMPV₂` (exact equality) throughout.

**What's needed**:
- Define and work with BNT (Basis of Normal Tensors) — partially done in `BasisNormal.lean` (Vandermonde separation), but the BNT structure itself (grouping blocks by gauge-phase equivalence classes) is not formalized.
- Extend the block separation core lemma to handle proportional MPVs: the peeling argument needs to track the proportionality constants and show they must factor as products of phases and weight ratios.
- Show that proportional MPVs with all-zero constant force equality of BNT.
- Handle the BNT permutation $j_k$ (blocks of $B$ match blocks of $A$ up to reordering).

**Difficulty**: **Medium-Hard.** Estimate: **800–1500 lines.** The main mathematical challenge is tracking the proportionality constants through the peeling argument and showing they must be pure phases times weight ratios.

### C.4 Block Permutation Handling

**Literature**: In the proportional case, blocks of $B$ need not appear in the same order as blocks of $A$. There is a permutation $j_k$ of the BNT indices.

**Lean status**: **Partially formalized.** The `piAlgEquiv_decomposition` theorem produces a permutation $σ : \mathrm{Fin}\, r \simeq \mathrm{Fin}\, r$ and per-block gauge transforms. However, this is for the Pi-algebra automorphism that arises when per-block `SameMPV` is already established. The BNT-level permutation (matching which block of $A$ corresponds to which block of $B$) in the proportional case is **not** handled.

**What's needed for the proportional case**: Extend the block separation to identify which block of $B$ corresponds to which BNT element of $A$, tracking the phase and gauge transform.

**Difficulty**: **Medium.** Estimate: **300–600 lines** (on top of the proportional MPV infrastructure from C.3).

### C.5 Unitary Gauge (CFII)

**Literature**: Corollary in both papers states that if tensors are in CFII (CF + trace-preserving CP + full-rank diagonal fixed point), the gauge matrices $X_k$ can be taken **unitary**.

**Lean status**: **Not formalized as a separate theorem**, but the DS gauge hypothesis ($\sum_i A_i^\dagger A_i = I$) is already assumed in `IsCanonicalForm`. The step from "invertible $X$" to "unitary $X$" under CFII is not proved.

**What's needed**:
- Show that under CFII normalization, the fixed point of the CP map is $I$.
- Show that if both $A$ and $B$ are in CFII and $B^i = X A^i X^{-1}$, then $X$ can be replaced by a unitary $U$ (via polar decomposition: $X = U |X|$, and the fixed-point condition forces $|X| = I$).

**Difficulty**: **Medium.** Estimate: **200–400 lines.** Needs the uniqueness of the fixed point (which is partially formalized) and polar decomposition from Mathlib.

### C.6 `overlap_tendsto_one` Derivation from Primitivity

**Lean status**: The hypothesis `overlap_tendsto_one` (that the self-overlap converges to 1) is **assumed** in `IsCanonicalForm` rather than derived from a primitivity definition.

**What's needed**:
- Define `IsPrimitiveCP`: the CP map is irreducible and has no $p$-periodic eigenvalues (i.e., $\lambda = 1$ is the **unique** eigenvalue of modulus 1).
- Prove that primitivity + DS gauge implies $\text{mpvOverlap}(A, A, N) \to 1$ as $N \to \infty$.
- This requires: spectral radius theory for CP maps (partially in Mathlib), convergence of $\mathcal{E}^N$ to the projection onto the fixed point subspace (Perron–Frobenius for CP maps).

**Difficulty**: **Hard.** Estimate: **800–1200 lines.** This is deep spectral theory for completely positive maps, much of which is not yet in Mathlib.

### C.7 Assumptions on $B$

**Our formalization** assumes:
- `hB_inj`: All blocks of $B$ are injective.
- `hB_ds`: All blocks of $B$ satisfy DS gauge.

**Literature**: The FT is stated for $A$ and $B$ both in CF. The additional assumptions on $B$ are thus part of the CF hypothesis. However, the literature's CF definition does not require the *comparison* tensor $B$ to satisfy all the same conditions as $A$ — only that $B$ is also in CF (which includes its own block structure, weight ordering, etc.).

In our formalization, $B$ is given as a family of block tensors `B : (k : Fin r) → MPSTensor d (dim k)` with the **same** block structure (same number of blocks, same block dimensions) as $A$. The literature's Corollary IV.3 **concludes** this (same dimensions), but we **assume** it. This is an additional assumption not present in the literature's strongest statements.

**What's needed**: Handle the case where $B$ might have different block dimensions or different number of blocks, and prove they must match.

**Difficulty**: **Hard.** Estimate: **500–1000 lines.** This requires the full BNT theory.

### Summary of What Is vs. What Should Be

**What our formalization proves** (sorry-free):
> Given two families of block-diagonal tensors in CF (both with known block structure, DS gauge, injectivity, strict weight ordering, and primitivity overlap condition), if they generate the same MPV for all $N$, then each block pair is related by an invertible gauge transform: $B_k^i = X_k A_k^i X_k^{-1}$.

**What the literature's most general statement says** (Theorem IV.4):
> Given any two tensors whose CP maps are block-diagonal with irreducible (possibly periodic) blocks, if they generate the same MPV for all $N$, then they are related by a diagonal phase matrix $Z$ (commuting with $A$) and an invertible gauge $Y$: $Z A^i = Y B^i Y^{-1}$.

**Principal gaps** (in decreasing order of difficulty):
1. Canonical form existence from arbitrary tensors (not formalized at all)
2. Periodic/irreducible case (Theorem IV.4's $Z$ matrix)
3. Deriving primitivity consequences from spectral theory (currently assumed)
4. Proportional MPV case with BNT theory
5. Different block structures for $A$ and $B$
6. Unitary gauge upgrade

The formalization captures the **core algebraic heart** of the FT-MPS (the single-block Skolem–Noether argument + block separation via peeling + assembly) but not the full generality of the analytic/spectral-theoretic aspects (canonical form existence, periodic blocks, Perron–Frobenius theory for CP maps).
