3) Scientist-facing summaries (hypotheses / conclusion / proof idea)

3.1 MPSTensor.fundamentalTheorem_singleBlock  (TNLean/MPS/FundamentalTheorem.lean)
- Hypotheses:
  * A is injective: the matrices {A(i)} span the full matrix algebra.
  * SameMPV A B: for every length N and spin configuration σ, tr(evalWord A σ) = tr(evalWord B σ).
- Conclusion:
  * GaugeEquiv A B: there exists an invertible X such that B(i) = X A(i) X^{-1} for all i.
- Proof idea (high level):
  * Build the unique linear map T extending A(i) ↦ B(i) (linear extension step).
  * Use injectivity + SameMPV to show T is nonzero and multiplicative (an algebra endomorphism).
  * A nonzero multiplicative endomorphism of a full matrix algebra is bijective.
  * Apply Skolem–Noether: every matrix-algebra automorphism is inner, hence conjugation by some X.

3.2 MPSTensor.fundamentalTheorem_multiBlock_full  (TNLean/PiAlgebra/FundamentalTheoremComplete.lean)
- Hypotheses:
  * A and B are families of blocks A_k, B_k with (possibly different) block dimensions.
  * Each A_k is injective.
  * Per-block SameMPV: SameMPV (A_k) (B_k) for every block k (this is the “hSep” input).
- Conclusion:
  * Per-block gauge: for each k, there exists X_k with B_k(i) = X_k A_k(i) X_k^{-1}.
  * Global gauge for the block-diagonal tensors toTensorFromBlocks μ A and toTensorFromBlocks μ B.
- Proof idea:
  * Apply the single-block theorem independently to each block.
  * Assemble the X_k into one block-diagonal gauge matrix (a block-diagonal element of GL).
  * Reindex the Σ-type block matrix into a Fin(∑ dim_k)-indexed matrix; obtain global conjugation.

3.3 MPSTensor.fundamentalTheorem_multiBlock_fromSameMPV₂  (TNLean/PiAlgebra/FundamentalTheoremComplete.lean)
- Hypotheses:
  * SameMPV₂ holds globally on the two block-diagonal tensors toTensorFromBlocks μ A and toTensorFromBlocks μ B.
  * Each A_k is injective.
  * Separation hypothesis hSep is GIVEN explicitly: ∀k, SameMPV (A_k) (B_k).
- Conclusion:
  * Per-block gauge equivalence.
  * Global gauge equivalence of the block-diagonal tensors.
  * A structural “decomposition” statement for the induced Pi-algebra equivalence (permutation + inner conjugations).
- Proof idea:
  * Treat hSep as the only physics-heavy input.
  * Then the result is purely algebraic: single-block FT + block-diagonal assembly + Pi-algebra decomposition.

3.4 MPSTensor.sameMPV₂_single_block  (TNLean/PiAlgebra/FundamentalTheoremComplete.lean)
- Hypotheses:
  * One block (r = 1) with scaling μ₀ ≠ 0.
  * SameMPV₂ holds for the corresponding 1-block “block-diagonal” tensors.
- Conclusion:
  * SameMPV A₀ B₀ for the unique block.
- Proof idea:
  * The global summed identity collapses to μ₀^N · mpv(A₀,σ) = μ₀^N · mpv(B₀,σ).
  * Divide by μ₀^N (nonzero) to get equality of MPVs.

3.5 MPSTensor.sameMPV_of_sameMPV₂_injective / fundamentalTheorem_multiBlock_noSep  (TNLean/PiAlgebra/BlockSeparationProof.lean)
- Hypotheses (claimed sufficient in the file):
  * μ : Fin r → ℂ is injective and nonzero.
  * SameMPV₂ holds globally for the block-diagonal tensors.
- Claimed conclusion:
  * Per-block SameMPV (and hence per-block + global gauge) WITHOUT any explicit separation hypothesis.
- Status / correctness:
  * NOT TRUSTWORTHY. The key lemma per_block_trace_eq_of_summed_blocks is `sorry` and is known-false as stated.
  * The file contains an explicit counterexample (block-swap symmetry) showing failure for r ≥ 2.
- What seems needed to make a true theorem:
  * Additional canonical-form normalization (e.g. DS gauge / fixed spectral radius per block) to forbid rescaling-and-swap.
  * Or a weakened conclusion that allows identifying blocks only up to an allowed permutation/rescaling mechanism.

3.6 MPSTensor.block_separation_core / fundamentalTheorem_canonicalForm  (TNLean/PiAlgebra/CanonicalFormSep.lean)
- Hypotheses (canonical form):
  * Each block is injective.
  * DS gauge normalization: ∑_i A_{k,i}† A_{k,i} = I (and similarly for B).
  * Strict ordering of block weights: StrictAnti (k ↦ ‖μ_k‖), plus μ_k ≠ 0.
  * Global SameMPV₂ for the block-diagonal tensors.
- Intended conclusion:
  * Per-block SameMPV (hSep), hence per-block gauge and global gauge (fundamentalTheorem_canonicalForm).
- Proof idea (as documented in the file):
  * Use dominance/“peeling” via strict ‖μ_k‖ ordering to isolate the leading block asymptotically.
  * Use DS gauge + spectral gap of transfer operators to control bounds and convergence.
  * Induct on the number of blocks.
- Status:
  * OPEN GAP: the main lemma block_separation_core is `sorry` for r ≥ 2, so canonical-form end-to-end FT is not yet established.
