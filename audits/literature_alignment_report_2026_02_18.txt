Key literature alignment diagnosis.
  1) Lean’s single-block FT is mathematically correct but does *not* follow the standard “channel/
     mixed transfer” proof path used in e.g. the MPDO and review papers. This is acceptable for a
     formalization, but for an academic talk the narrative should explicitly relate the Lean route
     to the standard one (“same content, different proof skeleton”).

  2) Lean already *does* contain the main spectral-theory lemma that powers the literature’s
     “equal-or-orthogonal” / “off-diagonal damping” step:
       - mixedTransfer spectral radius ≤ 1 (normalized Kraus)
       - spectral radius = 1 forces gauge-phase equivalence
       - otherwise spectral radius < 1 and powers → 0
     (TNLean/Spectral/SpectralGap.lean).
     This is highly aligned with Wolf’s channel framing (Schwarz maps, peripheral eigenvectors).

  3) The open block-separation lemma should be closed using the spectral/mixed-transfer route
     (as in the literature), not by the current “peeling + boundedness” approach alone.
     Memo-level issue: peeling yields exponential *decay* but not *exact equality*.

Concrete recommendation (proof strategy).
  Replace the “Newton–Girard/Vandermonde separation on repeated words” as the primary strategy for
  block separation by a literature-aligned pipeline:

    (a) canonical form hypotheses (primitive/DS gauge) ⇒ equal-or-orthogonal for blocks
        via mixed transfer spectral radius

    (b) equal-or-orthogonal ⇒ eventual linear independence of block MPVs (Lemma “Lem1” in MPDO
        appendix / 1708 paper)

    (c) linear independence + global MPV equality ⇒ block matching (permutation fixed by strict
        |μ| ordering) ⇒ per-block SameMPV (and then per-block gauge by single-block FT).

Concrete recommendation (slides).
  Current slide deck (presentation20260218_tao_technical.tex) is Lean-architecture driven.
  For “literature alignment”, reorganize around the literature spine:

    Canonical form → equal-or-orthogonal (mixed transfer) → BNT / linear independence → FT.

  Move Skolem–Noether and Pi-algebra decomposition to backup slides.

1. Common literature proof spine (across the listed papers)
================================================================================

The following “spine” appears (with variants) in 1210.6613, 1606.00608, 1708.00029, and
2011.12127, and is also the conceptual background for Wielandt-type blocking bounds.

S1. Canonical / standard / irreducible form (after blocking)
  - Put tensor in block diagonal form, with blocks corresponding to irreducible/primitive transfer
    maps (normal tensors).
  - Normalize each block (trace-preserving / unital “DS gauge” in many expositions).

S2. Equal-or-orthogonal / off-diagonal damping lemma
  - For two normal blocks A,B, the mixed transfer F_{AB}(X)=∑_i A_i X B_i† satisfies:
      either ρ(F_{AB}) < 1  (cross terms decay)
      or     ρ(F_{AB}) = 1  (then A,B are gauge-phase equivalent).

S3. Basis of normal tensors (BNT) + eventual linear independence
  - Choose a subset of blocks which are pairwise “orthogonal” (cross correlations → 0).
  - Show that for N large enough, the corresponding MPVs are linearly independent.

S4. Fundamental Theorem (proportional and equal cases)
  - Proportional case: blocks match up to permutation + phase + gauge.
  - Equal case: stronger constraints; in the periodic/irreducible generalization, an extra root-of-
    unity diagonal degree of freedom appears.

S5. Optional quantitative ingredients
  - Quantum Wielandt inequalities: bounds on blocking length needed for injectivity/primitivity.
  - Correlation length estimates: spectral gap of transfer operator ⇒ exponential decay.

Lean coverage note.
  Lean currently fully covers S2 (mixed transfer spectral gap) and the algebraic consequences of S4
  *assuming per-block separation*; it does not yet cover S1 (canonical-form existence) or the S3→S4
  separation step in the form used in the literature.

3. Lean development vs. literature: where the proof route differs
================================================================================

3.1 Single-block FT: Skolem–Noether route vs. mixed-transfer route
  Literature (esp. 1606.00608 appendix; 2011 review) often proves the injective/normal FT via:
    - DS gauge (TP + fixed point),
    - Cauchy–Schwarz / Kadison–Schwarz,
    - peripheral eigenvector rigidity ⇒ gauge-phase.

  Lean proves the same mathematical statement via:
    - linear extension T : M_D→M_D extending A_i ↦ B_i,
    - show T multiplicative,
    - matrix algebra simplicity ⇒ bijective,
    - Skolem–Noether ⇒ inner automorphism.

  Assessment: acceptable as formal route. For a talk, explicitly say:
    “We formalize the same gauge-equivalence content, but use a ring-theoretic proof that is
     well-supported by Mathlib (Skolem–Noether), while we separately formalize the channel-based
     equal-or-orthogonal lemmas needed for the multi-block narrative.”

3.2 Multi-block FT: Pi-algebra decomposition vs. BNT narrative
  Literature expresses the multi-block uniqueness as “block permutation + gauge per block” inside
  the canonical-form/BNT framework.

  Lean expresses the same structure theorem as:
    - build an algebra automorphism of Π_k Mat_{D_k}(ℂ)
    - apply a general decomposition theorem: automorphism = block permutation + inner maps.

  Assessment: mathematically identical; just a more abstract packaging. In slides, present the
  physics statement (permutation + per-block conjugation) first, and then note that Lean uses a
  general algebra theorem to obtain it.

3.3 Canonical form assumptions in Lean are currently stronger than in the literature
  Lean predicate IsCanonicalForm μ A includes StrictAnti (k ↦ ‖μ_k‖) and a single-sided TP gauge
  ∑ A_i†A_i = I. The literature canonical form allows:
    - repeated blocks up to phases (handled by BNT),
    - periodic/irreducible blocks (handled by irreducible form),
    - equal moduli among μs (phases on the unit circle).

  This is fine for a “first formal target” but should be clearly stated as a restriction.

4. Literature-aligned strategy to close the main gap: canonical-form block separation
================================================================================

Target gap.
  Close MPSTensor.block_separation_core (PiAlgebra/CanonicalFormSep.lean) for r ≥ 2.

Why the current local tools are insufficient.
  CanonicalFormSep already contains:
    - DS gauge ⇒ uniform trace bounds
    - a “peeling” lemma ⇒ exponential decay of the leading-block repeated-word trace difference

  But decay does not imply equality: |δ(L)| ≤ C ρ^L allows δ(L) ≠ 0.
  The literature resolves this by *orthogonality/linear-independence* arguments driven by the
  mixed transfer operator spectral theory.

Recommended (literature-aligned) closure plan.

(4.1) Strengthen or derive hypotheses so spectral lemmas apply.
  • Many strict spectral-gap results for mixedTransferMap require injectivity on both tensors.
  • CanonicalFormSep.block_separation_core currently assumes injectivity only for A blocks.

  Recommendation:
    - Either (preferred) strengthen IsCanonicalForm to include block_injective for B as well, or
    - Prove “B_k injective” as a lemma from the canonical-form hypotheses + global SameMPV₂
      (this is plausible but nontrivial).

(4.2) Use the already formalized equal-or-orthogonal lemma.
  Lean already proves the core statement:

    spectralRadius_mixedTransfer_lt_one:
      IsInjective A ∧ IsInjective B ∧ TP(A) ∧ TP(B) ∧ ¬GaugePhaseEquiv(A,B)
        ⟹  ρ(F_AB) < 1

    mixedTransfer_pow_tendsto_zero:
      ρ(F_AB) < 1 ⟹ F_AB^n(X) → 0

  This matches:
    - UncleHMPS lemma:offdiagonal-damping
    - MPDO appendix Lemma equalMPS (channel-based part)
    - Wolf ch06: peripheral spectrum of irreducible Schwarz maps.

(4.3) Prove eventual linear independence (Lemma “Lem1” / “Lem1t”).
  Literature uses: if pairwise overlaps → 0 and norms → 1, then vectors are independent for N
  sufficiently large.

  Implementation idea in Lean:
    - Define MPV vectors explicitly as functions mpvFun A N : (Fin N → Fin d) → ℂ (already exists)
      and equip that finite function space with the standard ℓ² inner product.
    - Express overlaps via mixed transfer traces (CrossCorrelation already gives “trace → 0”).
    - Show Gram matrix → diagonal PD, hence det ≠ 0 for N large, hence linear independence.

  This step is the clean, literature-aligned replacement for the “peeling-only” approach.

(4.4) Use uniqueness of decomposition for large N to separate blocks.
  Once {V_N(A_k)} are linearly independent and likewise for B, global equality of the canonical-
  form states forces a blockwise identification. With strict ordering of ‖μ_k‖, the block permutation
  is fixed (sorted by magnitude).

  Then apply the already proved single-block FT to convert SameMPV into per-block GaugeEquiv.

(4.5) De-emphasize Newton–Girard as a primary route.
  Lean’s Newton–Girard development is correct and useful, but the BlockSeparationProof.lean plan
  demonstrates that a purely “trace powers ⇒ charpoly ⇒ Vandermonde” strategy is fragile and can
  state false lemmas unless canonical-form normalization is carefully built in.
  The spectral/mixed-transfer route is closer to the literature and already matches the existing
  Lean Spectral/ modules.
