18 Concrete Examples

This chapter collects concrete MPS tensors from [ CPGSV21 , Section III ] . Each example is specified by its physical dimension \(d\), bond dimension \(D\), and explicit matrix family \(\{ A^i\} \). Key structural properties (injectivity, RFP status, symmetry) are stated for each tensor.

18.1 The GHZ state

The Greenberger–Horne–Zeilinger (GHZ) state is the prototypical non-injective, renormalization-fixed-point MPS.

Definition 18.1 GHZ tensor
✓
#

Uses

  • Definition 2.1

Lean declarations

  • MPSTensor.ghzTensor

Set \(d = D = 2\). The GHZ tensor is the family \(\{ A^0, A^1\} \subset M_{2}(\mathbb {C})\) defined by

\[ A^0 = |0\rangle \! \langle 0| = \operatorname{diag}(1,0), \qquad A^1 = |1\rangle \! \langle 1| = \operatorname{diag}(0,1). \]

For a system of \(N\) sites the resulting MPV is the GHZ state

\[ |\mathrm{GHZ}_N\rangle = |0\rangle ^{\otimes N} + |1\rangle ^{\otimes N}. \]
Definition 18.2 Two-site computational vector
✓
#

Lean declarations

  • MPSTensor.twoSiteKet

For \(a,b\in \{ 0,1\} \), the vector

\[ |ab\rangle \in (\{ 0,1\} ^{2}\to \mathbb {C}) \]

is the indicator of the configuration

\[ (\sigma _1,\sigma _2)=(a,b). \]
Definition 18.3 Constant product vector
✓
#

Lean declarations

  • MPSTensor.constantKet

For \(a\in \{ 0,1\} \) and \(N\ge 0\),

\[ |a\rangle ^{\otimes N}(\sigma _1,\ldots ,\sigma _N) = \begin{cases} 1, & (\sigma _1,\ldots ,\sigma _N)=(a,\ldots ,a),\\ 0, & \text{otherwise}. \end{cases} \]
Theorem 18.4 GHZ transfer map is idempotent
✓
#

Uses

  • Definition 18.1
  • Definition 2.5

Lean declarations

  • MPSTensor.ghz_transferMap_idempotent

The transfer map of the GHZ tensor satisfies \(\mathcal{E}_A^2 = \mathcal{E}_A\).

Proof ▶

The transfer map acts by \(\mathcal{E}_A(X)_{ij} = \delta _{ij} X_{ij}\) (extraction of the diagonal). This operation is idempotent.

Theorem 18.6 GHZ tensor is not injective
✓
#

Uses

  • Definition 18.1
  • Definition 2.28

Lean declarations

  • MPSTensor.ghz_not_isInjective

The GHZ tensor is not injective: \(\operatorname{span}_{\mathbb {C}}\{ A^0, A^1\} \) is the two-dimensional space of diagonal matrices in \(M_{2}(\mathbb {C})\), which does not equal \(M_{2}(\mathbb {C})\).

Proof ▶

Every element of \(\operatorname{span}_{\mathbb {C}}\{ A^0, A^1\} \) has zero off-diagonal entries, so the matrix with all entries equal to \(1\) is not in the span.

Theorem 18.7 \(\mathbb {Z}_2\) on-site symmetry of the GHZ tensor
✓
#

Uses

  • Definition 18.1
  • Definition 12.4

Lean declarations

  • MPSTensor.ghz_isOnSiteSymmetric_Z2

Let \(\mathbb {Z}_2 = \{ 1, \sigma _x\} \) act on \(\mathbb {C}^2\) by the Pauli-\(X\) matrix \(\sigma _x\). The GHZ tensor is on-site symmetric under this action: for each \(g \in \mathbb {Z}_2\), the twisted tensor \(\widetilde A_g\) defines the same MPV family as \(A\), i.e. \(\mathcal{V}(A)=\mathcal{V}(\widetilde A_g)\).

Proof ▶

The identity element is trivial. For the generator \(g = \sigma _x\), the twisted tensor satisfies \(\widetilde A_g^i = A^{1-i}\), i.e. the two matrices are swapped. Conjugation by \(\sigma _x\) implements the same swap, giving gauge equivalence and hence the same MPV family.

Definition 18.8 Virtual \(\mathbb {Z}_2\) representation of the GHZ tensor
✓
#

Uses

  • Definition 18.1

Lean declarations

  • MPSTensor.ghzVirtualZ2Action

The virtual \(\mathbb {Z}_2\) representation on the bond space is the homomorphism \(\mathbb {Z}_2 \to M_{2}(\mathbb {C})\) sending the identity to \(I\) and the generator to \(\sigma _z = \operatorname{diag}(1, -1)\).

Theorem 18.9 Virtual \(\mathbb {Z}_2\) symmetry of the GHZ tensor
✓
#

Uses

  • Definition 18.1
  • Definition 18.8

Lean declarations

  • MPSTensor.ghz_virtual_Z2_symmetric

The matrix \(\sigma _z\) commutes with every GHZ matrix \(A^i\). A non-scalar matrix commuting with all \(A^i\) is the hallmark of a non-injective (\(\mathbb {Z}_2\)-injective) tensor: the on-site symmetry of Theorem 18.7 swaps the two one-dimensional injective blocks \(A^0 \leftrightarrow A^1\), while \(\sigma _z\) acts on each block by a phase.

Proof ▶

Both \(A^0\) and \(A^1\) are diagonal, so each commutes with the diagonal matrix \(\sigma _z\).

Theorem 18.10 GHZ order parameter is a transfer-map fixed point
✓
#

Uses

  • Definition 18.1
  • Definition 2.5

Lean declarations

  • MPSTensor.ghz_orderParam_fixed

The order-parameter operator \(\sigma _z\) is a fixed point of the GHZ transfer map, \(\mathcal{E}_A(\sigma _z) = \sigma _z\).

Proof ▶

Since \(\mathcal{E}_A(X) = \sum _i A^i X (A^i)^{\dagger }\) and both \(A^i\) are diagonal, the claim is the entrywise computation \(\mathcal{E}_A(\sigma _z) = \sigma _z\).

Remark 18.11
#

A primitive (injective) tensor has a one-dimensional transfer-map fixed-point space, spanned by a positive matrix that equals the identity only in a normalized gauge. That the GHZ transfer map fixes the additional non-scalar operator \(\sigma _z\) is the transfer-matrix signature of the extra non-decaying mode behind its long-range ferromagnetic correlations.

18.2 The AKLT state

The Affleck–Kennedy–Lieb–Tasaki (AKLT) state is the canonical example of a normal (block-injective) MPS with non-trivial symmetry-protected topological (SPT) order.

Definition 18.12 AKLT tensor
✓
#

Uses

  • Definition 2.1

Lean declarations

  • MPSTensor.akltTensor

Set \(d = 3\) (spin-1) and \(D = 2\). The AKLT tensor is the family \(\{ A^0, A^1, A^2\} \subset M_{2}(\mathbb {C})\) defined via the Pauli matrices by

\[ A^0 = \frac{1}{\sqrt{3}} \sigma _z, \qquad A^1 = \frac{\sqrt{2}}{\sqrt{3}} |0\rangle \! \langle 1|, \qquad A^2 = -\frac{\sqrt{2}}{\sqrt{3}} |1\rangle \! \langle 0|, \]

where \(\sigma _z = |0\rangle \! \langle 0| - |1\rangle \! \langle 1|\), so equivalently \(A^1 = \frac{1}{\sqrt{3}} \sigma _+\) and \(A^2 = -\frac{1}{\sqrt{3}} \sigma _-\) for \(\sigma _+ = \sqrt{2} |0\rangle \! \langle 1|\) and \(\sigma _- = \sqrt{2} |1\rangle \! \langle 0|\). Equivalently, the matrices are the Clebsch–Gordan coefficients coupling spin-\(1\) and spin-\(\tfrac 12\) to spin-\(\tfrac 12\).

Theorem 18.13 AKLT tensor is not injective
✓
#

Uses

  • Definition 18.12
  • Definition 2.28

Lean declarations

  • MPSTensor.aklt_not_isInjective

The AKLT tensor is not injective at a single site: \(\operatorname{span}_{\mathbb {C}}\{ A^0, A^1, A^2\} \) is the three-dimensional space of traceless matrices \(\mathfrak {sl}(2,\mathbb {C})\), which is a proper subspace of \(M_{2}(\mathbb {C})\).

Theorem 18.14 AKLT tensor is normal
✓
#

Uses

  • Definition 18.12

The transfer map of the AKLT tensor has a unique eigenvalue of maximal modulus (a simple spectral-radius eigenvalue); in particular the AKLT tensor is normal. Concretely, the length-\(2\) blocked tensor is injective.

Theorem 18.20 The AKLT \(\mathbb {Z}_2 \times \mathbb {Z}_2\) phase is non-trivial
✓
#

Uses

The AKLT factor system (Theorem 18.19), built from the anticommuting gauges \(i\sigma _y\) and \(\sigma _z\) (Theorem 18.16), is not a coboundary: its commutator phase on the two generators is \(-1\). Its class in \(H^2(\mathbb {Z}_2 \times \mathbb {Z}_2, \mathrm{U}(1)) \cong \mathbb {Z}_2\) is therefore non-trivial, so the AKLT chain realizes a non-trivial SPT phase rather than a trivial product state.

Remark 18.21
#

This \(\mathbb {Z}_2 \times \mathbb {Z}_2\) is the smallest subgroup of \(\mathrm{SO}(3)\) that still carries a non-trivial \(2\)-cocycle. That minimality, and the full continuous classification over \(\mathrm{SO}(3)\), are not formalized (Remark 18.25).

Definition 18.22 Cartesian AKLT tensor
✓
#

Uses

Lean declarations

The Cartesian presentation of the AKLT tensor has \(d = 3\), \(D = 2\), and \(A^x = \sigma _x\), \(A^y = \sigma _y\), \(A^z = \sigma _z\), the three Pauli matrices. This is the rotationally natural form used in [ CPGSV21 ] (around line 1159); it is the AKLT state in the Cartesian basis of the spin-\(1\) site, the counterpart of the \(|m\rangle \)-basis form of Definition 18.12.

Theorem 18.23 The spin-\(\tfrac 12\) cover is surjective onto \(\mathrm{SO}(3)\)
✓
#

Lean declarations

  • MPSTensor.spinHalfCover_surjective_onto_SO3

Every rotation \(R \in \mathrm{SO}(3)\) is realized by conjugating the Pauli vector by some \(U \in \mathrm{SU}(2)\): there is \(U \in \mathrm{SU}(2)\) with \(\bigl(\tfrac 12 \operatorname{tr}(\sigma _i\, U \sigma _j U^{-1})\bigr)_{ij} = R\). Equivalently, the adjoint double cover \(\mathrm{SU}(2)\to \mathrm{SO}(3)\) is surjective. The proof factors \(R = R_z(\alpha )\, R_x(\beta )\, R_z(\gamma )\) by its \(ZXZ\) Euler decomposition and maps each coordinate rotation to an explicit \(\mathrm{SU}(2)\) generator: \(\operatorname{diag}(e^{-i\theta /2}, e^{i\theta /2}) \mapsto R_z(\theta )\) and \(\bigl(\begin{smallmatrix} \cos \frac\beta 2 & -i\sin \frac\beta 2 \\ -i\sin \frac\beta 2 & \cos \frac\beta 2 \end{smallmatrix}\bigr) \mapsto R_x(\beta )\).

Theorem 18.24 \(\mathrm{SO}(3)\) on-site symmetry of the AKLT tensor
✓
#

Uses

  • Definition 18.22
  • Definition 12.4
  • Theorem 18.23

Lean declarations

  • MPSTensor.aklt_isOnSiteSymmetric_SO3

The Cartesian AKLT tensor is on-site symmetric under the spin-\(1\) (defining) representation of \(\mathrm{SO}(3)\) on the physical index.

Remark 18.25
#

The bond index carries the projective spin-\(\tfrac 12\) representation: the gauge for a rotation is either of its two \(\mathrm{SU}(2)\) preimages, so the assignment is a representation only up to sign and factors through the double cover \(\mathrm{SU}(2)\to \mathrm{SO}(3)\). Its class is the non-trivial element of \(H^2(\mathrm{SO}(3),\mathrm{U}(1))\cong \mathbb {Z}_2\), witnessing the SPT phase.

18.3 The cluster state

The cluster state is a universal resource for measurement-based quantum computation and provides another example of a non-trivial SPT phase.

Definition 18.26 Cluster-state tensor
✓
#

Uses

  • Definition 2.1

Lean declarations

  • MPSTensor.clusterTensor

Set \(d = D = 2\). The cluster-state tensor is the family \(\{ A^0, A^1\} \subset M_{2}(\mathbb {C})\) defined by

\[ A^0 = |+\rangle \! \langle 0| = \frac{1}{\sqrt{2}} \begin{pmatrix} 1 & 0 \\ 1 & 0 \end{pmatrix}, \qquad A^1 = |-\rangle \! \langle 1| = \frac{1}{\sqrt{2}} \begin{pmatrix} 0 & 1 \\ 0 & -1 \end{pmatrix}, \]

where \(|\pm \rangle = \tfrac {1}{\sqrt{2}}(|0\rangle \pm |1\rangle )\). The review  [ CPGSV21 ] writes the reflected convention \(A^0 = |0\rangle \! \langle +|\), \(A^1 = |1\rangle \! \langle -|\); the transpose taken here is the same state read in the opposite direction and carries the same SPT order.

Theorem 18.27 Cluster-state tensor is not injective
✓
#

Uses

  • Definition 18.26
  • Definition 2.28
  • Definition 18.26

Lean declarations

  • MPSTensor.cluster_not_isInjective

The cluster-state tensor is not injective: \(\operatorname{span}_{\mathbb {C}}\{ A^0, A^1\} = \operatorname{span}_{\mathbb {C}}\{ |+\rangle \! \langle 0|, |-\rangle \! \langle 1|\} \) has dimension \(2\), not \(\dim M_{2}(\mathbb {C}) = 4\).

Proof ▶

Both \(A^0\) and \(A^1\) are rank-one matrices with linearly independent column spaces, so they are linearly independent. Their span is two-dimensional, which is strictly less than \(\dim M_{2}(\mathbb {C}) = 4\).

Theorem 18.28 Cluster-state tensor is normal
✓
#

Uses

  • Definition 18.26
  • Definition 2.28

For \(N\ge 2\), the common nearest-neighbour cyclic ground space of the GHZ tensor satisfies

\[ \mathcal G_{N,2}(A_{\mathrm{GHZ}}) = \operatorname{span}_{\mathbb {C}}\{ |0\rangle ^{\otimes N}, |1\rangle ^{\otimes N} \} . \]

Proof ▶

Let \(\psi \in \mathcal G_{N,2}(A_{\mathrm{GHZ}})\). For every cyclic nearest-neighbour window and outside assignment \(\tau \),

\[ \left.\psi \right|_{[i,i+1],\tau } \in \mathcal G_2(A_{\mathrm{GHZ}}) = \operatorname{span}_{\mathbb {C}}\{ |00\rangle ,|11\rangle \} . \]

Taking the outside assignment from the word \(\sigma \) gives

\[ \left.\psi \right|_{[i,i+1],\sigma }(\sigma _i,\sigma _{i+1}) = \psi (\sigma ). \]

Hence a mixed nearest-neighbour word has zero coefficient:

\[ \sigma _i\ne \sigma _{i+1} \quad \Longrightarrow \quad \psi (\sigma )=0. \]

A nonconstant binary cyclic word has a mixed edge,

\[ \sigma \notin \{ (0,\ldots ,0),(1,\ldots ,1)\} \quad \Longrightarrow \quad \exists i,\ \sigma _i\ne \sigma _{i+1}. \]

Therefore

\[ \psi = \psi (0,\ldots ,0)|0\rangle ^{\otimes N} + \psi (1,\ldots ,1)|1\rangle ^{\otimes N}. \]

\[ |0\rangle ^{\otimes N},\ |1\rangle ^{\otimes N} \in \mathcal G_{N,2}(A_{\mathrm{GHZ}}), \]
