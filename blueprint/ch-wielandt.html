8 Wielandt Bound

This chapter proves a quantum analog of Wielandt’s inequality: for a normal MPS tensor with bond dimension \(D\), the cumulative span of matrix products stabilises after at most \(D^2\) steps. The results follow  [ SPGWC10 ] .

\[ S_n(A) = \operatorname{span}_{\mathbb {C}}\{ A^w : |w| = n\} \subseteq M_{D}(\mathbb {C}). \]

Theorem 8.5 Cumulative span stabilisation
✓
#

Uses

  • Definition 8.2

Lean declarations

  • MPSTensor.cumulativeSpan_stable

If \(T_n(A) = T_{n+1}(A)\), then \(T_m(A) = T_n(A)\) for all \(m \ge n\).

Proof ▶

If \(T_n = T_{n+1}\), then \(S_{n+1} \subseteq T_n\). Left-multiplying any element of \(S_{n+1}\) by \(A^i\) gives an element of \(S_{n+2}\), so \(S_{n+2} \subseteq T_{n+1} = T_n\). By induction, \(S_m \subseteq T_n\) for all \(m {\gt} n\), hence \(T_m = T_n\).

8.2 Fitting decomposition

Definition 8.10 Fitting decomposition
✓
#

Lean declarations

  • Wielandt.FittingDecomposition

A Fitting decomposition of a linear endomorphism \(f : V \to V\) on a finite-dimensional vector space over an algebraically closed field consists of:

  1. \(f\) is nilpotent on the generalized \(0\)-eigenspace,

  2. \(f\) is invertible on each generalized \(\mu \)-eigenspace for \(\mu \neq 0\),

  3. the generalized eigenspaces span \(V\),

  4. the generalized eigenspaces are linearly independent.

Theorem 8.11 Fitting decomposition exists
✓
#

Uses

  • Definition 8.10

Lean declarations

  • Wielandt.fittingDecomposition

Every linear endomorphism on a finite-dimensional vector space over an algebraically closed field admits a Fitting decomposition.

Proof ▶

Nilpotency on the zero generalized eigenspace follows from the definition of generalized eigenspaces. Invertibility on nonzero generalized eigenspaces follows because \(f - \mu \) is nilpotent there, so \(f = \mu (1 - (1-f/\mu ))\) is invertible. Spanning and independence are standard results for generalized eigenspaces over algebraically closed fields. In  [ SPGWC10 ] and [ Wol12 , Lemma 6.3(b) ] , the Jordan normal form is used directly. The argument is phrased in terms of generalized eigenspaces instead.

8.3 Nonzero trace product

Theorem 8.13 Cumulative span reaches \(M_{D}(\mathbb {C})\) — explicit bound
✓
#

Uses

Lean declarations

  • MPSTensor.cumulativeSpan_eq_top_of_isNormal_bound

If \(A\) is normal, then there exists \(n \le D^2\) with \(T_n(A) = M_{D}(\mathbb {C})\).

Proof ▶

By Theorem 8.6, if \(T_n \neq T_{n+1}\) then \(\dim T_{n+1} {\gt} \dim T_n\). Since \(\dim T_n \le D^2\) (Lemma 8.4), after at most \(D^2\) steps either \(T_n = M_{D}(\mathbb {C})\) or \(T_n\) has stabilised. Stabilisation before reaching \(M_{D}(\mathbb {C})\) contradicts normality (which requires \(S_{L_0} = M_{D}(\mathbb {C})\) for some \(L_0\), hence \(T_{L_0} = M_{D}(\mathbb {C})\)).

Theorem 8.15 Nonzero trace product
✓
#

Uses

  • Definition 2.29
  • Theorem 8.14

Lean declarations

  • MPSTensor.exists_nonzero_trace_word

If \(A\) is normal, then there exists a word \(w\) of length \(\le D^2\) such that \(\operatorname{tr}(A^w) \neq 0\). This is [ SPGWC10 , Lemma 1 ] . For the trace argument below, the nontrivial case is \(D \ge 1\); when \(D = 0\) the normality hypothesis is vacuous.

Proof ▶

By Theorem 8.14, \(\mathbb {1}\in T_{D^2}(A)\), so \(\mathbb {1}\) is a linear combination of word products of length \(\le D^2\). In the nontrivial case \(D \ge 1\), at least one has nonzero trace (since \(\operatorname{tr}(\mathbb {1}) = D \neq 0\)).

Theorem 8.16 Cumulative span reaches \(M_{D}(\mathbb {C})\) — sharp bound
✓
#

Uses

  • Definition 8.2
  • Definition 2.29
  • Theorem 8.6
  • Lemma 8.4

Lean declarations

  • MPSTensor.cumulativeSpan_eq_top_of_isNormal_sharp

If \(A\) is normal, then \(T_{D^2 - d' + 1}(A) = M_{D}(\mathbb {C})\), where \(d' = \dim S_1(A) = \operatorname{kr}(A)\). This is the sharp form of [ SPGWC10 , Lemma 1 ] : the cumulative span already reaches \(M_{D}(\mathbb {C})\) by index \(D^2 - d' + 1 \le D^2\).

Proof ▶

The key observation is that if \(T_n(A) \neq T_{n+1}(A)\), the dimension strictly increases. Since \(S_1(A) \subseteq T_1(A)\) and \(\dim S_1(A) = d'\), we have \(\dim T_1(A) \ge d'\). After at most \(D^2 - d'\) additional strict-growth steps the dimension reaches \(D^2 = \dim M_{D}(\mathbb {C})\). Stabilisation before reaching \(M_{D}(\mathbb {C})\) contradicts normality. Hence \(T_{D^2 - d' + 1}(A) = M_{D}(\mathbb {C})\).

Theorem 8.17 Sharp nonzero trace product
✓
#

Uses

  • Definition 2.29
  • Theorem 8.16

Lean declarations

  • MPSTensor.exists_nonzero_trace_word_sharp

If \(A\) is normal, then there exists a word \(w\) of length \(\le D^2 - \operatorname{kr}(A) + 1\) such that \(\operatorname{tr}(A^w) \neq 0\). This is [ SPGWC10 , Lemma 1 ] with the sharp length bound.

Proof ▶

By Theorem 8.16, the identity \(\mathbb {1}\) lies in \(T_{D^2 - d' + 1}(A)\), hence is a linear combination of word products of length \(\le D^2 - d' + 1\). At least one such product has nonzero trace.

Theorem 8.18 Sharp nonzero trace product — positive length
✓
#

Uses

  • Definition 2.29
  • Theorem 8.17
  • Theorem 8.16

Lean declarations

  • MPSTensor.exists_nonzero_trace_word_sharp_pos

If \(A\) is normal and \(D \ge 2\), then there exists a word \(w\) of positive length \(1 \le |w| \le D^2 - \operatorname{kr}(A) + 1\) such that \(\operatorname{tr}(A^w) \neq 0\). This strengthens Theorem 8.17 by requiring \(|w| \ge 1\), a feature needed for the blocking argument in the general Wielandt bound below.

Proof ▶

Define the positive-level cumulative span \(V_n = \operatorname{span}\{ A^w : 1 \le |w| \le n\} \). For \(D \ge 2\), the span \(S_0(A) = \operatorname{span}\{ \mathbb {1}\} \) has dimension \(1\), so \(V_1 \supseteq S_1(A)\) has dimension \(\ge d' \ge 1\). The same stabilization-or-strict-growth dichotomy shows \(V_{D^2 - d' + 1} = M_{D}(\mathbb {C})\): if \(V_n\) stabilises and is not \(M_{D}(\mathbb {C})\), then \(V_n\) is closed under left-multiplication by each \(A^i\), so \(S_{N}(A) \subseteq V_n \neq M_{D}(\mathbb {C})\) for all \(N \ge 1\), contradicting normality. Since \(V_{D^2-d'+1} = M_{D}(\mathbb {C})\), the identity \(\mathbf{1}\) lies in \(V_{D^2-d'+1}\) and is therefore a linear combination of word products of positive length. At least one such product has nonzero trace (since \(\operatorname{tr}(\mathbf{1}) = D \geq 2 \neq 0\)).

8.4 Eigenvector extraction

Theorem 8.19 Nonzero trace implies eigenvector
✓
#

Lean declarations

  • exists_eigenvector_of_trace_ne_zero

If \(M \in M_{D}(\mathbb {C})\) has \(\operatorname{tr}(M) \neq 0\), then \(M\) has a nonzero eigenvalue \(\mu \neq 0\) and a corresponding eigenvector \(\varphi \neq 0\) with \(M \varphi = \mu \varphi \).

Proof ▶

A matrix with nonzero trace has a nonzero eigenvalue (since trace = sum of eigenvalues counted with multiplicity). The Fitting decomposition (Theorem 8.11) ensures that the corresponding generalized eigenspace is nontrivial. On that generalized eigenspace, \(M - \mu \mathbb {1}\) is nilpotent, so its kernel is nontrivial. Any nonzero vector in \(\ker (M - \mu \mathbb {1})\) is therefore a genuine eigenvector with eigenvalue \(\mu \).

Theorem 8.20 Eigenvalue extraction
✓
#

Uses

  • Definition 2.29

Proof ▶

By Theorem 8.15, there is a word \(w_0\) with \(\operatorname{tr}(A^{w_0}) \neq 0\) and \(|w_0| \le D^2\). By Theorem 8.19, \(A^{w_0}\) has a nonzero eigenvector.

Remark 8.65 Primitivity: spectral gap vs. positive definiteness
✓
#

The primitivity condition used in §8.10.1 has three parts: TP normalization \(\sum _i (A^i)^\dagger A^i = \mathbb {1}\), a nonzero PSD fixed point \(\rho \), and the spectral gap \(\rho _{\mathrm{spec}}(\mathcal{E}_A - P_\rho ) {\lt} 1\). The notion of “primitive” in [ SPGWC10 , Proposition 3 ] additionally requires \(\rho \) to be positive definite (full rank). Without positive definiteness, the spectral-gap condition alone does not imply irreducibility or normality. A counterexample is \(D = 2\), \(\rho = \operatorname{diag}(1,0)\).

With the additional hypothesis \(\rho {\gt} 0\), Theorem 8.66 gives irreducibility. Together with Burnside’s theorem (Chapter 9, §9.4.1) and Theorem 8.75, one obtains

\[ \text{primitive} + \rho \text{ positive definite} \; \Rightarrow \; \text{irreducible} \; \Rightarrow \; \operatorname{alg}(A) = M_{D}(\mathbb {C}) \; \Rightarrow \; \text{normal}. \]

The last step uses the aperiodicity condition \(\mathbb {1}\in S_1(A)\) from Remark 8.70. After restoring the positive-definite hypothesis, [ SPGWC10 , Proposition 3 ] identifies that primitive/strongly irreducible condition with eventual full Kraus rank, i.e. with our notion of normality.

Proof ▶

By contrapositive. Assume \(A\) has an invariant projection \(P\) (\(P \neq 0\), \(P \neq \mathbb {1}\), \((\mathbb {1}- P) A^i P = 0\) for all \(i\)). Set \(\sigma = P \rho P\).

  2. Convergence: By Theorem 8.64, \(\mathcal{E}_A^n(\sigma ) \to \frac{\operatorname{tr}(\sigma )}{\operatorname{tr}(\rho )} \rho \). Hence \((\mathbb {1}- P) \cdot \frac{\operatorname{tr}(\sigma )}{\operatorname{tr}(\rho )} \rho = 0\).

  3. Nonzero scalar: Since \(P \neq 0\) and \(\rho \) is positive definite, \(\operatorname{tr}(P \rho P) {\gt} 0\), so \(\frac{\operatorname{tr}(\sigma )}{\operatorname{tr}(\rho )} \neq 0\). Therefore \((\mathbb {1}- P) \rho = 0\).

  4. Contradiction: Since \(\rho \) is positive definite, it is a unit in \(M_{D}(\mathbb {C})\). From \((\mathbb {1}- P) \rho = 0\) we get \(\mathbb {1}- P = 0\), i.e. \(P = \mathbb {1}\), contradicting \(P \neq \mathbb {1}\).

Remark 8.70 Aperiodicity condition
✓
#

We say \(A\) satisfies the aperiodicity condition if \(\mathbb {1}\in S_1(A)\), i.e. the identity matrix lies in the span of the Kraus operators \(\{ A^i\} \). This is an additional hypothesis. The one-sided TP normalization \(\sum _i (A^i)^\dagger A^i = \mathbb {1}\) does not imply \(\mathbb {1}\in S_1(A)\); it only gives a quadratic identity among the Kraus operators. The aperiodicity condition is kept explicit in this section for generality. Remark 8.78 Normal canonical form avoids the aperiodicity step

The aperiodicity results in §8.11 are needed only for the block-injective argument through Definition 2.29, where one upgrades cumulative spanning to eventual block injectivity. The normal-canonical theory used later in Chapters 9–11 does not pass through this step: it works directly with irreducible TP-normalized blocks whose blocked transfer maps are primitive and, once the corresponding weighted block data are available, places them in normal canonical form in the sense of [ CPGSV17a ] . There are analogous cumulative-span lemmas for blocked irreducible tensors and eigenvector-spreading arguments, but they are auxiliary here and we do not list them separately.
