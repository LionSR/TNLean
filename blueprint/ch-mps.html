2 Matrix Product Vectors

Fix a physical dimension \(d\) and a bond dimension \(D\). This chapter studies the tensors that generate matrix product vector (MPV) families. All tensors here are translation-invariant with periodic boundary conditions (PBC).

2.1 Basic definitions

Definition 2.1 MPS tensor
✓
#

Lean declarations

  • MPSTensor

A (translation-invariant, PBC) MPS tensor with physical dimension \(d\) and bond dimension \(D\) is a collection of matrices

\[ \{ A^i\} _{i=0}^{d-1}, \qquad A^i \in M_{D}(\mathbb {C}), \]

indexed by a physical index \(i \in \{ 0, \ldots , d{-}1\} \). Such a tensor defines an MPV family. Diagrammatically,

The black node denotes the tensor \(A\), the horizontal legs are virtual, and the upper leg is the physical index \(i\).

Definition 2.2 Word evaluation
✓
#

Uses

  • Definition 2.1

Lean declarations

  • MPSTensor.evalWord

Given a word \(w = (i_1, \ldots , i_L) \in \{ 0,\ldots ,d{-}1\} ^L\), the word evaluation is the matrix product

\[ A^w \; :=\; A^{i_1} A^{i_2} \cdots A^{i_L} \; \in \; M_{D}(\mathbb {C}), \]

with the convention that the empty word evaluates to the identity: \(A^\varnothing = \mathbb {1}_D\). In tensor-network notation,

in which each black node denotes the same local tensor \(A\), the virtual legs remain open, and the physical legs are labelled by the word \((i_1,\ldots ,i_L)\).

Lemma 2.3 Multiplicativity of word evaluation
✓
#

Uses

  • Definition 2.2
  • Definition 2.2

Lean declarations

  • MPSTensor.evalWord_append

For any words \(w_1, w_2\), \(A^{w_1 \cdot w_2} = A^{w_1} \, A^{w_2}\).

Proof ▶

Immediate from the definition of \(A^w\).

Definition 2.4 Matrix product vector
✓
#

Uses

  • Definition 2.2

Lean declarations

  • MPSTensor.mpv

The matrix product vector (MPV) of a tensor \(A\) at system size \(N\) is the vector

\[ |V^{(N)}(A)\rangle = \sum _{i_1, \ldots , i_N = 0}^{d-1} \operatorname{tr}\! (A^{i_1} A^{i_2} \cdots A^{i_N}) |i_1, \ldots , i_N\rangle \in (\mathbb {C}^d)^{\otimes N}. \]

Equivalently, for a configuration \(\sigma = (i_1, \ldots , i_N) \in \{ 0,\ldots ,d{-}1\} ^N\), we write

\[ V^{(N)}(A)_\sigma := \operatorname{tr}\! (A^{i_1} A^{i_2} \cdots A^{i_N}). \]

The coefficient function \(\sigma \mapsto V^{(N)}(A)_\sigma \) gives the components; the displayed ket is the corresponding vector in \((\mathbb {C}^d)^{\otimes N}\). For a general word \(w\), we also write

\[ c_w(A) := \operatorname{tr}(A^w), \]

The MPV family generated by \(A\) is the collection \(\mathcal{V}(A) = \bigl\{ |V^{(N)}(A)\rangle \bigr\} _{N \ge 1}\). The coefficient \(V^{(N)}(A)_\sigma \) is the periodic contraction

of \(N\) copies of the local tensor \(A\), with the outer virtual legs closed by the trace.

Definition 2.5 Transfer map
✓
#

Uses

  • Definition 2.1

Lean declarations

  • MPSTensor.transferMap

The transfer map associated to a tensor \(A\) is the linear map \(\mathcal{E}_A : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) defined by

\[ \mathcal{E}_A(X) \; =\; \sum _{i=0}^{d-1} A^i \, X \, (A^i)^\dagger . \]

Diagrammatically, the transfer map is the double-layer contraction

in which the upper node denotes \(A\), the lower node denotes \(A^\dagger \), and the physical index is summed over between them.

Remark 2.6
✓
#

The transfer map is automatically positive: if \(X \ge 0\) then \(\mathcal{E}_A(X) \ge 0\), because each summand \(A^i X (A^i)^\dagger \) is positive semidefinite. The abstract positive-map framework appears in Chapter 4.

2.3 Gauge equivalence and same MPV

Definition 2.10 Gauge equivalence
✓
#

Uses

  • Definition 2.1

Lean declarations

  • MPSTensor.GaugeEquiv

Two tensors \(A\) and \(B\) of the same bond dimension \(D\) are gauge equivalent if there exists an invertible matrix \(X \in \mathrm{GL}_D(\mathbb {C})\) such that, for every \(i \in \{ 0,\ldots ,d{-}1\} \),

\[ B^i \; =\; X \, A^i \, X^{-1} \]

In tensor-network notation,

the black node denotes the tensor named in the surrounding formula, the upper leg is the physical index \(i\), and the two red side nodes denote the gauge matrices acting on the virtual legs.

Definition 2.11 Same MPV
✓
#

Uses

  • Definition 2.4

Lean declarations

  • MPSTensor.SameMPV

Two tensors \(A\) and \(B\) (of the same bond dimension) generate the same MPV family if \(|V^{(N)}(A)\rangle = |V^{(N)}(B)\rangle \) for every system size \(N\).

Definition 2.12 Same MPV — different bond dimensions
✓
#

Uses

  • Definition 2.4

Lean declarations

  • MPSTensor.SameMPV₂

For tensors \(A\) and \(B\) of possibly different bond dimensions \(D_1\) and \(D_2\), we write \(\mathcal{V}(A) = \mathcal{V}(B)\) if \(|V^{(N)}(A)\rangle = |V^{(N)}(B)\rangle \) for every \(N\).

Definition 2.13 Positive-length same MPV — different bond dimensions
✓
#

Uses

  • Definition 2.4

Lean declarations

  • MPSTensor.SameMPV₂Pos

For tensors \(A\) and \(B\) of possibly different bond dimensions, we write \(\mathcal{V}^+(A) = \mathcal{V}^+(B)\) if \(|V^{(N)}(A)\rangle = |V^{(N)}(B)\rangle \) for every \(N {\gt} 0\).

Remark 2.14
✓
#

Definition 2.11 is the same-bond-dimension specialization of Definition 2.12. We keep both because later results use both the same-dimension and different-dimension forms.

Definition 2.15 Proportional MPV
✓
#

Uses

  • Definition 2.4

Lean declarations

  • MPSTensor.ProportionalMPV₂

Two tensors \(A\) and \(B\) (of possibly different bond dimensions) generate proportional MPV families if for every \(N\) there exists \(c_N \in \mathbb {C}\) such that \(|V^{(N)}(A)\rangle = c_N \, |V^{(N)}(B)\rangle \).

Lemma 2.16 Scaling of word evaluation
✓
#

Uses

  • Definition 2.2
  • Definition 2.2

Lean declarations

  • MPSTensor.evalWord_smul

Let \(\zeta A\) denote the tensor with matrices \(\{ \zeta \, A^i\} _{i=0}^{d-1}\). Then for any word \(w\),

\[ (\zeta A)^w \; =\; \zeta ^{|w|} \, A^w, \]

where \(|w|\) is the length of \(w\).

Proof ▶

Induction on \(w\): the base case gives \(\zeta ^0 = 1\), and each step picks up one factor of \(\zeta \).

Lemma 2.17 Scaling of MPV components
✓
#

Uses

  • Definition 2.4
  • Lemma 2.16
  • Lemma 2.16

Lean declarations

  • MPSTensor.mpv_smul

Let \(\zeta A\) denote the tensor with matrices \(\{ \zeta \, A^i\} _{i=0}^{d-1}\). Then for every \(N\) and every configuration \(\sigma = (i_1, \ldots , i_N)\),

\[ V^{(N)}(\zeta A)_\sigma \; =\; \zeta ^{N} \, V^{(N)}(A)_\sigma . \]
Proof ▶

Apply Lemma 2.16 to the word \(w = (i_1, \ldots , i_N)\) of length \(N\), then take the trace; the scalar \(\zeta ^{N}\) pulls out by linearity.

Definition 2.18 Gauge-phase equivalence
✓
#

Uses

  • Definition 2.1

Lean declarations

  • MPSTensor.GaugePhaseEquiv
  • MPSTensor.GaugeEquiv.toGaugePhaseEquiv

Two tensors \(A\) and \(B\) are gauge-phase equivalent if there exist \(X \in \mathrm{GL}_D(\mathbb {C})\) and \(\zeta \in \mathbb {C}\setminus \{ 0\} \) such that, for every physical index \(i\),

\[ B^i \; =\; \zeta \, X \, A^i \, X^{-1} \]
Proof ▶

Gauge equivalence is the special case \(\zeta =1\).

Remark 2.19
✓
#

After spectral-radius normalization one may restrict \(\zeta \) to a unit-modulus phase \(e^{i\phi }\). We allow an arbitrary nonzero scalar because later canonical-form statements keep the block-scaling factors explicit.

Lemma 2.20 Gauge covariance of word evaluation
✓
#

Uses

  • Definition 2.2
  • Definition 2.2

Lean declarations

  • MPSTensor.evalWord_gauge

If \(B^i = X \, A^i \, X^{-1}\) for all \(i\), then for every word \(w\),

\[ B^w \; =\; X \, A^w \, X^{-1}. \]
Proof ▶

Induction on the word length, using \(X X^{-1} = \mathbb {1}\).

Theorem 2.21 Gauge invariance of MPVs
✓
#

Uses

  • Definition 2.10
  • Definition 2.11
  • Lemma 2.20

Lean declarations

  • MPSTensor.GaugeEquiv.sameMPV

If \(A\) and \(B\) are gauge equivalent, then they generate the same MPV family.

Proof ▶

Let \(B^i = X A^i X^{-1}\). By Lemma 2.20, \(B^w = X \, A^w \, X^{-1}\) for every word \(w\). Then \(\operatorname{tr}(B^w) = \operatorname{tr}(X \, A^w \, X^{-1}) = \operatorname{tr}(A^w)\) by cyclicity of the trace.

Theorem 2.22 Gauge-phase scaling of MPVs
✓
#

Uses

  • Definition 2.18
  • Definition 2.4
  • Lemma 2.16
  • Lemma 2.20

Lean declarations

  • MPSTensor.mpv_eq_pow_mul_of_gaugePhase

If, for every physical index \(i\),

\[ B^i = \zeta \, X \, A^i \, X^{-1} \]

then for every system size \(N\) and configuration \(\sigma \) one has

\[ V^{(N)}(B)_\sigma = \zeta ^N V^{(N)}(A)_\sigma . \]
Proof ▶

Combine gauge covariance of word evaluation with scalar scaling, then take traces.

2.4 Injectivity and normality

Definition 2.23 Injectivity
✓
#

Uses

  • Definition 2.1

Lean declarations

  • MPSTensor.IsInjective

A tensor \(A\) is injective if its matrices span the full matrix algebra: \(\operatorname{span}_{\mathbb {C}}\{ A^i : i = 0,\ldots ,d{-}1\} = M_{D}(\mathbb {C})\).

Definition 2.24 \(L\)-block injectivity
✓
#

Uses

  • Definition 2.1

Lean declarations

  • MPSTensor.IsNBlkInjective

A tensor \(A\) is \(L\)-block injective if \(\operatorname{span}_{\mathbb {C}}\{ A^{i_1} \cdots A^{i_L} : (i_1,\ldots ,i_L) \in \{ 0,\ldots ,d{-}1\} ^L\} = M_{D}(\mathbb {C})\). An injective tensor is \(1\)-block injective.

Lemma 2.25 One-site injectivity gives one-block injectivity
✓
#

Uses

  • Definition 2.23
  • Definition 2.24

Lean declarations

  • MPSTensor.isNBlkInjective_one_of_isInjective

If the one-site matrices \(\{ A^i\} _{i=0}^{d-1}\) span \(M_{D}(\mathbb {C})\), then the length-one word products span \(M_{D}(\mathbb {C})\), so \(A\) is \(1\)-block injective.

Proof ▶

The length-one words are exactly the letters \(i \in \{ 0,\ldots ,d-1\} \), and evaluating a length-one word gives the matrix \(A^i\). Thus the length-one word span is the same span as in one-site injectivity.

Definition 2.26 Normal tensor
✓
#

Uses

  • Definition 2.24

Lean declarations

  • MPSTensor.IsNormal

A tensor \(A\) is normal [ CPGSV21 ] if there exists a finite \(L_0\) such that \(A\) is \(L_0\)-block injective.

Remark 2.27
✓
#

The algebraic characterization of normality in Definition 2.26 — eventual full matrix span, equivalently eventual block injectivity — is equivalent to the spectral characterization used in [ CPGSV21 , Definition IV.1 ] : after TP normalization, the transfer map \(\mathcal{E}_A\) is a primitive channel, equivalently irreducible with trivial peripheral spectrum. This equivalence is proved in [ SPGWC10 , Proposition 3 ] . In what follows, the directions of the equivalence are supplied by Theorem 8.66, Theorem 8.76, and Theorem 9.30. In particular, every subsequent use of “normal” here refers to this single notion; Definition 9.38 employs the spectral formulation, but it describes the same class of tensors.

2.5 Canonical form

Definition 2.28 Block-injective canonical form
✓
#

Uses

  • Definition 2.1
  • Definition 2.23

Lean declarations

  • MPSTensor.CanonicalForm

Here, a block-injective canonical form for a tensor consists of:

  1. a number of blocks \(r \ge 1\),

  2. bond dimensions \(D_1, \ldots , D_r\),

  3. injective tensors \(\{ A_k^i\} _{i=0}^{d-1}\) with \(A_k^i \in M_{D_k}(\mathbb {C})\) for each block \(k \in \{ 1,\ldots ,r\} \),

  4. scaling factors \(\mu _1, \ldots , \mu _r \in \mathbb {C}\).

The associated tensor is the block-diagonal tensor

\[ A^i \; =\; \bigoplus _{k=1}^{r} \mu _k \, A_k^i. \]

This block-injective formulation of the canonical form is used in [ PGVWC07 ] ; see also  [ CPGSV21 ] for the normal canonical form.

Remark 2.29
✓
#

Here we use the block-injective canonical form: each block is injective in the sense of Definition 2.23. This is stronger than the NT-block canonical form of [ CPGSV21 ] , where the blocks are only normal in the spectral sense. Chapter 9 treats the normal canonical form needed for the blocking-based multi-block theory, while Chapter 14 develops the separate periodic irreducible-form extension. This section isolates the stronger block-injective form used in the injective and block-injective arguments.

Definition 2.30 Block projection
✓
#

Lean declarations

  • Matrix.blockProjection

For a finite direct sum \(\bigoplus _k \mathbb {C}^{D_k}\), the block projection \(P_j\) is the matrix that is the identity on the \(j\)-th summand and zero on all other summands.

Definition 2.31 Block-diagonal matrix
✓
#

Lean declarations

  • Matrix.IsBlockDiagonal'

A matrix on \(\bigoplus _k \mathbb {C}^{D_k}\) is block diagonal if it is a direct sum of square matrices, one on each summand.

Lemma 2.32 Off-block characterization
✓
#

Uses

  • Definition 2.31

Lean declarations

  • Matrix.isBlockDiagonal'_iff_offBlock_zero

A matrix on \(\bigoplus _k \mathbb {C}^{D_k}\) is block diagonal if and only if every entry from one summand to a distinct summand is zero.

Proof ▶

The forward direction is immediate from the definition of a direct sum of matrices. Conversely, if all off-block entries vanish, the diagonal blocks reconstruct the original matrix.

Lemma 2.33 Commutation with block projections

Theorem 2.36 MPV decomposition
✓

  • MPSTensor.mpv_toTensorFromBlocks_eq_sum

The MPV of a block-diagonal tensor decomposes as

\[ |V^{(N)}(A)\rangle \; =\; \sum _{k=1}^{r} \mu _k^N \, |V^{(N)}(A_k)\rangle . \]

Equivalently, for each configuration \(\sigma \) one has

\[ V^{(N)}(A)_\sigma \; =\; \sum _{k=1}^{r} \mu _k^N \, V^{(N)}(A_k)_\sigma . \]
Proof ▶

By Lemma 2.35, the full word evaluation is block diagonal with blocks \(\mu _k^N A_k^w\) for a word \(w\) of length \(N\). Taking the trace of a block-diagonal matrix gives the sum of the block traces: \(\operatorname{tr}(A^w) = \sum _k \mu _k^N \operatorname{tr}(A_k^w)\).

2.6 Blocking

Definition 2.37 Blocked tensor
✓
#

Uses

Lean declarations

Given a tensor \(A\) and a blocking length \(L\), the \(L\)-blocked tensor is the tensor with physical index set \(\{ 0,\ldots ,d{-}1\} ^L\) (hence physical dimension \(d^L\)) and the same bond dimension \(D\), whose matrices are indexed by words \((i_1, \ldots , i_L) \in \{ 0,\ldots ,d{-}1\} ^L\):

\[ (A^{[L]})^{(i_1, \ldots , i_L)} \; =\; A^{i_1} A^{i_2} \cdots A^{i_L}. \]
Lemma 2.38 Blocked word evaluation
✓

If \(w = (J_1, \ldots , J_m)\) is a word in the blocked alphabet, where each \(J_k\) is an \(L\)-tuple in \(\{ 0, \ldots , d{-}1\} ^L\), then concatenating the block words gives a word \(\widetilde{w}\) of length \(mL\) in the original alphabet and one has

2.7 MPV overlap

Definition 2.41 MPV as Hilbert-space vector
✓
#

Uses

  • Definition 2.4

Lean declarations

  • MPSTensor.mpvState

We regard \(|V^{(N)}(A)\rangle \) as an element of \(\mathbb {C}^{d^N}\) indexed by configurations \(\sigma \in \{ 0,\ldots ,d{-}1\} ^N\), with \(\sigma \)-component \(V^{(N)}(A)_\sigma \) as in Definition 2.4. This allows us to use the standard Euclidean inner product on \(\mathbb {C}^{d^N}\).

Definition 2.42 MPV inner product
✓
#

Uses

  • Definition 2.41

Lean declarations

  • MPSTensor.mpvInner

The MPV inner product between two tensors \(A\) and \(B\) at system size \(N\) is

\[ \langle V^{(N)}(A) | V^{(N)}(B) \rangle \; :=\; \sum _{\sigma \in \{ 0,\ldots ,d{-}1\} ^N} \overline{V^{(N)}(A)_\sigma } \; V^{(N)}(B)_\sigma , \]

i.e. the Euclidean inner product on \(\mathbb {C}^{d^N}\) (conjugate-linear in the first argument).

Definition 2.43 MPV overlap
✓
#

Uses

  • Definition 2.4

Lean declarations

  • MPSTensor.mpvOverlap

The MPV overlap between two tensors \(A\) (of bond dimension \(D_1\)) and \(B\) (of bond dimension \(D_2\)) at system size \(N\) is

\[ O_{AB}(N) \; :=\; \sum _{\sigma \in \{ 0,\ldots ,d{-}1\} ^N} V^{(N)}(A)_\sigma \; \overline{V^{(N)}(B)_\sigma }. \]

This is the bilinear overlap (without complex conjugation) used later. By Lemma 2.44, it is the complex conjugate of the Hilbert-space inner product from Definition 2.42.
