11 Canonical Form Reduction

This chapter reduces a general MPS tensor to block-diagonal data via iterated invariant-subspace decomposition. Two reduction schemes are relevant. The block-injective reduction aims at injective TP-normalized blocks and the canonical form predicate of Chapter 12. ✓

For any nonzero scaling factors \(\mu _k\), set \(\eta _k := \mu _k / |\mu _k|\) and \(B_k := |\mu _k| A_k\). Then \(\bigoplus _k \mu _k A_k\) and \(\bigoplus _k \eta _k B_k\) generate the same MPV family. Equivalently, the \(k\)-th block contribution factors as

Definition 11.10 Support projection
✓
#

For a positive semi-definite matrix \(\rho \ge 0\), the support projection \(P\) is the orthogonal projection onto the range of \(\rho \). It is constructed via the spectral decomposition: if \(\rho = U \operatorname{diag}(\lambda _1,\ldots ,\lambda _D) U^\dagger \), then \(P = U \operatorname{diag}(\mathbf{1}_{\lambda _1{\gt}0},\ldots ,\mathbf{1}_{\lambda _D{\gt}0}) U^\dagger \), where \(\mathbf{1}_{\lambda _j{\gt}0}\) is \(1\) if \(\lambda _j {\gt} 0\) and \(0\) otherwise.

✓

Theorem 11.12 PSD fixed point gives invariant projection
✓

Let \(\rho \ge 0\) be a fixed point of \(\mathcal{E}_A\), and let \(P\) be the support projection of \(\rho \). Then for all \(i\), \((\mathbb {1}- P) A^i P = 0\): the range of \(P\) is invariant under all \(A^i\).

Theorem 11.13 Two-block decomposition
✓
#

Uses

  • Definition 2.5
  • Theorem 11.12
  • Theorem 11.9

Lean declarations

  • MPSTensor.exists_twoBlock_decomp_of_posSemidef_fixedPoint_strict

If the transfer map has a PSD fixed point \(\rho \) that is not positive definite (i.e. \(\rho \) has a nontrivial kernel), then \(A\) is unitarily equivalent to a two-block upper-triangular tensor. The trace of the upper-triangular product equals the sum of the diagonal-block traces, so the block-diagonal tensor obtained by discarding the off-diagonal blocks has the same MPV coefficients as the conjugated tensor, and hence as \(A\).

Theorem 11.15 Irreducible block decomposition
✓
#

Uses

  • Definition 11.14
  • Theorem 11.13

Lean declarations

Every MPS tensor \(A\) can be decomposed (up to unitary equivalence) into a block-diagonal tensor \(\bigoplus _{k=1}^r A_k\) where each \(A_k\) is irreducible. The block-diagonal tensor generates the same MPV as \(A\).

Proof ▶

Theorem 11.20 Irreducible tensor implies irreducible action
✓
#

If \(A\) is irreducible as a tensor (Definition 11.14), then the Kraus operators act irreducibly on \(\mathbb {C}^D\) (Definition 11.19).

Theorem 11.21 Burnside’s theorem for Kraus algebras
✓
#

Uses

  • Definition 11.17
  • Definition 11.19

Lean declarations

  • MPSTensor.burnside_matrix

Assume \(D {\gt} 0\). If \(A\) acts irreducibly on \(\mathbb {C}^D\), then the generated unital subalgebra is the full matrix algebra:

\[ \operatorname{alg}(A) = M_{D}(\mathbb {C}). \]

This is the complex finite-dimensional case of Burnside’s theorem (equivalently, Jacobson’s density theorem); see  .

Proof ▶

Transport \(\operatorname{alg}(A)\) along the algebra equivalence \(M_{D}(\mathbb {C}) \simeq _{\mathbb {C}} \operatorname {End}_{\mathbb {C}}(\mathbb {C}^D)\). The irreducible-action hypothesis makes \(\mathbb {C}^D\) a simple module for this algebra. Over the algebraically closed field \(\mathbb {C}\), Schur’s lemma identifies the endomorphisms commuting with the action with scalars, and Jacobson density then forces the action map to be surjective onto all of \(\operatorname {End}_{\mathbb {C}}(\mathbb {C}^D)\).

11.14 Open directions

Remark 11.83 Remaining steps toward normal canonical form
✓
#

The arbitrary-input reduction (Theorem 11.40) produces a zero tail together with left-canonical live blocks. After a common blocking step (Theorem 11.58), the blocked blocks have primitive transfer maps.

To conclude full normal canonical form, two further ingredients are still needed: tensor irreducibility for each blocked block (blocking does not in general preserve tensor irreducibility), and pairwise distinct weight norms. The cyclic-sector decomposition (Theorems 11.48 and 11.49) provides the infrastructure for the periodic-to-aperiodic splitting.
