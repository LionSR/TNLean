16 Quantum Dynamical Semigroups

This chapter develops the theory of one-parameter semigroups of quantum channels and characterizes their generators via the GKSL (Gorini–Kossakowski–Sudarshan–Lindblad) theorem, following  [ Wol12 , Chapter 7 ] .

Section 16.1 establishes definitions and the basic exponential form (Proposition 16.14). Section 16.2 gives the Duhamel formula and perturbation bound. Section 16.3 develops the generator theory and proves the GKSL theorem.

16.1 Dynamical semigroups and the exponential form

Definition 16.1 Dynamical semigroup
✓
#

Lean declarations

  • IsDynSemigroup

A family of linear maps \(T : \mathbb {R}\to (M_{D}(\mathbb {C}) \to _{\ell } M_{D}(\mathbb {C}))\) is a dynamical semigroup if

  1. (semigroup law) \(T_{t+s} = T_t \circ T_s\) for all \(t,s \ge 0\); and

  2. (initial condition) \(T_0 = \operatorname{id}\).

This is [ Wol12 , Equation (7.1) ] .

✓
#

Uses

Lean declarations

✓

Theorem 16.14 Every continuous semigroup is exponential
✓
#

Uses

  • Definition 16.2
  • Definition 16.3

Lean declarations

  • continuousDynSemigroup_eq_exp

Every norm-continuous dynamical semigroup \(T\) on \(M_{D}(\mathbb {C})\) is of the form

\[ T_t = e^{tL} \]

for some generator \(L \in \mathrm{End}_{\mathbb {C}}(M_{D}(\mathbb {C}))\) and every \(t \ge 0\). This is [ Wol12 , Proposition 7.1 ] .

Proof ▶

Since \(T_0 = \mathbb {1}\) and \(T\) is continuous, the integral \(M_\varepsilon = \int _0^\varepsilon T_s\, ds\) is invertible for small \(\varepsilon {\gt} 0\). Using the semigroup property, \(T_t = M_\varepsilon ^{-1}(M_{t+\varepsilon } - M_t)\) is differentiable for \(t \ge 0\). ODE uniqueness then gives \(T_t = e^{tL}\) for all \(t \ge 0\).

16.2 Perturbation theory

Theorem 16.15 Duhamel formula
✓
#

Uses

  • Definition 16.3
  • Theorem 16.7

Lean declarations

  • duhamel_formula

Let \(\Delta = L' - L\). For \(t \ge 0\),

\[ e^{tL'} - e^{tL} = \int _0^t e^{(t-s)L} \Delta \, e^{sL'}\, ds. \]

This is [ Wol12 , Lem. 7.1 ] .

Proof ▶

Define \(f(s) = e^{(t-s)L} e^{sL'}\). Then \(f'(s) = e^{(t-s)L} \Delta \, e^{sL'}\) (Theorem 16.7), and \(e^{tL'} - e^{tL} = f(t) - f(0) = \int _0^t f'(s)\, ds\).

Theorem 16.16 Perturbation bound
✓
#

Uses

\[ \bigl\| e^{tL'} - e^{tL}\bigr\| \le t\, \| \Delta \| \cdot \sup _{s \in [0,t]}\| e^{sL}\| \cdot \sup _{s \in [0,t]}\| e^{sL'}\| , \]

where \(\Delta = L' - L\). This is [ Wol12 , Corollary 7.1 ] .

Apply the triangle inequality and submultiplicativity of the operator norm to the Duhamel integral (Theorem 16.15).

Lemma 16.18 Dyson-series summability from factorial majorant
✓

16.3 GKSL/Lindblad generators

This section characterizes generators of semigroups of completely positive and trace-preserving maps. The central result is the GKSL theorem (Theorem 16.54), which shows that such generators have the standard Lindblad form.

16.3.1 Generator decomposition and conditional complete positivity

Definition 16.26 Generator decomposition
✓
#

Lean declarations

  • GeneratorDecomp

A generator decomposition consists of a completely positive map \(\phi : M_{d}(\mathbb {C}) \to M_{d}(\mathbb {C})\) and a matrix \(\kappa \in M_{d}(\mathbb {C})\), defining the linear map

\[ L(\rho ) = \phi (\rho ) - \kappa \rho - \rho \kappa ^\dagger . \]

This is [ Wol12 , Equation (7.14) ] .

Definition 16.27 Conditional complete positivity
✓
#

Uses

  • Definition 16.26

Lean declarations

  • IsCCP

A linear map \(L : M_{d}(\mathbb {C}) \to M_{d}(\mathbb {C})\) is conditionally completely positive (CCP) if it admits a generator decomposition, i.e. there exist a CP map \(\phi \) and a matrix \(\kappa \) such that \(L(\rho ) = \phi (\rho ) - \kappa \rho - \rho \kappa ^\dagger \). This is condition 1 of [ Wol12 , Proposition 7.2 ] .

Definition 16.28 Trace-annihilating map
✓
#

Lean declarations

  • IsTraceAnnihilating

A linear map \(L : M_{d}(\mathbb {C}) \to M_{d}(\mathbb {C})\) is trace-annihilating if \(\operatorname{tr}(L(\rho )) = 0\) for all \(\rho \in M_{d}(\mathbb {C})\). This is the infinitesimal version of trace preservation.

Definition 16.29 Trace constraint
✓
#

Uses

  • Definition 16.26

Lean declarations

  • GeneratorDecomp.isTraceConstraint

A generator decomposition \((\phi , \kappa )\) satisfies the trace constraint if \(\phi ^*(\mathbb {1}) = \kappa + \kappa ^\dagger \), where \(\phi ^*\) is the Hilbert–Schmidt adjoint. This is the condition in [ Wol12 , Equation (7.20) ] .

Theorem 16.30 Trace constraint implies trace-annihilating
✓
#

Uses

  • Definition 16.26
  • Definition 16.28
  • Definition 16.29

Lean declarations

  • GeneratorDecomp.traceAnnihilating_of_traceConstraint

If \((\phi , \kappa )\) satisfies \(\phi ^*(\mathbb {1}) = \kappa + \kappa ^\dagger \), then \(L(\rho ) = \phi (\rho ) - \kappa \rho - \rho \kappa ^\dagger \) is trace-annihilating.
