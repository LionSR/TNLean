4 The Single-Block Fundamental Theorem

This chapter proves the single-block injective Fundamental Theorem of MPS [ CPGSV21 , Section III.B ] : if an injective MPS tensor \(A\) generates the same MPV family as another tensor \(B\) of the same bond dimension, then there exists an invertible matrix \(X\) such that \(B^i = X A^i X^{-1}\) for all physical indices \(i\). The proof is purely algebraic and does not require the later overlap or normal-form reductions.

4.1 Trace pairing

Theorem 4.1 Nondegeneracy of the trace pairing
✓
#

Lean declarations

  • Matrix.trace_mul_right_eq_zero_iff

For \(M \in M_{D}(\mathbb {C})\), \(\operatorname{tr}(M N) = 0\) for all \(N \in M_{D}(\mathbb {C})\) if and only if \(M = 0\).

Proof ▶

Testing against the matrix units \(E_{ji}\) (the matrix with \(1\) in position \((j,i)\) and \(0\) elsewhere) gives \(\operatorname{tr}(E_{ji} M) = M_{ij}\). Hence if \(\operatorname{tr}(MN) = 0\) for all \(N\), then every entry of \(M\) vanishes.

Viewing the word \(w\) as a configuration of length \(|w|\), Definition 2.4 gives the corresponding MPV coefficient as the trace coefficient \(c_w(A)=\operatorname{tr}(A^w)\). The same-MPV hypothesis gives equality at every system size.

Lemma 4.5 Nonvanishing trace on injective tensors

If \(D \ge 1\), \(A\) is injective, and \(A\) and \(B\) generate the same MPV family, then \(B^i\) cannot be identically zero: \(\neg (\forall i, B^i = 0)\).

Proof ▶

If \(B^i = 0\) for all \(i\), then Lemma 4.2 gives \(\operatorname{tr}(A^w) = 0\) for every word \(w\), in particular for all length-1 words. Since the \(\{ A^i\} \) span \(M_{D}(\mathbb {C})\), the trace functional vanishes identically, contradicting \(\operatorname{tr}(\mathbb {1}_D) = D \neq 0\).

4.2 Linear extension

Theorem 4.6 Existence and uniqueness of the linear extension
✓
#

Uses

  • Definition 2.23
  • Definition 2.11
  • Theorem 4.4
  • Lemma 4.2

Lean declarations

  • MPSTensor.linearExtension_exists_unique

If \(A\) is injective and \(A\), \(B\) generate the same MPV family, then there exists a unique linear map \(T : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) satisfying \(T(A^i) = B^i\) for all \(i\).

Proof ▶

Write \(\ell _A : \mathbb {C}^d \to M_{D}(\mathbb {C})\) for the map that sends a coefficient vector \(c\) to \(\sum _i c_i A^i\) (linear combination with the \(A^i\)). Applying Lemma 4.2 to all words of length \(2\) shows \(\Phi _A \circ \ell _A = \Phi _B \circ \ell _B\). Since \(A\) is injective, \(\ell _A\) is surjective, giving \(\mathrm{range}(\Phi _A) \subseteq \mathrm{range}(\Phi _B)\). By Theorem 4.4, \(\ker \Phi _A = \{ 0\} \), so rank–nullity gives \(\dim (\mathrm{range}(\Phi _A)) = \dim (M_{D}(\mathbb {C}))\). Hence \(\dim (\mathrm{range}(\Phi _B)) \ge \dim (M_{D}(\mathbb {C}))\), and rank–nullity again gives \(\ker \Phi _B = \{ 0\} \). A left inverse \(g\) of \(\Phi _B\) gives \(T := g \circ \Phi _A\), and uniqueness follows because the \(\{ A^i\} \) span \(M_{D}(\mathbb {C})\).

Theorem 4.7 Multiplicativity of the linear extension
✓
#

Uses

  • Definition 2.23
  • Definition 2.11
  • Theorem 4.6
  • Lemma 4.2

Lean declarations

  • MPSTensor.linearExtension_mul

If \(A\) is injective and \(A\), \(B\) generate the same MPV family, and \(T : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) is the unique linear map satisfying \(T(A^i) = B^i\) for all \(i\), then \(T(MN) = T(M) T(N)\) for all \(M, N \in M_{D}(\mathbb {C})\).

Proof ▶

Step 1 (\(\Phi _B \circ T = \Phi _A\)). Applying Lemma 4.2 to words of length \(2\) gives \(\Phi _B(T(A^i)) = \Phi _B(B^i) = \Phi _A(A^i)\) for all \(i\). Since the \(\{ A^i\} \) span \(M_{D}(\mathbb {C})\), \(\Phi _B \circ T = \Phi _A\).

Step 2 (\(T(A^i A^j) = B^i B^j\)). Lemma 4.2 applied to the word \((i,j,k)\) gives \(\operatorname{tr}(A^i A^j A^k) = \operatorname{tr}(B^i B^j B^k)\) for all \(i,j,k\). So \(\Phi _B(T(A^i A^j))_k = \Phi _A(A^i A^j)_k = \operatorname{tr}(A^i A^j A^k) = \operatorname{tr}(B^i B^j B^k) = \Phi _B(B^i B^j)_k\). The same rank–nullity argument used in the proof of Theorem 4.6 gives \(\ker \Phi _B = \{ 0\} \). Hence \(T(A^i A^j) = B^i B^j\).

Step 3 (bilinear extension). For fixed \(j\), the maps \(M \mapsto T(M \cdot A^j)\) and \(M \mapsto T(M) \cdot B^j\) agree on the spanning set \(\{ A^i\} \), hence on all of \(M_{D}(\mathbb {C})\). Applying this once more with \(j\) replaced by a general matrix gives full multiplicativity.

4.3 Simplicity and Skolem–Noether

Theorem 4.8 Simplicity of \(M_{D}(\mathbb {C})\)
✓
#

Lean declarations

  • MPSTensor.linear_mul_endomorphism_bijective

A nonzero multiplicative \(\mathbb {C}\)-linear endomorphism of \(M_{D}(\mathbb {C})\) is bijective.

Proof ▶

The kernel of a multiplicative linear map is a two-sided ideal. Since \(M_{D}(\mathbb {C})\) is a simple ring (it has no nontrivial two-sided ideals), the kernel is either \(\{ 0\} \) or \(M_{D}(\mathbb {C})\). The latter would force the map to be zero, contradicting the hypothesis. Hence the map is injective, and surjectivity follows from finite-dimensionality.

Theorem 4.9 Skolem–Noether
✓
#

Lean declarations

  • MPSTensor.skolemNoether_matrix

Every \(\mathbb {C}\)-algebra automorphism of \(M_{D}(\mathbb {C})\) is inner: for any \(f \in \mathrm{Aut}_{\mathbb {C}\text{-alg}}(M_{D}(\mathbb {C}))\), there exists \(X \in \mathrm{GL}_D(\mathbb {C})\) such that \(f(M) = X M X^{-1}\) for all \(M\).

Proof ▶

Via the canonical identification \(M_{D}(\mathbb {C}) \cong \mathrm{End}_{\mathbb {C}}(\mathbb {C}^D)\), the automorphism \(f\) induces an automorphism of \(\mathrm{End}_{\mathbb {C}}(\mathbb {C}^D)\). Every algebra automorphism of \(\mathrm{End}(V)\) is conjugation by an invertible linear map (this is a standard result in the theory of central simple algebras). Translating back to matrices gives the invertible \(X\).

Lemma 4.10 Unit preservation of multiplicative surjective maps
✓
#

Lean declarations

  • MPSTensor.linearMapToAlgHom

If \(T : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) is a multiplicative surjective \(\mathbb {C}\)-linear map, then \(T(\mathbb {1}) = \mathbb {1}\).

Proof ▶

By surjectivity, there exists \(X\) with \(T(X) = \mathbb {1}\). Then \(T(\mathbb {1}) = T(X) \cdot T(\mathbb {1}) = T(X \cdot \mathbb {1}) = T(X) = \mathbb {1}\), using multiplicativity and \(T(X) = \mathbb {1}\). With \(T(\mathbb {1}) = \mathbb {1}\) established, \(T\) preserves the unit and hence is a \(\mathbb {C}\)-algebra homomorphism.

4.4 Proof of the single-block theorem

Theorem 4.11 Single-block Fundamental Theorem
✓
#

  • Theorem 4.8

Lean declarations

  • MPSTensor.fundamentalTheorem_singleBlock

Let \(A\) be an injective MPV tensor and let \(B\) be an MPV tensor of the same bond dimension. If \(A\) and \(B\) generate the same MPV family, then they are gauge equivalent: there exists \(X \in \mathrm{GL}_D(\mathbb {C})\) such that \(B^i = X A^i X^{-1}\) for all \(i\).

Proof ▶

By Theorem 4.6, there is a unique linear \(T\) with \(T(A^i) = B^i\). By Theorem 4.7, \(T\) is multiplicative. \(T\) is nonzero: if \(T = 0\) then \(B^i = 0\) for all \(i\), contradicting Lemma 4.5. By Theorem 4.8, \(T\) is bijective. By Lemma 4.10, \(T\) is a \(\mathbb {C}\)-algebra automorphism. Theorem 4.9 gives an invertible \(X\) with \(T(M) = X M X^{-1}\), and evaluating at \(M = A^i\) yields \(B^i = X A^i X^{-1}\).
