4 Quantum Channels and Positive Maps

This chapter develops the theory of positive maps and quantum channels on matrix algebras, following  [ Wol12 ] .

4.1 Positive maps and channels

Definition 4.1 Positive map
✓
#

Lean declarations

  • IsPositiveMap

A linear map \(E : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) is positive if \(E(X) \ge 0\) whenever \(X \ge 0\), where \(X \ge 0\) means that \(X\) is positive semidefinite.

Remark 4.2
✓
#

We use the Loewner order on matrices: \(X \le Y\) means \(Y - X \ge 0\).

Definition 4.3 Trace-preserving map
✓
#

Lean declarations

  • IsTracePreservingMap

A linear map \(E : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) is trace-preserving if \(\operatorname{tr}(E(X)) = \operatorname{tr}(X)\) for all \(X\).

Definition 4.4 Completely positive map
✓
#

Lean declarations

  • IsCPMap

A linear map \(E : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) is completely positive (CP) if it admits a Kraus representation: there exist operators \(\{ K_i\} _{i=0}^{r-1}\) with \(K_i \in M_{D}(\mathbb {C})\) such that, for every \(X \in M_{D}(\mathbb {C})\),

\[ E(X) = \sum _{i=0}^{r-1} K_i X K_i^\dagger . \]

We use the Kraus characterisation because it matches the later Kadison–Schwarz arguments. By Choi’s theorem, this is equivalent to \(E \otimes \operatorname{id}_n\) being positive for all \(n\).

Theorem 4.5 CP maps are positive
✓
#

Uses

  • Definition 4.4
  • Definition 4.1

Lean declarations

  • IsCPMap.isPositiveMap

Every completely positive map is positive.

Proof ▶

If \(X \ge 0\), then each summand \(K_i X K_i^\dagger \ge 0\) (conjugation preserves positive semidefiniteness), so the sum is PSD.

Theorem 4.6 Channels are positive
✓
#

Uses

  • Definition 4.8
  • Definition 4.1
  • Theorem 4.5

Lean declarations

  • IsChannel.pos

Every quantum channel is positive.

Proof ▶

A quantum channel is completely positive by definition, so Theorem 4.5 applies.

Theorem 4.7 Positive maps preserve Hermiticity
✓
#

Uses

  • Definition 4.1

Lean declarations

  • IsPositiveMap.map_isHermitian

If \(E\) is positive and \(X = X^\dagger \), then \(E(X) = E(X)^\dagger \).

Proof ▶

Decompose \(X = X_+ - X_-\) into its positive and negative parts. Positivity makes \(E(X_+)\) and \(E(X_-)\) positive semidefinite, hence Hermitian, so their difference \(E(X)\) is Hermitian.

Definition 4.8 Quantum channel
✓
#

Uses

  • Definition 4.4
  • Definition 4.3

Lean declarations

  • IsChannel

A linear map \(E : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) is a quantum channel if it is both completely positive and trace-preserving (CPTP), following  [ Wol12 ] .

Definition 4.9 Density matrices
✓
#

Lean declarations

  • densityMatrices
  • mem_densityMatrices

The set of density matrices in \(M_{D}(\mathbb {C})\) is

\[ \mathcal{D}_D = \{ \rho \in M_{D}(\mathbb {C}) : \rho \ge 0, \operatorname{tr}(\rho ) = 1\} . \]
Theorem 4.10 Density matrices are compact
✓
#

Uses

  • Definition 4.9

Lean declarations

  • densityMatrices_isCompact

\(\mathcal{D}_D\) is compact (in the entry-wise topology on \(M_{D}(\mathbb {C})\)).

Proof ▶

The PSD cone is closed (preimage of the closed non-negative cone under continuous quadratic forms), and if \(\rho \ge 0\) with \(\operatorname{tr}(\rho ) = 1\) then each entry satisfies \(|\rho _{ij}|^2 \le \rho _{ii}\rho _{jj} \le 1\): the diagonal entries are non-negative and sum to \(1\), and the off-diagonal bound is the PSD Cauchy–Schwarz inequality. Hence the matrix entries are uniformly bounded, and Heine–Borel gives compactness.

Theorem 4.11 Density matrices are convex
✓
#

Uses

  • Definition 4.9

Lean declarations

  • densityMatrices_isConvex

\(\mathcal{D}_D\) is convex.

Proof ▶

Convex combination of PSD matrices is PSD, and trace is linear.

Theorem 4.12 Density matrices are nonempty
✓
#

Uses

  • Definition 4.9

Lean declarations

  • densityMatrices_nonempty

For \(D \ge 1\), \(\mathcal{D}_D \neq \varnothing \).

Proof ▶

\(\frac{1}{D} \mathbb {1}_D\) is a density matrix.

Theorem 4.13 Channels preserve density matrices
✓
#

Uses

  • Definition 4.8
  • Definition 4.9
  • Theorem 4.5

Lean declarations

  • IsChannel.map_densityMatrices

If \(E\) is a quantum channel, then \(E(\mathcal{D}_D) \subseteq \mathcal{D}_D\).

Proof ▶

Complete positivity implies positivity (Theorem 4.5), so \(E(\rho ) \ge 0\); trace preservation gives \(\operatorname{tr}(E(\rho )) = \operatorname{tr}(\rho ) = 1\).

4.2 Irreducibility

Definition 4.14 Orthogonal projection
✓
#

Lean declarations

  • IsOrthogonalProjection

A matrix \(P \in M_{D}(\mathbb {C})\) is an orthogonal projection if \(P = P^\dagger \) and \(P^2 = P\).

Definition 4.15 Irreducible map
✓
#

Uses

  • Definition 4.14

Lean declarations

  • IsIrreducibleMap

A linear map \(E : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) is irreducible if whenever \(P\) is an orthogonal projection satisfying \(E(P M_{D}(\mathbb {C}) P) \subseteq P M_{D}(\mathbb {C}) P\), then \(P = 0\) or \(P = \mathbb {1}\). This is [ Wol12 , Theorem 6.2, condition (1) ] . The definition applies to any linear map; complete positivity is not required.

Theorem 4.16 Injectivity implies irreducibility
✓
#

Uses

  • Definition 2.28
  • Definition 2.5
  • Definition 4.15

Lean declarations

  • MPSTensor.injective_implies_irreducibleCP

If \(A\) is an injective MPS tensor, then its transfer map \(\mathcal{E}_A\) is irreducible.

Proof ▶

If \(P\) is an invariant projection for \(\mathcal{E}_A\), then \((\mathbb {1}- P) A^i P = 0\) for all \(i\). Since the \(\{ A^i\} \) span \(M_{D}(\mathbb {C})\), \((\mathbb {1}- P) M P = 0\) for all \(M\), so \(P = 0\) or \(P = \mathbb {1}\).

Remark 4.17 Transfer map is CP
✓
#

Uses

  • Definition 2.5
  • Definition 4.4

Lean declarations

The transfer map \(\mathcal{E}_A(X) = \sum _i A^i X (A^i)^\dagger \) is completely positive: it is already written in Kraus form, with Kraus operators \(\{ A^i\} \).

Theorem 4.18 Transfer map is a channel
✓
#

Uses

  • Definition 2.5
  • Definition 4.8
  • Remark 4.17

Lean declarations

  • MPSTensor.transferMap_isChannel

If \(\sum _i (A^i)^\dagger A^i = \mathbb {1}\), then the transfer map \(\mathcal{E}_A(X) = \sum _i A^i X (A^i)^\dagger \) is a quantum channel (CPTP).

Proof ▶

Complete positivity is the observation of Remark 4.17. Trace preservation: \(\operatorname{tr}(\mathcal{E}_A(X)) = \sum _i \operatorname{tr}(A^i X (A^i)^\dagger ) = \operatorname{tr}((\sum _i (A^i)^\dagger A^i) X) = \operatorname{tr}(X)\) by cyclicity and the normalization hypothesis.

4.3 Cesàro fixed-point theorem

Definition 4.19 Cesàro mean
✓
#

Lean declarations

  • cesaroMean

The Cesàro mean of a linear map \(E\) at order \(N\), applied to a starting matrix \(\rho _0\), is

\[ \sigma _N = \frac{1}{N} \sum _{n=0}^{N-1} E^n(\rho _0). \]
Theorem 4.20 Cesàro telescope
✓
#

Uses

  • Definition 4.19
  • Definition 4.19

Lean declarations

  • cesaroMean_telescope

\(E(\sigma _N) - \sigma _N = \frac{1}{N}(E^N(\rho _0) - \rho _0)\).

Proof ▶

Telescoping: \(E(\sigma _N) = \frac{1}{N}\sum _{n=1}^{N} E^n(\rho _0)\) and subtracting \(\sigma _N\) cancels all but the first and last terms.

Theorem 4.21 Hermitian fixed-point decomposition
✓
#

Uses

  • Definition 4.8

Lean declarations

  • IsChannel.posSemidef_parts_of_hermitian_fixedPoint

Let \(E\) be a quantum channel and \(X = E(X)\) a Hermitian fixed point with decomposition \(X = P_+ - P_-\) into orthogonal positive and negative parts (\(P_\pm \ge 0\), \(\operatorname{tr}(P_+ P_-) = 0\)). Then \(E(P_+) = P_+\) and \(E(P_-) = P_-\). This is [ Wol12 , Proposition 6.8 ] .

Proof ▶

Let \(Q\) be the support projection of \(P_+\). Trace preservation and positivity give \(\operatorname{tr}(Q E(P_-)) = 0\) and \(E(P_-) \ge 0\), so \(Q E(P_-) = 0\). The fixed-point equation \(E(P_+) - E(P_-) = P_+ - P_-\) then identifies \(E(P_\pm ) = P_\pm \).

Theorem 4.22 Cesàro fixed-point theorem
✓
#

Uses

  • Definition 4.8
  • Definition 4.9
  • Definition 4.19
  • Theorem 4.10
  • Theorem 4.11
  • Theorem 4.12
  • Theorem 4.13
  • Theorem 4.20

Lean declarations

  • IsChannel.exists_posSemidef_fixedPoint

Every quantum channel \(E : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) (with \(D \ge 1\)) has a nonzero positive semidefinite fixed point: there exists \(\rho \ge 0\), \(\rho \neq 0\), with \(E(\rho ) = \rho \).

Proof ▶

Start with any \(\rho _0 \in \mathcal{D}_D\) (Theorem 4.12). Each Cesàro mean \(\sigma _N\) lies in \(\mathcal{D}_D\) (by convexity and channel preservation). By compactness (Theorem 4.10), extract a convergent subsequence \(\sigma _{N_k} \to \rho \). The telescope identity (Theorem 4.20) gives \(\| E(\sigma _N) - \sigma _N\| = \frac{1}{N}\| E^N(\rho _0) - \rho _0\| \to 0\), so \(E(\rho ) = \rho \) by continuity. In  [ Wol12 , Theorem 6.11 ] , existence is proved via Brouwer’s fixed-point theorem; we use the Cesàro argument instead.

4.4 Fixed-point projection

Definition 4.23 Fixed-point projection
✓
#

Lean declarations

  • fixedPointProj

Given a linear map \(E : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) and a fixed point \(\rho \) with \(E(\rho ) = \rho \) and \(\operatorname{tr}(\rho ) \neq 0\), the fixed-point projection is the rank-one map

\[ P(X) = \frac{\operatorname{tr}(X)}{\operatorname{tr}(\rho )} \rho . \]

This projects onto the span of \(\rho \) along the kernel of the trace functional. When the fixed point is unique up to scaling, this span is the full fixed-point space.

Theorem 4.24 Power decomposition
✓
#

Uses

  • Definition 4.23
  • Definition 4.3
  • Definition 4.23

Lean declarations

  • pow_eq_fixedPointProj_add_compl_pow

If \(E\) is trace-preserving and \(E(\rho ) = \rho \), then for \(n \ge 1\),

\[ E^n = P + (E - P)^n, \]

where \(P\) is the fixed-point projection.

Proof ▶

\(P\) is idempotent (\(P^2 = P\)), the fixed-point relation \(E(\rho ) = \rho \) gives \(E \circ P = P\), and trace preservation gives \(P \circ E = P\). These imply \((E-P) \circ P = 0\) and \(P \circ (E-P) = 0\), so in the binomial expansion of \((P + (E-P))^n\) all cross terms vanish, leaving \(E^n = P^n + (E-P)^n = P + (E-P)^n\).

4.5 Peripheral spectrum and primitivity

Definition 4.25 Peripheral spectrum
✓
#

Lean declarations

  • peripheralSpectrum

The peripheral spectrum of a bounded linear operator \(T\) is the set of spectral values \(\mu \) with \(|\mu |\) equal to the spectral radius of \(T\).

Definition 4.26 Peripheral eigenvalues
✓
#

Lean declarations

  • peripheralEigenvalues

The peripheral eigenvalues of a linear map \(E\) on a finite-dimensional space are the eigenvalues \(\lambda \) on the unit circle: \(|\lambda | = 1\). For a channel, the spectral radius is \(1\) [ Wol12 , Proposition 6.1 ] , so these coincide with the eigenvalues of maximal modulus.

The condition \(|\lambda | = 1\) is appropriate for channels, since the spectral radius of a channel is \(1\); for a general linear map with spectral radius \(r \neq 1\) the condition would generalise to \(|\lambda | = r\).

Theorem 4.27 \(1\) is a peripheral eigenvalue
✓
#

Uses

  • Definition 4.26

Lean declarations

  • one_mem_peripheralEigenvalues

If \(E\) has a nonzero fixed point \(\rho \neq 0\) with \(E(\rho ) = \rho \), then \(1\) is a peripheral eigenvalue of \(E\).

Proof ▶

\(\rho \) is an eigenvector with eigenvalue \(1\), and \(|1| = 1\).

Remark 4.28 Peripheral eigenvalues lie on the unit circle
✓
#

Uses

  • Definition 4.26

Lean declarations

Every peripheral eigenvalue \(\lambda \) satisfies \(|\lambda | = 1\). This is simply the defining condition in Definition 4.26.

Theorem 4.29 Finiteness of peripheral eigenvalues
✓
#

Uses

  • Definition 4.26

Lean declarations

  • peripheralEigenvalues_finite

On a finite-dimensional space, the set of peripheral eigenvalues is finite.

Proof ▶

Peripheral eigenvalues are a subset of all eigenvalues, which is a finite set in finite dimensions.

Theorem 4.30 Power-stable peripheral eigenvalues are roots of unity
✓
#

Uses

  • Definition 4.26

Lean declarations

  • peripheral_isRootOfUnity_of_pow_eigenvalue

Let \(E\) be a linear endomorphism on a finite-dimensional space. If \(\mu \) is a peripheral eigenvalue of \(E\) and \(\mu ^n\) is an eigenvalue of \(E\) for every \(n \ge 1\), then \(\mu \) is a root of unity.

Proof ▶

Since \(\mu ^n\) is an eigenvalue for every \(n\), the pigeonhole principle on the finite eigenvalue set gives \(\mu ^a = \mu ^b\) for some \(a \neq b\), hence \(\mu ^{|a-b|} = 1\).

Theorem 4.31 Peripheral powers imply root of unity
✓
#

Uses

  • Definition 4.26
  • Theorem 4.30

Lean declarations

  • peripheral_isRootOfUnity_of_closed_powers

If the set of peripheral eigenvalues of a linear endomorphism \(E\) on a finite-dimensional space is closed under powers (\(\mu \in \mathrm{peripheral}(E)\) and \(n \ge 1\) imply \(\mu ^n \in \mathrm{peripheral}(E)\)), then every peripheral eigenvalue is a root of unity.

Proof ▶

The closure hypothesis provides \(\mu ^n \in \mathrm{peripheral}(E)\) for all \(n\), which in particular means \(\mu ^n\) is an eigenvalue. Theorem 4.30 then gives \(\mu ^p = 1\) for some \(p {\gt} 0\).

4.5.1 Periodicity removal by powering

Remark 4.32
✓
#

The results in this subsection are purely spectral: they use only finiteness of a set of complex numbers and the spectral mapping theorem for powers. In particular, they do not rely on complete positivity. For transfer maps, replacing a tensor by its \(p\)-blocked tensor replaces \(\mathcal{E}_A\) by \(\mathcal{E}_A^p\), so this powering step is the operator-theoretic form of blocking.

Lemma 4.33 Common exponent for a finite set of roots of unity
✓

Lemma 4.34 Powering collapses peripheral eigenvalues
✓

Uses

Let \(E : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) be a linear map, and assume that \(E\) has a nonzero fixed point \(\rho \neq 0\). Let \(p {\gt} 0\). If every peripheral eigenvalue \(\mu \) of \(E\) satisfies \(\mu ^p = 1\), then \(\mathrm{peripheral}(E^p) = \{ 1\} \).

For a normalized MPS tensor, the transfer map is naturally trace-preserving but need not be unital. In general one should not expect a single gauge choice to make it both unital and trace-preserving.

Remark 4.35 Bi-canonical special case
✓
#

When a Kraus map happens to be both unital and trace-preserving (the “bi-canonical” case), the Kadison–Schwarz equality at peripheral eigenvectors (Theorem 4.41) shows directly that the peripheral eigenvalues are closed under powers, and hence are roots of unity by Theorem 4.31. This is the simplest argument, but it requires both normalizations simultaneously. The more general adjoint-fixed-point formulation below covers the case where only unitality and a positive definite adjoint fixed point are available.

Theorem 4.37 Kadison–Schwarz inequality
✓

4.5.3 Peripheral closure via adjoint fixed point

The following results replace the trace-preservation hypothesis by the existence of a positive definite fixed point for the adjoint Kraus map, matching the weighted-trace argument in [ CPGSV17a , Appendix A ] .

Lemma 4.45 Peripheral closure via adjoint fixed point
✓
#

Uses

  • Definition 4.36
  • Definition 4.15
  • Theorem 4.37
  • Theorem 6.4
  • Theorem 4.40
  • Theorem 4.42

Lean declarations

  • MPSTensor.peripheralEigenvalues_pow_mem_of_irreducible_unital_of_adjoint_fixedPoint

Let \(E(X) = \sum _i K_i X K_i^\dagger \) be a Kraus map that is unital (\(\sum _i K_i K_i^\dagger = \mathbb {1}\)), and assume:

  1. the adjoint Kraus map \(E^*(X) = \sum _i K_i^\dagger X K_i\) has a positive definite fixed point \(\rho {\gt} 0\), and

  2. \(E\) is irreducible.

Then \(\mathrm{peripheral}(E)\) is closed under powers: if \(\mu \in \mathrm{peripheral}(E)\) then \(\mu ^n \in \mathrm{peripheral}(E)\) for every \(n \in \mathbb {N}\).

Proof ▶

Let \(E(X) = \mu X\) with \(|\mu | = 1\), and set \(G := E(X^\dagger X) - E(X)^\dagger E(X)\). By Kadison–Schwarz, \(G \ge 0\). Since \(E^*(\rho ) = \rho \),

\[ \operatorname{tr}(\rho G) = \operatorname{tr}(\rho E(X^\dagger X)) - \operatorname{tr}(\rho E(X)^\dagger E(X)) = \operatorname{tr}(E^*(\rho ) X^\dagger X) - \operatorname{tr}(\rho E(X)^\dagger E(X)) = \operatorname{tr}(\rho X^\dagger X) - |\mu |^2 \operatorname{tr}(\rho X^\dagger X) = 0. \]

Thus the trace of the Kadison–Schwarz gap against the adjoint fixed point vanishes. Because \(\rho {\gt} 0\) and \(G \ge 0\), this forces \(G = 0\), so \(E(X^\dagger X) = E(X)^\dagger E(X)\). Hence \(X^\dagger X\) is a PSD fixed point, which is positive definite by irreducibility (Theorem 6.4). So \(X\) is invertible. By Theorem 4.40 (KS equality implies Kraus commutation), \(X\) commutes with each Kraus operator; iterating via the left multiplicative identity (Theorem 4.42) gives \(E(X^n) = \mu ^n X^n\), so \(\mu ^n\) is peripheral.

Theorem 4.51 Complementary transfer-map gap for primitive maps
✓
#

Uses

  • Definition 4.49
  • Definition 4.23
  • Definition 4.3

Lean declarations

  • compl_eigenvalue_norm_lt_one_of_primitive

Let \(E : M_{D}(\mathbb {C}) \to M_{D}(\mathbb {C})\) be trace-preserving, and let \(\rho \neq 0\) satisfy \(E(\rho ) = \rho \) and \(\operatorname{tr}(\rho ) \neq 0\). Let

\[ P(X) = \frac{\operatorname{tr}(X)}{\operatorname{tr}(\rho )} \rho \]

be the fixed-point projection associated to \(\rho \). Assume that:

  1. \(E\) is primitive;

  2. every eigenvalue \(\mu \) of \(E\) satisfies \(|\mu | \le 1\);

  3. whenever \(E(X) = X\) and \(\operatorname{tr}(X) = 0\), one has \(X = 0\).

Then every eigenvalue \(\nu \) of \(E - P\) satisfies \(|\nu | {\lt} 1\).

Proof ▶

Let \((E - P)(X) = \nu X\) with \(X \neq 0\). Then

\[ E(X) = \nu X + P(X). \]

If \(\operatorname{tr}(X) = 0\), then \(P(X) = 0\), so \(E(X) = \nu X\) and hence \(\nu \) is an eigenvalue of \(E\) with \(|\nu | \le 1\). If \(|\nu | = 1\), primitivity forces \(\nu = 1\), so \(X\) is a trace-zero fixed point of \(E\); by assumption this implies \(X = 0\), a contradiction. Thus \(|\nu | {\lt} 1\).

If instead \(\operatorname{tr}(X) \neq 0\), trace preservation and \(\operatorname{tr}(P(X)) = \operatorname{tr}(X)\) give

\[ \operatorname{tr}(X) = \operatorname{tr}(E(X)) = \nu \operatorname{tr}(X) + \operatorname{tr}(P(X)) = \nu \operatorname{tr}(X) + \operatorname{tr}(X), \]

so \(\nu \operatorname{tr}(X) = 0\) and therefore \(\nu = 0\). In either case, \(|\nu | {\lt} 1\).

Theorem 4.54 Fixed points form a \(*\)-subalgebra
✓

Proof ▶

Thus \(X\) lies in the multiplicative domain, and Theorem 4.42 yields \(E(XY) = E(X) E(Y) = XY\).
