Quantum Entropy

This chapter develops the von Neumann entropy and its basic properties for finite-dimensional quantum systems.

Von Neumann entropy

Definition .1 Von Neumann entropy

vonNeumannEntropy For a Hermitian matrix \(\rho \in M_{D}(\mathbb {C})\) with eigenvalues \(\lambda _0, \ldots , \lambda _{D-1}\), the von Neumann entropy is

\[ S(\rho ) = -\sum _{i=0}^{D-1} \lambda _i \log \lambda _i, \]

where \(0 \log 0 := 0\).

Theorem .2 Entropy is non-negative

vonNeumannEntropy_nonneg def:von_neumann_entropy For any density matrix \(\rho \), \(S(\rho ) \ge 0\).

▼
Each eigenvalue \(\lambda _i\) of a density matrix satisfies \(0 \le \lambda _i \le 1\), and \(-x \log x \ge 0\) on \([0, 1]\). thm:density_eigenvalues_le_one
▼

Lemma .3 Eigenvalue sum

densityMatrices_eigenvalues_sum_one The eigenvalues of a density matrix sum to \(1\).

▼
Follows from \(\tr (\rho ) = \sum _i \lambda _i = 1\).
▼

Theorem .4 Eigenvalue bound

densityMatrices_eigenvalues_le_one lem:density_eigenvalues_sum Each eigenvalue of a density matrix lies in \([0, 1]\).

▼
Non-negativity comes from positive semidefiniteness. The upper bound follows because the eigenvalues are non-negative and sum to \(1\).
▼

Theorem .5 Entropy upper bound

vonNeumannEntropy_le_log_dim def:von_neumann_entropy For a density matrix \(\rho \in M_{D}(\mathbb {C})\) with \(D \ge 1\),

\[ S(\rho ) \le \log D. \]

▼
By Jensen’s inequality applied to the concave function \(-x \log x\), the entropy is maximized when all eigenvalues equal \(1/D\). lem:density_eigenvalues_sum, thm:density_eigenvalues_le_one
▼

Tripartite partial traces

Definition .6 Partial trace over \(A\)

Matrix.traceA_ABC For a tripartite matrix \(\rho _{ABC}\) on \(\mathbb {C}^{d_A} \otimes \mathbb {C}^{d_B} \otimes \mathbb {C}^{d_C}\), the partial trace over \(A\) is

\[ (\tr _A \rho _{ABC})_{(b_1, c_1)(b_2, c_2)} = \sum _{a=0}^{d_A - 1} (\rho _{ABC})_{(a, b_1, c_1)(a, b_2, c_2)}. \]

Definition .7 Partial trace over \(C\)

Matrix.traceC_ABC The partial trace over \(C\):

\[ (\tr _C \rho _{ABC})_{(a_1, b_1)(a_2, b_2)} = \sum _{c=0}^{d_C - 1} (\rho _{ABC})_{(a_1, b_1, c)(a_2, b_2, c)}. \]

Definition .8 Partial trace over \(AC\)

Matrix.traceAC_ABC The partial trace over \(A\) and \(C\):

\[ (\tr _{AC} \rho _{ABC})_{b_1 b_2} = \sum _{a=0}^{d_A - 1} \sum _{c=0}^{d_C - 1} (\rho _{ABC})_{(a, b_1, c)(a, b_2, c)}. \]

Lemma .9 Partial trace preserves Hermiticity

Matrix.traceA_ABC_isHermitian def:traceA_ABC, def:traceC_ABC, def:traceAC_ABC If \(\rho _{ABC}\) is Hermitian, then \(\tr _A(\rho _{ABC})\), \(\tr _C(\rho _{ABC})\), and \(\tr _{AC}(\rho _{ABC})\) are all Hermitian. The same holds for bipartite partial traces \(\tr _A(\rho _{AB})\) and \(\tr _B(\rho _{AB})\).

▼
Follows from \(\overline{\rho _{ji}} = \rho _{ij}\) applied entry-wise inside the summation defining each partial trace.
▼

Strong subadditivity

Theorem .10 Strong subadditivity

strong_subadditivity def:von_neumann_entropy, def:traceA_ABC, def:traceC_ABC, def:traceAC_ABC For a tripartite density matrix \(\rho _{ABC}\),

\[ S(\rho _{ABC}) + S(\rho _B) \le S(\rho _{AB}) + S(\rho _{BC}). \]

Definition .11 SSA equality

IsSSAEquality def:von_neumann_entropy, def:traceA_ABC, def:traceC_ABC, def:traceAC_ABC A tripartite density matrix \(\rho _{ABC}\) satisfies SSA equality if

\[ S(\rho _{ABC}) + S(\rho _B) = S(\rho _{AB}) + S(\rho _{BC}). \]

Definition .12 Quantum Markov decomposition

HayashiMarkovDecomposition def:ssa_equality A Hayashi Markov decomposition of a tripartite state \(\rho _{ABC}\) consists of a finite direct-sum decomposition

\[ H_B \cong \bigoplus _j H_{B_j^L} \otimes H_{B_j^R}, \]

together with a unitary change of basis on \(B\), a probability vector \((p_j)_j\), and density matrices \(\rho _{A B_j^L}\) and \(\rho _{B_j^R C}\) such that, in the adapted basis, the state becomes

\[ \bigoplus _j p_j\, \rho _{A B_j^L} \otimes \rho _{B_j^R C}. \]

The terminology follows Hayashi’s presentation of quantum Markov structure  [ Hay06 ] ; the block decomposition used by the MPDO argument is the structure theorem of [ HJPW04 ] .

Theorem .13 Hayashi SSA-equality characterization

hayashi_ssa_equality_characterization def:ssa_equality, def:hayashi_markov_decomposition For a tripartite density matrix \(\rho _{ABC}\),

\[ S(\rho _{ABC}) + S(\rho _B) = S(\rho _{AB}) + S(\rho _{BC}) \]

holds if and only if \(\rho _{ABC}\) admits a quantum Markov decomposition on the middle subsystem \(B\).

This equality criterion is cited here as an input for the later MPDO arguments. It is the equality criterion needed by Appendix C of arXiv:1606.00608. The equality conditions are reviewed in [ Hay06 , Rus02 ] ; the structural block-decomposition formulation used for MPDOs appears in [ HJPW04 ] .

Mutual information

Definition .14 Mutual information

mutualInformation def:von_neumann_entropy The quantum mutual information of a bipartite state \(\rho _{AB}\) is

\[ I(A{:}B) = S(\rho _A) + S(\rho _B) - S(\rho _{AB}). \]

Theorem .15 Mutual information is non-negative

Entropy.mutualInformation_ssa_trivial_B_nonneg def:mutual_information, thm:strong_subadditivity For any bipartite density matrix \(\rho _{AB}\), \(I(A{:}B) \ge 0\).

▼
thm:strong_subadditivity Apply strong subadditivity (Theorem .10) with trivial \(B\) (one-dimensional middle system). The SSA inequality \(S(\rho _{ABC}) + S(\rho _B) \le S(\rho _{AB}) + S(\rho _{BC})\) reduces to subadditivity \(S(\rho _{AC}) \le S(\rho _A) + S(\rho _C)\), which gives \(I(A{:}C) \ge 0\).
▼

Entropy formulations

This section states the entropy formulations used later in the development: von Neumann entropy, strong subadditivity, quantum Markov decomposition, and mutual information. These statements are cited from the standard entropy literature and supply the entropy-theoretic input for the later MPDO arguments.

Definition .16 Entropy formulation of von Neumann entropy

Entropy.vonNeumannEntropy def:von_neumann_entropy This formulation has the same value \(S(\rho ) = -\sum _i \lambda _i \log \lambda _i\) as in Definition .1.

Theorem .17 Entropy formulation of strong subadditivity

Entropy.strongSubadditivity def:entropy_von_neumann_entropy, def:traceA_ABC, def:traceC_ABC, def:traceAC_ABC For any tripartite density matrix \(\rho _{ABC}\) on \(A \otimes B \otimes C\),

\[ S(\rho _{ABC}) + S(\rho _B) \le S(\rho _{AB}) + S(\rho _{BC}). \]

This formulation introduces no new axiom: it is the same strong-subadditivity statement as Theorem .10. The underlying axiom’s proof in [ LR73 ] is deferred to the umbrella entropy issue.

Definition .18 Entropy formulation of quantum Markov decomposition

Entropy.QuantumMarkovDecomposition def:hayashi_markov_decomposition This is the entropy formulation of the Hayashi Markov decomposition from Definition .12.

Theorem .19 SSA equality iff quantum Markov decomposition

Entropy.ssaEquality_iff_exists_quantumMarkovDecomposition def:ssa_equality, def:entropy_quantum_markov_decomposition, thm:hayashi_ssa_equality_characterization For any tripartite density matrix \(\rho _{ABC}\), equality in strong subadditivity holds if and only if \(\rho _{ABC}\) admits a quantum Markov decomposition on the middle subsystem \(B\).

This formulation introduces no new axiom: it is the same equality criterion as Theorem .13.

Definition .20 Entropy formulation of mutual information

Entropy.mutualInformation def:entropy_von_neumann_entropy This formulation has the same value \(I(A{:}B) = S(\rho _A) + S(\rho _B) - S(\rho _{AB})\) as in Definition .14.

Theorem .21 Subadditivity from trivial-middle SSA

Entropy.subadditivity_ssa_trivial_B def:entropy_von_neumann_entropy, def:traceA_ABC, def:traceC_ABC For a tripartite density matrix \(\rho _{ABC}\) with \(\dim B = 1\),

\[ S(\rho _{ABC}) \le S(\rho _{AB}) + S(\rho _{BC}). \]

▼
Apply Theorem .17 with one-dimensional middle subsystem. The reduced middle state has trace \(1\) by Theorem .27, hence its entropy vanishes by Theorem .28. thm:entropy_strong_subadditivity, thm:trace_traceac_abc, thm:entropy_fin_one_zero
▼

Mutual information: monotonicity and area-law bound

The two inequalities below are the downstream MPDO-facing consequences of the entropy inequalities in this chapter. The monotonicity inequality is the strong-subadditivity content underlying the MPDO mutual-information monotonicity \(I_L \le I_{L+1}\) (arXiv:1606.00608, Proposition C.1); the elementary area-law bound is the single-site entropy bound underlying the MPDO area-law bound \(I_L \le 4\log D\) (arXiv:1606.00608, cited from the Wolf area-law bound). Both inequalities follow directly from strong subadditivity (taken as an axiom) and the single-system \(S(\rho ) \le \log D\) bound; neither introduces a new axiom.

Theorem .22 Entropy nonnegativity for finite index sets

Entropy.vonNeumannEntropy_nonneg_of_posSemidef_trace_one def:entropy_von_neumann_entropy For any PSD Hermitian matrix \(\rho \) with \(\tr (\rho ) = 1\) on an arbitrary finite index set, \(S(\rho ) \ge 0\). This is the analogue of Theorem .2 for arbitrary finite index sets: it does not require the index set to be \(\{ 0,\ldots ,D{-}1\} \), so it applies to bipartite density matrices on \(\mathbb {C}^{d_A} \otimes \mathbb {C}^{d_B}\).

▼
The eigenvalues of a PSD matrix are non-negative, and eigenvalues of a trace-\(1\) Hermitian matrix with non-negative eigenvalues are bounded above by \(1\) (each single eigenvalue is at most the total sum). Apply \(x\log x \le 0\) on \([0, 1]\) pointwise and sum.
▼

Theorem .23 Mutual information is monotone under enlargement

Entropy.mutualInformation_monotone_tripartite def:entropy_mutual_information, def:entropy_von_neumann_entropy, def:traceA_ABC, def:traceC_ABC, def:traceAC_ABC, thm:entropy_strong_subadditivity For any tripartite density matrix \(\rho _{ABC}\) on \(A \otimes B \otimes C\),

\[ I(A{:}B)_{\tr _C \rho _{ABC}} \; \le \; I(A{:}BC)_{\rho _{ABC}}, \]

where the left-hand side is the bipartite mutual information of the reduced state \(\rho _{AB} = \tr _C(\rho _{ABC})\) and the right-hand side is evaluated by expanding \(I(A{:}BC) = S(\rho _A) + S(\rho _{BC}) - S(\rho _{ABC})\).

▼
thm:entropy_strong_subadditivity Expanding both sides in entropy form and cancelling the common \(S(\rho _A)\) term, the inequality reduces to strong subadditivity \(S(\rho _{ABC}) + S(\rho _B) \le S(\rho _{AB}) + S(\rho _{BC})\). The bipartite \(B\)-reduced state of \(\rho _{AB}\) matches the tripartite \(B\)-reduced state \(\tr _{AC}(\rho _{ABC})\), ensuring the \(S(\rho _B)\) term is the one supplied by Theorem .17.
▼

Theorem .24 Elementary area-law bound on mutual information

Entropy.mutualInformation_le_log_dim_add_log_dim def:entropy_mutual_information, thm:entropy_le_log_dim, thm:entropy_nonneg_finite_index For a bipartite density matrix \(\rho _{AB}\) on \(\mathbb {C}^{d_A} \otimes \mathbb {C}^{d_B}\) with \(d_A, d_B \ge 1\) whose single-system reduced states are obtained by partial trace,

\[ I(A{:}B) \; \le \; \log d_A + \log d_B. \]

▼
thm:entropy_le_log_dim, thm:entropy_nonneg_finite_index The reduced states are density matrices because partial trace preserves positivity and trace. Combine \(S(\rho _A) \le \log d_A\) and \(S(\rho _B) \le \log d_B\) (Theorem .5) with \(S(\rho _{AB}) \ge 0\) (Theorem .22).
▼

Trivial-factor corollaries

The partial traces and von Neumann entropy are introduced in Chapter ??. This supplement collects elementary trace-preservation identities and the dimension-1 entropy bound that follow directly from those definitions. They are listed as separate results because each proof requires only unfolding definitions and re-indexing finite sums.

Theorem .25 Partial trace over \(A\) preserves the full trace

Matrix.trace_eq_trace_traceA_ABC def:traceA_ABC For any tripartite matrix \(\rho _{ABC}\),

\[ \tr (\rho _{ABC}) = \tr (\tr _A(\rho _{ABC})). \]

▼
Unfolding the definitions,

\[ \tr (\tr _A(\rho _{ABC})) = \sum _{b,c} \sum _{a} (\rho _{ABC})_{(a,b,c)(a,b,c)} = \tr (\rho _{ABC}). \]

▼

Theorem .26 Partial trace over \(C\) preserves the full trace

Matrix.trace_eq_trace_traceC_ABC def:traceC_ABC For any tripartite matrix \(\rho _{ABC}\),

\[ \tr (\rho _{ABC}) = \tr (\tr _C(\rho _{ABC})). \]

▼
Unfolding the definitions,

\[ \tr (\tr _C(\rho _{ABC})) = \sum _{a,b} \sum _{c} (\rho _{ABC})_{(a,b,c)(a,b,c)} = \tr (\rho _{ABC}). \]

▼

Theorem .27 Partial trace over \(AC\) preserves the full trace

Matrix.trace_eq_trace_traceAC_ABC def:traceAC_ABC For any tripartite matrix \(\rho _{ABC}\),

\[ \tr (\rho _{ABC}) = \tr (\tr _{AC}(\rho _{ABC})). \]

▼
Unfolding the definitions and interchanging the outer sums,

\[ \tr (\tr _{AC}(\rho _{ABC})) = \sum _{b} \sum _{a,c} (\rho _{ABC})_{(a,b,c)(a,b,c)} = \tr (\rho _{ABC}). \]

▼

Theorem .28 Entropy vanishes in dimension \(1\)

Entropy.vonNeumannEntropy_eq_zero_of_fin_one def:entropy_von_neumann_entropy If \(\rho \in M_{1}(\mathbb {C})\) is Hermitian and \(\tr (\rho ) = 1\), then

\[ S(\rho ) = 0. \]

▼
The unique eigenvalue equals the trace, hence equals \(1\), and \(-1 \cdot \log 1 = 0\):

\[ \lambda _0 = \tr (\rho ) = 1, \qquad S(\rho ) = -\lambda _0 \log \lambda _0 = -1 \cdot \log 1 = 0. \]

▼
