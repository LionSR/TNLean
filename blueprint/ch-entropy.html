19 Quantum Entropy

This chapter develops the von Neumann entropy and its basic properties for finite-dimensional quantum systems.

19.1 Von Neumann entropy

Definition 19.1 Von Neumann entropy
✓
#

Lean declarations

  • vonNeumannEntropy

For a Hermitian matrix \(\rho \in M_{D}(\mathbb {C})\) with eigenvalues \(\lambda _0, \ldots , \lambda _{D-1}\), the von Neumann entropy is

\[ S(\rho ) = -\sum _{i=0}^{D-1} \lambda _i \log \lambda _i, \]

where \(0 \log 0 := 0\).

Theorem 19.2 Entropy is non-negative
✓
#

Uses

  • Definition 19.1
  • Theorem 19.4

Lean declarations

  • vonNeumannEntropy_nonneg

For any density matrix \(\rho \), \(S(\rho ) \ge 0\).

Proof ▶

Each eigenvalue \(\lambda _i\) of a density matrix satisfies \(0 \le \lambda _i \le 1\), and \(-x \log x \ge 0\) on \([0, 1]\).

Lemma 19.3 Eigenvalue sum
✓
#

Lean declarations

  • densityMatrices_eigenvalues_sum_one

The eigenvalues of a density matrix sum to \(1\).

Proof ▶

Follows from \(\operatorname{tr}(\rho ) = \sum _i \lambda _i = 1\).

Theorem 19.4 Eigenvalue bound
✓
#

Uses

  • Lemma 19.3

Lean declarations

  • densityMatrices_eigenvalues_le_one

Each eigenvalue of a density matrix lies in \([0, 1]\).

Proof ▶

Non-negativity comes from positive semidefiniteness. The upper bound follows because the eigenvalues are non-negative and sum to \(1\).

Theorem 19.5 Entropy upper bound
✓
#

Uses

  • Definition 19.1
  • Lemma 19.3
  • Theorem 19.4

Lean declarations

  • vonNeumannEntropy_le_log_dim

For a density matrix \(\rho \in M_{D}(\mathbb {C})\) with \(D \ge 1\),

\[ S(\rho ) \le \log D. \]
Proof ▶

By Jensen’s inequality applied to the concave function \(-x \log x\), the entropy is maximized when all eigenvalues equal \(1/D\).

19.2 Tripartite partial traces

Definition 19.6 Partial trace over \(A\)
✓
#

Lean declarations

  • Matrix.traceA_ABC

For a tripartite matrix \(\rho _{ABC}\) on \(\mathbb {C}^{d_A} \otimes \mathbb {C}^{d_B} \otimes \mathbb {C}^{d_C}\), the partial trace over \(A\) is

\[ (\operatorname{tr}_A \rho _{ABC})_{(b_1, c_1)(b_2, c_2)} = \sum _{a=0}^{d_A - 1} (\rho _{ABC})_{(a, b_1, c_1)(a, b_2, c_2)}. \]
Definition 19.7 Partial trace over \(C\)
✓
#

Lean declarations

  • Matrix.traceC_ABC

The partial trace over \(C\):

\[ (\operatorname{tr}_C \rho _{ABC})_{(a_1, b_1)(a_2, b_2)} = \sum _{c=0}^{d_C - 1} (\rho _{ABC})_{(a_1, b_1, c)(a_2, b_2, c)}. \]
Definition 19.8 Partial trace over \(AC\)
✓
#

Lean declarations

  • Matrix.traceAC_ABC

The partial trace over \(A\) and \(C\):

\[ (\operatorname{tr}_{AC} \rho _{ABC})_{b_1 b_2} = \sum _{a=0}^{d_A - 1} \sum _{c=0}^{d_C - 1} (\rho _{ABC})_{(a, b_1, c)(a, b_2, c)}. \]
Lemma 19.9 Partial trace preserves Hermiticity
✓
#

Uses

  • Definition 19.6
  • Definition 19.7
  • Definition 19.8

Lean declarations

  • Matrix.traceA_ABC_isHermitian

If \(\rho _{ABC}\) is Hermitian, then \(\operatorname{tr}_A(\rho _{ABC})\), \(\operatorname{tr}_C(\rho _{ABC})\), and \(\operatorname{tr}_{AC}(\rho _{ABC})\) are all Hermitian. The same holds for bipartite partial traces \(\operatorname{tr}_A(\rho _{AB})\) and \(\operatorname{tr}_B(\rho _{AB})\).

Proof ▶

Follows from \(\overline{\rho _{ji}} = \rho _{ij}\) applied entry-wise inside the summation defining each partial trace.

A tripartite density matrix \(\rho _{ABC}\) satisfies SSA equality if

Definition 19.12 Quantum Markov decomposition
✓
#

Uses

  • Definition 19.11

Lean declarations

  • HayashiMarkovDecomposition

A Hayashi Markov decomposition of a tripartite state \(\rho _{ABC}\) consists of a finite direct-sum decomposition

\[ H_B \cong \bigoplus _j H_{B_j^L} \otimes H_{B_j^R}, \]

together with a unitary change of basis on \(B\), a probability vector \((p_j)_j\), and density matrices \(\rho _{A B_j^L}\) and \(\rho _{B_j^R C}\) such that, in the adapted basis, the state becomes

\[ \bigoplus _j p_j\, \rho _{A B_j^L} \otimes \rho _{B_j^R C}. \]
Theorem 19.13 Hayashi SSA-equality characterization
#

Uses

  • Definition 19.11
  • Definition 19.12

Lean declarations

  • hayashi_ssa_equality_characterization

For a tripartite density matrix \(\rho _{ABC}\),

\[ S(\rho _{ABC}) + S(\rho _B) = S(\rho _{AB}) + S(\rho _{BC}) \]

holds if and only if \(\rho _{ABC}\) admits a quantum Markov decomposition on the middle subsystem \(B\).

This equality criterion is cited here as an input for the later MPDO arguments. It is the equality criterion needed by Appendix C of arXiv:1606.00608. We cite Hayashi, J. Phys. A 37 (2004) L205–L208, and Ruskai, J. Math. Phys. 43 (2002) 4358; the structural block-decomposition formulation used for MPDOs also appears in Hayden–Jozsa–Petz– Winter, Commun. Math. Phys. 246 (2004) 359–374.

19.4 Mutual information

Definition 19.14 Mutual information
✓
#

Uses

  • Definition 19.1

Lean declarations

  • mutualInformation

The quantum mutual information of a bipartite state \(\rho _{AB}\) is

\[ I(A{:}B) = S(\rho _A) + S(\rho _B) - S(\rho _{AB}). \]
Theorem 19.15 Mutual information is non-negative
✓
#

Uses

  • Definition 19.14
  • Theorem 19.10
  • Theorem 19.10

Lean declarations

  • Entropy.mutualInformation_ssa_trivial_B_nonneg

For any bipartite density matrix \(\rho _{AB}\), \(I(A{:}B) \ge 0\).

Proof ▶

Apply strong subadditivity (Theorem 19.10) with trivial \(B\) (one-dimensional middle system). The SSA inequality \(S(\rho _{ABC}) + S(\rho _B) \le S(\rho _{AB}) + S(\rho _{BC})\) reduces to subadditivity \(S(\rho _{AC}) \le S(\rho _A) + S(\rho _C)\), which gives \(I(A{:}C) \ge 0\).

19.6 Trivial-factor corollaries

The partial traces and von Neumann entropy are introduced in Chapter 19. This supplement collects elementary trace-preservation identities and the dimension-1 entropy bound that follow directly from those definitions. They are listed as separate results because each proof requires only unfolding definitions and re-indexing finite sums.

Theorem 19.22 Partial trace over \(A\) preserves the full trace
✓
#

Uses

  • Definition 19.6

Lean declarations

  • Matrix.trace_eq_trace_traceA_ABC

For any tripartite matrix \(\rho _{ABC}\),

\[ \operatorname{tr}(\rho _{ABC}) = \operatorname{tr}(\operatorname{tr}_A(\rho _{ABC})). \]
Proof ▶

Unfolding the definitions,

\[ \operatorname{tr}(\operatorname{tr}_A(\rho _{ABC})) = \sum _{b,c} \sum _{a} (\rho _{ABC})_{(a,b,c)(a,b,c)} = \operatorname{tr}(\rho _{ABC}). \]

✓
#

Theorem 19.25 Entropy vanishes in dimension \(1\)
✓
#

Uses

  • Definition 19.16

Lean declarations

  • Entropy.vonNeumannEntropy_eq_zero_of_fin_one

If \(\rho \in M_{1}(\mathbb {C})\) is Hermitian and \(\operatorname{tr}(\rho ) = 1\), then

\[ S(\rho ) = 0. \]
Proof ▶

The unique eigenvalue equals the trace, hence equals \(1\), and \(-1 \cdot \log 1 = 0\):

\[ \lambda _0 = \operatorname{tr}(\rho ) = 1, \qquad S(\rho ) = -\lambda _0 \log \lambda _0 = -1 \cdot \log 1 = 0. \]
